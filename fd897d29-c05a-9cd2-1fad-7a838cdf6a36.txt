        中午十二点一到，傅夜沉并未回总裁办，但是落以柔却让自己第一次不再等傅夜沉的电话，也不主动去打电话的情况下，自己准时准点地下班了。
<br/>
<br/>        李忆莎在总裁办外的待客厅的沙发上坐了一会儿，见落以柔正在关总裁办的门，于是掏出手机，给傅夜沉打了通电话。
<br/>
<br/>        “夜沉，你什么时候回？”
<br/>
<br/>        “小落在不在？”
<br/>
<br/>        “她还在。”李忆莎下意识地看了一眼准备离去的落以柔，随后叫住了她，“小落，傅总说要你接下电话。”
<br/>
<br/>        落以柔柳眉蹙了蹙，不得不走过去，接过李忆莎手中的手机。
<br/>
<br/>        “小落，帮我带李小姐先去订餐，看看她想吃哪家餐厅的饭菜。然后把地址发给我，我随后就到。”傅夜沉淡淡地命令道。
<br/>
<br/>        落以柔哽咽了一下，才故作淡定地回答道：“好，了解。”
<br/>
<br/>        挂了电话后，落以柔将手机还给了李忆莎。
<br/>
<br/>        “夜沉跟你说了什么？”李忆莎好奇地问道。
<br/>
<br/>        落以柔没好气地反问：“李小姐喜欢去哪家餐厅？我提前去订。”
<br/>
<br/>        “今天，我想吃日本料理，至于哪家餐厅嘛！你是秘书，你选吧！”李忆莎红唇微扬。
<br/>
<br/>        落以柔瞥了李忆莎一眼，干笑了笑。
<br/>
<br/>        大哥要请这位李小姐吃饭，自然是要去高档的餐厅。
<br/>
<br/>        滨江城里所有高档餐厅，每个餐厅出什么特色，落以柔早就了如指掌。
<br/>
<br/>        所以，这点小事一点都难不倒她。
<br/>
<br/>        李忆莎本想刁难一下落以柔，却没想到，她能把傅夜沉交代的每一件事情都办得妥妥当当。
<br/>
<br/>        日本料理的餐厅里。
<br/>
<br/>        落以柔给他们选了一个幽静靠窗的雅座，什么菜好吃，什么菜不好吃，该点什么菜，不该点什么菜，她全都悉数说给了服务员听。
<br/>
<br/>        李忆莎只要坐着，等着傅夜沉来即可。
<br/>
<br/>        落以柔跟服务员交代好所有的事情后，买了单，给傅夜沉发了位置信息，便直接离去了。
<br/>
<br/>        傅夜沉赶到的时候，他刚落座，服务员便离开陆陆续续地过来上菜。
<br/>
<br/>        “夜沉，你是不是还在生我的气？”李忆莎微微垂着眸子，弱弱地问道。
<br/>
<br/>        傅夜沉优雅地解开袖扣，挽起袖口后，自顾自地拿起了筷子：“你都已经自作主张地把无中生有的事情闹到你父亲和我爷爷耳中去了，拿你父亲和我爷爷来逼我，我还有什么话可说？”
<br/>
<br/>        “那天晚上，虽然你确实没有要了我，但是你对我……”李忆莎欲言又止，故作娇羞地低下头去。
<br/>
<br/>        “我这辈子，只碰过一个女人，可惜，那个女人不爱我。”傅夜沉淡定地说道，言下之意就是，她李忆莎没必要在他面前说这些“掩耳盗铃”的话。
<br/>
<br/>        这回，李忆莎的脸是真的红了，而且还红到了耳根。
<br/>
<br/>        她不是因为害羞而脸红，而是因为谎言被戳穿了，颜面尽失才脸红的。
<br/>
<br/>        其实，他作为一个男人，已经给足了她面子，只是她却在得寸进尺。
<br/>
<br/>        “我们俩的婚礼，你打算什么时候举行？”李忆莎主动岔开话题。
<br/>
<br/>        傅夜沉忽然间“嗤”地一声，冷笑了笑：“你觉得，没有结婚证，没有爱情，光只有双方长辈的祝福的婚礼，会幸福吗？”
<br/>
<br/>        “我会努力，让你爱上我。我会努力当一名贤妻良母。”李忆莎聋拉着脑袋，完全没有了刚刚在落以柔面前的那种威风。
<br/>
<br/>        “或许，没发生这件事情以前，我会尝试去接受你的心意。但是，这件事情已经发生了。你再怎么努力也没用了。因为，于你而言，我把自己变成了‘负心汉’。我不会当一个好丈夫，当然，我和你之间，将来也不会有孩子。”傅夜沉慢条斯理地说。
<br/>
<br/>        在双方长辈的施压下，在他现在对筱颖已经不再有任何奢望下，他可以陪她李忆莎走任何过场。
<br/>
<br/>        婚礼，他可以给她，要多豪华就有多豪华。
<br/>
<br/>        但是，他不会让她成为自己的合法妻子。
<br/>
<br/>        婚后，不管是他的人，还是他的心，就更不会交给她了。
<br/>
<br/>        既然，她李忆莎愿意拿她自己的青春来跟他耗，那么他傅夜沉就跟她耗着，反正从此以后，他心里就只装着筱颖一辈子了。
<br/>
<br/>        这是傅夜沉心里做的最坏的打算。
<br/>
<br/>        对于这件事情，这并不是一个正确的处理方式。
<br/>
<br/>        只是令他傅夜沉没想到的是，李忆莎居然是一个征服欲很强，脾性又很倔，信念又很执着的女人。
<br/>
<br/>        如果一开始，他就知道她是这种女人，一定不会让她有机会接近自己。
<br/>
<br/>        可惜，这世上没有后悔药。
