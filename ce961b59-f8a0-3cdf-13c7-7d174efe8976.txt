　　第128章
<br/>　　齐德强点点头：“大武立国百二十载，戎族视我软弱可欺，这是帝国男儿之耻。”
<br/>　　程大雷撇撇嘴，心道耻什么呀，又不是没人欺过。这些年来，帝国和戎族的战争中，几乎是百战百败，根本就是被人摁在地上摩擦。
<br/>　　“九万里江山，但凡有血性的男儿就该投身疆场，以身报国。我主唯才是用，只要诸位答应接受招安，主公已答应给诸位留一个前程，日后拜将封侯，福荫后人，也未必没有可能。”
<br/>　　“好！”
<br/>　　程大雷拍案而起，朗声道：“齐将一番话，当真是说进我心里。不瞒齐将，我也是热血男儿，某一腔血也是炙热滚烫，我早恨不得纵马疆场，保家卫国。”
<br/>　　齐德强有些发愣，程大雷怎么这么大反应，自己后面还有好多长篇大论没说呢。他愣愣的看着程大雷：“程寨主是……答应了？”
<br/>　　“这个么……山寨有好多兄弟，也不能由我一人说了算。这件事还得容我和他们商量商量，不过齐将放心，那些觉悟低的，我会好好给他们做思想工作的。”
<br/>　　“如此，我也不久留了。程寨主，临行之前，主公还交待一件事，十日后请幽州境内的绿林豪杰在幽州城外的风雨亭聚会，到时程寨主一定要到，大家再一起好好商议这件事。”
<br/>　　“我恨不得十日后就是现在呐！”程大雷。
<br/>　　“那么……我就先告辞了，咱们十日后再见。”
<br/>　　“我送送齐将。”
<br/>　　“不必麻烦。”
<br/>　　“送送……一定要送送。”
<br/>　　程大雷一直将齐德强四人送到山下，直到四人上了马，跑出好远，再回头的时候，程大雷还站在山下。
<br/>　　“他们还在后面？”齐德强道。
<br/>　　“是啊，一直在后面目送您呢。”
<br/>　　齐德强回头看了一眼，莫名其妙道：“这个人有病吧。”
<br/>　　“齐将有没有听过一种说法，叫做一见如故？”
<br/>　　山下，程大雷向远方伸出手，口中感慨道：“只恨吾没有伐树之能啊！”
<br/>　　“大当家，人都没影了。”
<br/>　　“走了么。”程大雷揉揉脸道：“回山！”
<br/>　　几家寨主又在大厅聚齐。
<br/>　　大家都在议论这件事，帝国四等人，山贼连贱民都算不上，大家心里何尝不想摘掉这山贼都帽子，让自己的妻儿能抬起胸脯，堂堂正正做人。可大家却也都是自由散漫惯了，未必受得管。以前和官兵对着干，真如果招安后穿上官兵的衣服，究竟是怎样一个情况，谁心里都打鼓。
<br/>　　在众人议论的时候，高飞虎发现程大雷坐在那里，一直没有吭声。
<br/>　　“大当家，你什么意思，真要接受他们招安？”
<br/>　　大家的目光都看向程大雷，大厅内渐渐安静下来。程大雷沉默半晌，开口道：“要打仗了。”
<br/>　　“和戎族打？”高飞虎一愣：“杨龙停真这么大魄力，这么多年来，主动出击攻打戎族，这可是第一次。”
<br/>　　程大雷摇摇头：“不是和戎族打，是要打黑石城。”
<br/>　　“黑石城……”
<br/>　　“纸，拿纸笔来。”程大雷忽然摆手。
<br/>　　“这个……”高飞虎尴尬道：“我们山寨没有笔。”
<br/>　　“咦！”
<br/>　　山贼的文化水平自然高不到哪去，偌大一座青牛山，说读书的只有程大雷一个，其实一点也不过分。没有纸笔什么的，也就太不值得大惊小怪。
<br/>　　程大雷白了他一眼，只好用石子在地上画图。
<br/>　　“这里是黑石城，这里是青牛山，这里是落叶城，幽州城在那儿？”程大雷一边画图一边道。
<br/>　　“大概在这个位置。”高飞虎给程大雷点出位置。
<br/>　　“从幽州城到黑石城，青牛山有唯一一条官道，如果其他地方走，地理极其复杂，不适合大规模行军。所以杨龙停要打黑石城，必须从青牛山经过。”
<br/>　　早些前便从陆哼口中得知，薛半川篡了韩虎踞后，杨龙停大发雷霆，发誓要替亡兄报仇。只不过，这件事发生已经很久，这大半年来杨龙停怎么一直没什么动静。
<br/>　　他在准备。
<br/>　　三军未动，粮草先行，大规模行军，人吃马嚼都是个问题。而又正好赶上这个冬天，真要动手也是明年开春的事情。
<br/>　　准备来准备去，自然就准备到程大雷头上。在行军的必经路上，一伙山贼像一根钉子般扎着，总要想办法把这根钉子抹了。要么是把这些人消灭掉，要么是让这些人当炮灰，死了干净。
<br/>　　招安招安，谁能看得起四等不入的山贼。还什么封妻荫子……
<br/>　　“大当家，咱们要接受他们招安么？”高飞虎。
<br/>　　“扯淡么。”程大雷望了他一眼，宋江招安了，最后死的多惨：“招安是不可能招安的，这辈子都不可能招安。”
<br/>　　“杨龙停招安我们，只要我们答应他，肯定会让我们去打黑石城，到时候我们死光了，他再出来占便宜。”
<br/>　　“这混蛋！”几家寨主勃然大怒：“他竟想让我们去送死！”
<br/>　　“大当家，风雨亭会，咱们就别去了，直接拒绝杨龙停？”高飞虎。
<br/>　　“真如果直接拒绝他，对方定然先拿我们开刀，幽州王呐，怕是不太好惹，现在正是咱们休养生息的时候。”
<br/>　　“那怎么办，降又不能降，打也打不过？”
<br/>　　程大雷沉默片刻：“诸位可听过一个说法，鹬蚌相争，渔翁得利。”
<br/>　　“大当家的意思是？”
<br/>　　“干活啦，干活啦。”程大雷忽然站起来：“大家都别歇着了，抓紧时间干活，开春之前必须把山寨建起来。”
<br/>　　“风雨亭咱们去不去？”
<br/>　　“去，当然要去，都是幽州绿林道上的朋友，趁这个机会，我也在他们面前露露面，会一会江湖上的朋友。”
<br/>　　本已停工的工程，此时迎着严寒重新开工，工程进度自然不会太快，但却也夜以继日但进行中。
<br/>　　弹指一挥，便是十日光阴，程大雷选了高飞虎、秦蛮、徐神机随他一起同赴风雨亭。
<br/>　　程大雷在绿林道没露过面，其他人还是认高飞虎多些。而有秦蛮在便不虞此行遇到什么危险。至于带上徐神机的原因……
<br/>　　原因不言自明。
