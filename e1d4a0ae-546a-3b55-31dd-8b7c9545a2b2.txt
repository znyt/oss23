　　“嫣然也上场了。”萧逸目光一凛，站了起来，紧紧地盯向那个区域。
<br/>　　林劲和铁牛对付顾长风也就罢了，可柳嫣然不过先天七重修为，而且说实话，对于烟雨碧柳这武魂，萧逸连听都没听说过，也不知到底强不强。
<br/>　　但，萧逸有一种直觉。
<br/>　　对付顾长风，柳嫣然似乎会更轻松。
<br/>　　萧逸很清楚，自己的直觉很准，但还是略微担忧地盯住战斗。
<br/>　　另一边，顾长风冷冷地盯着柳嫣然，“你们一个个地，都为了萧逸而得罪本公子，真的值得？”
<br/>　　柳嫣然没有回答，而是说道，“看来顾长风你并不想自己滚下去，看来我只能勉为其难地帮你一把。”
<br/>　　话音刚落，柳嫣然手上的红色长鞭气势激增。
<br/>　　下一秒，一阵诡异的气息笼罩了整个比武场。
<br/>　　一缕缕白雾忽然凭空而现。
<br/>　　“嗯？起雾了？”萧逸眉头一皱，看了眼周围的奇异白雾。
<br/>　　当他回过神来，再次看向柳嫣然和顾长风对战的比武台时，顿时一惊。
<br/>　　巨大的比武场上，分出十个区域，也就是十个比武台。
<br/>　　另外九个比武台，白雾环绕，但并不严重；唯独柳嫣然和顾长风身处的区域，被一股无比浓郁的白雾所弥漫。
<br/>　　肉眼无法看到浓雾中到底发生了什么。
<br/>　　只知道，浓雾中，不时有绿光闪烁，似乎是一条条碧绿柳絮。
<br/>　　“好强的气势，嫣然竟然强到这个地步？”萧逸大大地吃了一惊。
<br/>　　他无法看到浓雾内到底发生了什么，但却感知得到浓雾内柳嫣然的气势。
<br/>　　这股气势，竟不比之前的雷战差，甚至更强。
<br/>　　在场所有武者都被这股气势惊到，纷纷停下了战斗，往浓雾方向看去。
<br/>　　十数秒后，一道人影忽然从浓雾中飞出，竟是顾长风。
<br/>　　当顾长风落下比武台时，已经浑身鲜血，生死不知。
<br/>　　下一秒，浓雾消失，柳嫣然傲然站于台上，目光看向远处的萧逸，炫耀般笑着。
<br/>　　“这...”萧逸一惊。
<br/>　　他知道柳嫣然不弱，却万万想不到强到这个地步。
<br/>　　这时，一道身影忽然跃到他的区域之下，但没有上台，仅仅是在台下，正是秦飞扬。
<br/>　　“喂，萧逸。”秦飞扬不爽地说道，“本来和林劲商量好的，我们都不上场，就等着顾长风上场，我们才去挑战他。”
<br/>　　“一人教训他一次，给你出气。”
<br/>　　“可是，没想到嫣然这么猛，一下子就把他打晕了过去，我还没出风头呢，也没机会找他麻烦了。”
<br/>　　萧逸笑笑，自语道，“这三个家伙。”
<br/>　　“我也得去挑战了，露两手，可不能单单只有你们出风头。”
<br/>　　说罢，秦飞扬一个闪身，往其他站桩者的区域而去。
<br/>　　除了萧逸这个区域，其他九个区域的战斗都在不断进行着。
<br/>　　而其中，又以林劲、铁牛和柳嫣然三人的战斗最为精彩。
<br/>　　上场挑战他们的武者，无一例外都是被一招击败。
<br/>　　当然，这三人对付其他武者倒轻手了许多。
<br/>　　第一轮考核时，由于只需有一定的分数，便能通过考核；所以三人都很胡闹地隐藏了实力。
<br/>　　第二轮考核，要迎接一轮又一轮的挑战，他们只能实力全开。
<br/>　　萧逸一直都知道他们隐藏了实力，只是没想到，他们真正的实力竟超出了自己原有的预期这么多。
<br/>　　三人的修为并不高，只在先天七重、八重，但武魂却甚是不凡，特别是柳嫣然的烟雨碧柳，自己听都没有听说过，神秘至极。
<br/>　　在灵器的力量增幅下，三人的真正实力，完全达到了半步洞玄以上。
<br/>　　但这三人，却并不在北山榜之上，倒是有些奇怪。
<br/>　　不过，想到三人胡闹的性格，还有不爱出风头的性子，倒也恍然了。
<br/>　　一天后，五百个武者都经历了不少的战斗，挑战或被挑战。
<br/>　　最终，有十人始终站在比武台上，无人可以撼动，前十名，就此出炉。
<br/>　　另外的四百九十人，一旁观战的执事也给出了排名。
<br/>　　除了少数几人不服，并再次发生了几场战斗后，再没有其他异议。
<br/>　　十一到五百的排名，就此定了下来。
<br/>　　最后，是前十名间的战斗，到底谁是第一，谁是第十，还需分个高下。
<br/>　　萧逸、林劲、铁牛、柳嫣然、秦飞扬、雷战、何健、赵不群、何苍、王虎。
<br/>　　比武台上的十位站桩者，互相对视了几眼，都知道彼此的差距和实力水准。
<br/>　　互相切磋了几番，观战的执事打出分数，给出排名。
<br/>　　众人没有意见，排名便彻底定了下来。
<br/>　　第一，萧逸；第二雷战；第三林劲；第四铁牛；第五柳嫣然；第六秦飞扬；第七赵不群；第八何健；第九何苍；第十王虎。
<br/>　　顾长风并不在前十，他被柳嫣然打成重伤后，晕厥了过去。
<br/>　　一位执事将他救醒，但重伤的情况下，并未能再次成为站桩者，最后的排名只有11，比王虎还不如。
<br/>　　顺带一提，秦飞扬自从突破到先天九重后，实力大增，连赵不群都败于他手上。
<br/>　　事实上，他的排名，应该更前。
<br/>　　只是，这家伙在当站桩者时，面对一些武者的挑战，不停放水，战斗中又时常耍帅，便让得观战的执事给他打低了分数。
<br/>　　半晌，易老出现，接过其他执事的排名记录，看了一眼后，点了点头。
<br/>　　特别是看到萧逸时，一向严肃、不苟言笑的他，竟稍微露出了满意的笑容。
<br/>　　“嗯，不错，双考核第一，倒是让老夫有些意外。”易老淡淡地说着。
<br/>　　“谢易老夸奖。”萧逸不卑不亢地回答了一声。
<br/>　　说起易老，第二轮考核刚开始，他就离开了。
<br/>　　观战的，都是些普通执事。
<br/>　　直到所有战斗结束，他才重新回来。
<br/>　　想来，他应该在裂天剑派中地位不低，这普通的观战和打分，并不需要他来处理。
<br/>　　又或许，他是个不爱凑热闹的人。
<br/>　　这时，易老的声音再次响起，“首先，祝贺你们三百人，正式成为裂天剑派的弟子。”
<br/>　　第二轮考核，只取前三百名，现在还站在广场上的，也只有三百人。
<br/>　　另外两百人，已经被送出山门外了。
<br/>　　今年，来参加裂天剑派考核的武者，足有一万多人。
<br/>　　可如今，却只剩下三百人，可想而知淘汰率有多高。
<br/>　　另外，通过第二个考核后，这三百人已经是裂天剑派的真正弟子了。
<br/>　　第三个考核，考验和决定的，是其他事情。
<br/>　　“第三个考核，不需要你们战斗，原地休息半个时辰，恢复精力，即可随我离开。”易老淡淡地吩咐了一声。
<br/>(本章完)
