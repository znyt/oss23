　　经过多个世界修改，《道种诀》早就变得面目全非。但有一点始终未变。道种的核心一直都是苏重的灵魂力量。
<br/>　　用道种承载天机印，是苏重深思熟虑的决定。
<br/>　　凶戾气是天地间凶恶之气，对心神有严重侵蚀。修炼庚金剑气他对此感悟更深。现在只是刚刚修炼，心里却时常会涌现杀戮欲望。
<br/>　　好在有聚元内力中和戾气，副作用大大降低。但剑气符积累增多，必然会影响苏重。
<br/>　　就像原本轨迹中张小凡，每天带着一个邪异兵器，时常有失控发狂的危险。
<br/>　　苏重可是直接把戾气放在体内，副作用更大！
<br/>　　用道种承载天机印，能变相的保护神魂。有天机印守护，苏重对戾气的忍受力将会大大提升。
<br/>　　“是时候升级庚金剑气了。”苏重对本源的使用越发熟练。
<br/>　　心念一动，庞大本源融入技能面板，庚金剑气技能立刻变得模糊。
<br/>　　脑海内道种和天机印内容，瞬间变得清晰深刻。
<br/>　　以前一直感觉难以融合，现在却灵感不断。种种路径不断出现。
<br/>　　苏重眼睛发亮，很快找到方向开始实施强化。
<br/>　　只是几个呼吸时间，一点本源就消耗干净。而技能面板上，庚金剑气变成了“未知法诀”的字样。
<br/>　　庚金剑气最神奇的地方是剑气符，现在天机印融入道种，彻底掌控剑气符，整个法诀变得面目全非。
<br/>　　丹田内，一颗绿豆大小乳白色圆珠出现在中心，就好似定海神，牢牢把控丹田。原本四处游移的剑气符，此刻全被道种吸住旋转，成一个漩涡状。核心处正是全新道种。
<br/>　　庚金剑气最神奇就是剑气符，现在天机印融入道种彻底掌控剑气符，整个法诀变得面目全非。技能面板上才会出现未知法诀字样。
<br/>　　“还是继续叫道种算了，省的麻烦。”苏重也懒的起名。
<br/>　　他更看重全新《道种诀》的威力。走出房间快步没入黑竹林，苏重心念移动，直接激活剑气符。
<br/>　　噌噌噌……
<br/>　　一枚枚剑气瞬间破体而出。咔嚓咔嚓……
<br/>　　坚硬黑竹在剑气面前不堪一击，瞬间就被斩断。一波爆发，苏重面前出现一个宽五米长五十米的笔直通道。沿途黑竹林全被轻易斩断。
<br/>　　别看庚金剑气难练，但确实是上等杀伐利器，这还只是九十九道剑气。要是凝聚成千上万道剑气，猝不及防下爆发，一死就得一大片！
<br/>　　“看看道种诀的威力。”苏重站在原地不动。
<br/>　　丹田内乳白色道种一胀一缩，就好似心脏一样跳动，凶戾之气和草木精气同时向苏重汇聚，顺着毛孔丝丝缕缕钻入丹田。不等凶戾气肆虐，立刻就被吸入道种。同时吸入的还有草木精气。
<br/>　　道种不断跳动中，丝丝白色力量被放出，不等飘散，立刻就被围绕道种的剑气符捕捉吸收。原本暗淡的剑气符，瞬间就有了莹莹光泽。
<br/>　　道种涨缩间，吸收凶戾气和草木精气，放出的白色力量则被剑气符争抢吞噬。
<br/>　　剑气符就好似水中游鱼，绕着道种欢快抢食。
<br/>　　只用十个呼吸的时间，九十九枚剑气符就彻底恢复。
<br/>　　“不愧是诛仙剑阵标配，天机印吸收存储能量能力强大。用来喂饱剑气符，简直小菜一碟！”
<br/>　　“有道种在，凝聚剑气速度必然会更快。”苏重当机回竹屋，开始闭关凝聚庚金剑气符。
<br/>　　以前修炼，他要先汇聚凶戾气，再用聚元内力进中和，最后才凝聚出剑气符。
<br/>　　现在道种融合天机印，同时把聚元功加入其中。只要催动道种，就能一步到位，瞬间造就剑气符所需能量。他只要把道种放出来的力量凝聚成剑气符就好。
<br/>　　没有戾气暴动，不用专修聚元功中和戾气，一门心思凝聚剑气符，苏重修炼速度飞快。
<br/>　　没多大会儿功夫，他就凝聚出了一枚剑气符。
<br/>　　“以前至少需要三天才能凝聚成功，现在最多也就一刻钟时间。”功法本质提升，苏重高兴不以。
<br/>　　他刚想继续修炼，突然感觉一股凶戾之气冲天而起。
<br/>　　“后山？怎么回事？”修炼庚金剑气，苏重对凶戾气极熟。有道种加成，对凶戾气更加敏感。
<br/>　　“去看看！”苏重不在犹豫，起身快步离开竹屋。
<br/>　　在山间不断奔行，苏重不由苦笑：“没了聚元功，轻功都没法用。”
<br/>　　没了聚元内力，他只能用身体力量快速赶路。每次一蹦数十米，就想一只大蛤蟆蹦跳前进。
<br/>　　大竹峰后山是连绵山脉，山上种满黑竹。越往深处走，黑竹越茂密。根本就没有路径，苏重只能在竹子上借力，在半空跳跃奔行。
<br/>　　很快就到达目的地，在枝繁叶茂的黑竹林内，一块圆形空地突兀出现，其内寸草不生光秃秃一片。
<br/>　　此刻，田灵儿张小凡两人正趴在空地边缘昏迷不醒。
<br/>　　刚才冲天煞气已经消失不见，但空地中心一滩水洼处，仍释放着丝丝缕缕凶戾气。
<br/>　　注意到张小凡手中一根漆黑短棍，苏重了然点头：“怪不得煞气冲天，原来是这东西出世了。”
<br/>　　这根不起眼黑棍，是两件邪异古宝血炼而成，看着普通，实际凶恶得很。一个不慎就会被它噬魂夺魄。
<br/>　　苏重伸手将黑棍抓在手中，只觉入手清凉，丝毫没有异状：“两件邪宝融合，诡异平衡，反到让兵器内敛起来，不用法力催发，还真不容易发现这件法宝本质。”
<br/>　　把黑棍噬魂别在腰间，苏重一手提着张小凡，一手提着田灵儿。
<br/>　　临走时才发现，张小凡怀里还抱着一只灰色小猴：“主角装备到账，世界已经拉开帷幕！”把灰色猴子一并带上，苏重朝着大竹峰飞奔而去。
<br/>　　田不易居所，大竹峰众弟子在院子里焦急等待
<br/>　　“小师弟，到底是怎么回事？”宋大仁担忧道，张小凡和田灵儿两人早晨还好好，现在却昏倒被送回来，他心里念头纷飞，难道有人敢闯入大竹峰行凶不成？
<br/>　　“我也不太清楚，等师傅出来再说吧。”苏重没告诉众人他的发现。噬魂可不是个好东西，即便几位师兄弟值得信赖，但苏重觉得还是少几个人知道的好。
<br/>　　在几人焦急等待间，房门从里面打开，田不易面无表情走出。
<br/>　　“都聚在这里干什么，都散了吧！”田不易挥手就要赶走众人。
<br/>　　“师傅，师弟师妹没事吧？”宋大仁急忙开口闻到。
<br/>　　田不易间众人担忧，心里欣慰脸上表情松动：“不是什么大事，你们不要管了，我和你师娘会处理。”
<br/>　　然后对苏重摆摆手：“尔旦，你跟我进来。”
<br/>　　苏重一言不发跟着田不易进屋，是他吧两人待会来，田不易当然要过问。
<br/>　　“尔旦，你给我说说，是怎么回事？”刚才急着检查徒弟和女儿伤势，只是知道个大概，确定不是人为之后就没详细确认。现在有了时间，他当然要详细询问。
<br/>　　“是后山，那里有一处凶险之地，内力充斥凶戾之气。小凡和灵儿两人无意闯入其中，被戾气侵蚀才昏迷不行。”苏重没隐瞒如实道。
<br/>　　“就在后山？”田不易脸色不好看。这里是大竹峰，就是他的后花园，自己家里竟藏着这么个凶险之地。
<br/>　　这次幸好没事，要是真出了问题，他后悔都来不及。
<br/>　　“走，我们去看看。”田不易毫不迟疑，让苏茹留下照看张小凡和田灵儿，激发飞剑化作一到红光，裹挟着苏重腾空而起。
<br/>　　后山深处，田不易站在光秃秃空地上，眉头紧紧皱起。
<br/>　　“真是个凶恶之地，只是残留气息就如此危险！”田不易感受着皮肤上针扎般的刺痛凝重道。
<br/>　　“如此险地，灵儿两人是如何逃出去的！”田不易不得不多想。他是大竹峰首座，就怕有人打青云主意，从他女儿和徒弟身上下手。
