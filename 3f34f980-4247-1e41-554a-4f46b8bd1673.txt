　　这圣山，也是庞大无比，其中无数洞府，错综复杂。
<br/>　　这些洞府，也是有大有小，每一个都有些不同。
<br/>　　来到圣山修炼，并不规定指定洞府，所以可以自由挑选，楚风眠当然要挑选一座灵气浓郁的洞府，来修炼。
<br/>　　“这圣山之中，是以金玄九尊山灵气最为浓郁，先去那边看看再说。”
<br/>　　楚风眠心中自言自语一声，便是顺着那边走了过去。
<br/>　　这九玄金尊山，是整个圣山中灵气最为浓郁的一方地域，楚风眠进入其中，放眼一看，其中的无数洞府，已经全部是被人占据。
<br/>　　“看来这九玄金尊山，没有洞府可以修炼了。”
<br/>　　楚风眠低声自言自语一句，刚准备离开，去寻找其他洞府。
<br/>　　便是突然看到，在这九玄金尊山上，一阵灵力波动，随后在一座洞府中，一道身影，突然飞了出来。
<br/>　　这道身影看了楚风眠一眼，便是直接向着外面飞了出去。
<br/>　　毫不理会楚风眠。
<br/>　　“有人刚走，倒是正好。”
<br/>　　楚风眠笑了笑，身形一动，直接便是向着那人刚刚离开的洞府走去。
<br/>　　看来他的运气还算不错，正好是遇到了有人修炼结束出来了。
<br/>　　“走。”
<br/>　　楚风眠身形一动，直接便是向着那一方洞府飞了过去。
<br/>　　但是几乎同时，一尊年轻男子身影，也突然从外面，走了进来。
<br/>　　此人一身长袍，头戴玉冠，也几乎同时，看到了那唯一空缺的洞府。
<br/>　　“不错，这九玄金尊山，居然是有洞府空余。”
<br/>　　这年轻男子，扫视了一眼，看到楚风眠身上的衣着，居然是外门弟子的时候，他的眼神中，露出了几分不屑。
<br/>　　身形一动，也是同样向着那洞府冲了过去。
<br/>　　“站住！”
<br/>　　年轻男子，猛然大手一挥，突然一股灵力，便是封锁了那空余洞府，将楚风眠，直接便是拦在了外面。
<br/>　　“恩？”
<br/>　　楚风眠眉头一皱，打量了一眼这年轻男子。
<br/>　　这年轻男子的衣着，看起来应该是一尊内门弟子，他的实力，跟白萱，大约是在伯仲之间，应该也是地榜前十的一尊内门弟子。
<br/>　　这样的人物，楚风眠对付起来，也并非容易。
<br/>　　可是楚风眠却是并没有退让的打算，这洞府，本就是楚风眠先看到了，白白拱手相送，可不是楚风眠的性格。
<br/>　　“小子，这洞府归我了，你去找其他的洞府去吧。”
<br/>　　这年轻男子，看着楚风眠，一双眼睛便尽是不屑，丝毫没有将楚风眠这样一个外门弟子，放在眼中。
<br/>　　他高高在上，君临天下一般，看着楚风眠，语气便是跟命令，没有丝毫区别。
<br/>　　“一个外门弟子，没有资格来这九玄金尊山修炼，对你来说，随便去一个什么洞府都足够了！”
<br/>　　“是吗？这洞府是我先看重了，还是你自己去寻找其他洞府吧。”
<br/>　　楚风眠摇了摇头道，完全无视了这年轻男子的话。
<br/>　　“什么？一个小小的外门弟子，也敢跟我顶嘴，放肆！敢这样跟我说话？”
<br/>　　这年轻男子，看着楚风眠，一脸怒容，疯狂的咆哮了起来。
<br/>　　这嚣张气焰，令楚风眠都有些无语，这人不过是一个内门弟子，听这语气，还以为他是北邙学院的院长呢。

<br/>　　但是像是眼前这年轻男子一般，如此这种抢人洞府不成，直接咆哮的，简直是嚣张到了极点。

<br/>　　楚风眠不屑一笑道。
<br/>　　“这洞府是我看中的了，你不用打这主意了！”
<br/>　　别说是这洞府，对楚风眠本来就有着大用，就算是没用，就这年轻男子如此嚣张的话，楚风眠也不可能会让给他。

<br/>　　这洞府，今日楚风眠定不会让。
<br/>　　“该死！一个御风境的蝼蚁，区区外门弟子，居然敢跟我东一齐抗衡？”
<br/>　　这东一齐怒声开口道。
<br/>　　“给我滚，不滚我就废了你的境界！”
<br/>　　东一齐？
<br/>　　楚风眠看了一眼这年轻男子，突然的明白过来了此人的身份。
<br/>　　这东一齐，便是在内门中，位列地榜第八位的天才，虽然比起白萱要差一位，但是实力上，却是已经跟白萱在伯仲之间。

<br/>　　楚风眠满脸不屑的开口道，一双眼睛，尽是没有将这东一齐，放在眼中。
