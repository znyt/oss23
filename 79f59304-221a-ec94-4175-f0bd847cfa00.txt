<br/>    众多人王开始陆续回到营地，然后便被告知不用继续出去了，紫霞仙子已经准备渡劫成为大君了，现在，是真正决战的时候了。

<br/>        毕竟这一次又不像是陈明渡劫，解千愁压根没反应过来陈明就已经在渡劫了，等解千愁赶到，估计陈明渡劫都渡完了。

<br/>        这一次完全不一样，解千愁可是在外面看着的。

<br/>        这个距离，你想要解千愁不发现都难，解千愁绝对会过来插一脚。

<br/>        好处在于，自己现在已经集结了所有战力，对方并不是全部在场。

<br/>        众多修行者都是齐齐拱手道：“我等遵命！”

<br/>        说实话，若是在外界，让他们给紫霞仙子护道，有点可能，但是不会所有人全去，誓死搏杀，但是现在情况可是不同，毕竟他们若是有着两位大君，必定能够打赢这一场战争！
<br/>    
<br/>        紫霞仙子能够成功突破，他们才有机会活命。

<br/>        解千愁带领着十个王级军团走出大营，可以看得到对面最前方的陈明，而在陈明的身后，一个人王接着一个人王涌出。
<br/>    
<br/>        陈明和解千愁在两军阵前漠然对视，如同有着火花迸溅。

<br/>        身旁的悟道境们，却是没有等两位大君动手，此刻已经朝着对方冲去。

<br/>        道完，紫霞仙子飞天而起，左手之中已经是一柄瑰丽的长弓出现，作为群星仙门这样的古仙门之中最强大的天才，仙器自然不会少。
<br/>    
<br/>        仙器，陨星弓。
<br/>    
<br/>        紫霞仙子伸手拉住弦，弦拉满月，一道陨星般的光辉飞出，正中劫云。
<br/>    
<br/>        劫云之中一阵变化，一条紫色的雷蟒瞬间扑下，直接将紫霞仙子笼罩在其中，滚滚的雷霆如同暴雨般落下，瞬间将整个海面变成了雷霆地狱！

<br/>        战场之上，众多王级避开了两个正在对视的大君，将战场迁移到别的方向，大君之间的战斗，他们可没有兴趣参与。

<br/>        战场之中，陈明和解千愁周围方圆十里，已经是空荡无比，只留下他们两人。

<br/>        解千愁扫了紫霞仙子一眼，“你觉得，这一次，我会让她这么容易的渡劫不成？”
<br/>    
<br/>        陈明笑了笑，道：“不是还有我吗？”

<br/>        陈明干笑了一声，“哎呦，被发现了！那就只好直接动手了！”

<br/>        陈明一旦动手，可没有时间说话，心念一动，手握万道天竹，已经是开始挥动。

<br/>        然后大家你一招我一招，互相过招，这不是傻吗？！

<br/>        而从他背后飞来的星辰，也是在这一刻一一撞碎在弥天刀网之上。
