　　“道风大大，你真的……要上山吗？”
<br/>　　“北斗观就在前面，为什么不去。”
<br/>　　道风说话间，已经走到了山路的尽头，来到了山顶，两边各有四个弟子拦路，眼看道风上山，一起祭出法器，集合大伙的力量，布置出了一道结界，希望能将道风抵挡片刻。
<br/>　　北斗山上，尸横遍野，精魄横飞，都是被道风一路上斩杀的北斗观的弟子。
<br/>　　这已经是最后的防线，北斗观的精英弟子都被他差不多杀干净了。
<br/>　　道风伸出双手，轻轻拍打在结界上，用力一推，结界登时破碎，元神之力冲射出去，直接将几个震飞，坠落在地上，直接就挂了……
<br/>　　道风没有留情，他虽然不喜欢杀人（只是没兴趣去做），但如果不杀他们，他们起来还会抵抗，自己没有时间可以浪费。
<br/>　　这是他最后的机会了，他苦心营造这一切，就是为了这个最终的目的：斩杀星月奴的肉身！
<br/>　　“可是，你现在元神之力越来越弱了，这样下去，你会元神俱灭的！”
<br/>　　道风身边，一个姑娘不断地抗议着。
<br/>　　是一个身穿黄色长裙的姑娘，身体接近半透明，但是除了道风，没人能看见她。
<br/>　　她是秋莹，叶少阳最后一个收的妖仆，她有个极为特殊的专长：隐身！
<br/>　　之前，她跟着叶少阳等人一起去了承德，在叶少阳第一次发现古墓、与那个看不见的邪物打斗的时候，她就隐了身形和气息，跟着那邪物一起离去……
<br/>　　那邪物果然就是轩辕山七大长老之一的影魅，秋莹跟着他来到轩辕山，见影魅和清长风密谋：九星叠气阵虽然可以吸纳生灵进入，但那是人间的镜像空间，但又跟青冥界这样的不同，生灵进出那个世界，需要气息的平衡。
<br/>　　简单说，就是生灵进去的时候什么样，出来的时候还得是什么样，人间生灵，如果在这个镜像空间里积累了太多修为，回到人间，会造成人间规则的紊乱，也许会爆体而亡，也许会有更加可怕的结果。
<br/>　　连星月奴也不是很清楚，但是她不敢冒险，因此，她魂魄离体，进入那个空间去吸收愿力、积累修为，留下一具肉身在轩辕山，让清长风帮她妥善安置。
<br/>　　她的肉身，本来是在她自己的宗门的密室里放着，但是知道的人太多，星月奴让人带话给清长风，让他帮忙悄悄转移，清长风跟影魅商量，将星月奴的肉身悄悄挪到了自己的北斗观中，安置在自己的修所里，自己平时哪儿也不去，更没人知道这件事，算是绝对的保险。
<br/>　　然而，他万万没想到的是，当他搬运星月奴肉身时，秋莹就在他旁边……她这隐形的能力，就像包子的读心术一样，简直是个bug。
<br/>　　办成这件事后，影魅便进入了古墓空间，去为星月奴护法了。秋莹生怕进去了出不来，就没跟进去。这个时候，叶少阳已经被困在古墓里了，秋莹思来想去，只好去鬼域寻找道风，把这一切都告诉他。
<br/>　　道风这才打定主意，要来轩辕山斩杀星月奴的肉身，不过为了保密，他并没有说出这最核心的真相，而是自己谋划……本来，他也想过多找几个人一起，但那样容易被发现，而且自己还要照顾别人，他最最信任的，终究还是他自己。
<br/>　　他喜欢独来独往。
<br/>　　本来，他是在等一个机会，结果假叶少阳主动找上门，于是将计就计，策划了这场行动……
<br/>　　远处，几道光影快速地在丛林间穿越，飞射而来，道风知道那是清长风找来的救兵，那几个大佬。若是让他们赶到，就凭现在的自己，那真是一点希望都没有了。
<br/>　　“休要废话，快点带路！”道风呵斥一声，将最后一点元神之力压榨出来，纵身飞往北斗观。
<br/>　　而今的他，是元神状态，但又不是纯粹的元神——他凭借着炼气化神的法术，将那一口精气灌入元神，炼化出了一个分身，跟另外两个分身一样，有模拟血肉的形体，但是没法操控法器，只能靠元神之力伤人。
<br/>　　叶少阳的元神之力，能直接对付一半鬼妖，与之相比，道风的元神之力更强，地仙以下，基本上都能一击秒杀。
<br/>　　但元神之力恢复起来极慢，道风在这一战之中，又将元神、魂魄和肉身一分为三，各自为战，也是使出了自己所有的本事。
<br/>　　毫无保留，世人总算见识到了他的终极实力。
<br/>　　秋莹在前面引路，进了北斗观，又遇到一波伏击，快要耗尽元神之力的道风，拼死冲破结界，杀了几人，一直冲进了清长风的修所。
<br/>　　修所里有个书桌，书桌上有一块黑玉蟾蜍的镇纸，是开了光的镇宅灵兽，平时听清长风讲经布道、在北斗观浓郁的灵力熏染之下，早已经成精。
<br/>　　道风刚进去，这蟾蜍精暴起，化成人形，立刻朝他扑过来。
<br/>　　“休得动手！”
<br/>　　身后传来爆喝，道风回身看去，是那几个大佬，从半空中降临，顷刻就到，于是对秋莹道：“你快走，这里奇人众多，未必不会发现你！”
<br/>　　说完，把心一横，纵身上前，双手猛地掐住那玉蟾蜍精，引燃了元神，最后一股元神之力，爆发而出……
<br/>　　玉蟾蜍惨声大叫，拼命挣扎不出，化成灰烬。
<br/>　　“道风大大……”
<br/>　　受到这股巨大的元神之力的冲撞，秋莹也被震飞了出去。
<br/>　　道风靠着一缕不屈之念，朝着房间对面挪动——真的是挪动。
<br/>　　星月奴的肉身，就端坐在房间一头的蒲团上，摆着打坐的姿势，一动不动。前后左右各摆放着一只香炉，里面点着三色循魂香，熏染着肉身。
<br/>　　她的肉身，看上去年纪很小，只有十五六岁的样子，身穿一件锦绣罗衫，十分白净水灵，带着两个很大的耳环，眉心处贴着黄花，容貌十分标志，有点婴儿肥，看上去很可人的一个小姑娘。
<br/>　　她居然是掌控法术公会的大阴谋家星月奴的肉身……
<br/>　　道风艰难地挪到跟前，双手按着星月奴的肩膀，引燃了元神最后的力量……“住手！！”
