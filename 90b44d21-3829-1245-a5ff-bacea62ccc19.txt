　　芮冷玉手里提着松纹古定剑，御空飞行，从叶少阳头顶飞过去，叶少阳才回过神来，人整个顿时懵了。
<br/>　　“少阳！”
<br/>　　小九紧跟着从树林里飞来，身边跟着凤兮和林三生，之后是捉鬼联盟其他成员。
<br/>　　“发生什么事了？”叶少阳一眼扫过去，见大伙都在。
<br/>　　“我们遇到阴司的追兵，想要闯过去，结果冷玉突然跳下来往回飞奔，可能是为了逃脱他们的追捕，我们只好回来追她！”
<br/>　　“阴司！！”叶少阳大惊，就在这时，从对面视线可见的树林深处，渗进来一团黑气，仿佛最浓烈的烟雾一般，从树木草丛之间飞快地渗进来。
<br/>　　叶少阳一看就知道这是阴司高等阴神出没所驾驭的阴云，来不及多想阴司为什么也会来插一档子，急忙说道：“你们跟着我，暂时别跟他们交手，把他们引到山谷那边！”
<br/>　　说完，根本来不及去管来的是谁，自己认不认识，转身去追赶芮冷玉了。
<br/>　　芮冷玉一路奔行到谷口方向，钻入太阴山、空界联盟和尸族三方交战的战场之中，消失不见了。
<br/>　　叶少阳也顾不上谁跟谁，在战场上一通冲杀，到处寻找芮冷玉的身影。
<br/>　　“在这！”林三生喊了一声，率先飞了过去。叶少阳紧紧跟了过去。
<br/>　　芮冷玉似乎是在逃避身后的追兵，有点慌不择路，在人群中转了一会，竟然直接奔着山谷进去了。叶少阳在后面叫了几声，芮冷玉完全不理，只要也追了过去。
<br/>　　阴云总算从树林中蔓延出来，在山谷前的空地上铺展开来，阴云散去，却是牛头马面和黑白无常四神，身后跟着一众鬼差阴神，看到眼前三方混战的一幕，也是震惊了。
<br/>　　“诸位阴神，既然来了，速帮我空界杀敌，抵挡尸族入侵！”摇光仙子见到牛头马面等人之后，立刻急切地恳求道。她率领的空界强者们，在尸族入侵之后，形势巨变，现在已经完全处于劣势，见到阴神众神赶来，虽然惊诧，但也是眼前一亮，如同抓住了一根稻草，立刻求救。
<br/>　　阴司四神互相看了看，牛头将军对摇光仙子拱手说道：“仙子恕罪，我等奉命而来，捉拿转世鬼童，责任重大，不敢因旁事耽搁。”
<br/>　　“什么！”摇光仙子又惊又怒。
<br/>　　她身边一个道士嘶声喊道：“你们怎敢如此！我空界与阴司同理连枝，一向互助，现在危难在即，你们居然说是旁事？”
<br/>　　马面看了他一眼，冷冷说道：“陷空道人，这话你也好意思说，你们空界一向想跟阴司撇清关系，当年转轮王前来抓人，你们是怎么对付的，现在有了危难，又想起我们来了？阴司与太阴山作战，你们可曾派出过一兵一卒？”
<br/>　　空界诸人面面相觑，却是理亏，一句话也说不出。
<br/>　　牛头立刻命人穿过阵营去追芮冷玉。
<br/>　　太阴山和尸族势力听说他们只是来抓转世鬼童的，不想对付自己，乐得不行，自然不会自找麻烦，纷纷让路，让他们过去。
<br/>　　眼看着阴司众神从山谷中穿过去，一个头陀打扮的的和尚心急如焚，叹道：“即便如此，你等既然来了，也不能见死不救吧？”
<br/>　　白无常回头看他，眼皮一翻，说道：“说起来是应该帮忙，但是……我们来之前，天子殿有言，青冥界自此与阴司完全决断，自生自灭，互不相干，嘿嘿，你们应该也知道是怎么回事。”
<br/>　　天子殿……
<br/>　　空界众人面面相觑，果然是崔府君从中作梗。那个看上去永远温文尔雅的崔判官，原来也是有脾气的。不过这件事他们不怪崔府君，只是狠摇光仙子和黎山老母把事情做绝：杀了人家儿媳妇，这真的……虽然他有本事让人复活，但是像他那种大人物，面子上算是极大的侮辱，报复也是理所当然的。
<br/>　　摇光仙子心中也是后悔不迭，不过到这一步，也只能咬牙坚持了，怒喝道：“你们怕什么，没有别人帮忙，我们难道就要被赶尽杀绝不成，我们空界千年基业，什么时候依靠过别人了，诸位强打精神，只要支撑片刻，援兵便会赶到！”
<br/>　　听了她的鼓励，有几个人开始奋发杀敌，但也有人小声嘀咕。
<br/>　　“说的这么大义凛然，好像你刚才没恳求人家出手似的……”
<br/>　　摇光仙子羞怒不已。
<br/>　　“小天师……”
<br/>　　叶少阳追着芮冷玉进了山谷，才发现里面也是一场混战。芮冷玉不见了踪影，正在寻找，突然听见身后有人叫自己，转头看去，立刻看到了牛头马面和黑白无常，心立刻颤了一下。
<br/>　　“小天师也知道我等为了何事而来。”
<br/>　　叶少阳道：“知道，我们算是自己人，我不想跟你们为敌。”
<br/>　　白无常叹了口气道：“我等何尝不是……因此请小天师放下执念，现在回头，还来得及。”
<br/>　　叶少阳低头沉默了一瞬，抬起头，递给他们一个苦涩但却异常坚定的笑容，说道：“若这就是执念，那我执念太深，回不了头了。”
<br/>　　说完对一干门人说道：“帮我顶住他们，不许伤了一个。”
<br/>　　林三生等人立刻列阵以待。
<br/>　　白无常等人望着他们，也是十分别扭，道：“阴阳司司主，副司主，主簿，你们也都是阴间正神，我等今天领天子诰命而来，阻挡我等，就是对抗阴司……后果，你们自己清楚。”
<br/>　　林三生、小青小白等人互相看了看，露出微笑，林三生行礼拜道：“多谢七爷提醒，不过……我们今天站在这里，不是阴司的正神，我们都是捉鬼联盟的成员，我们的目的只有一个……”
<br/>　　“誓死追随老大！”小白坚定的说道。
<br/>　　众人一起点头。
<br/>　　“誓死追随！”
<br/>　　叶少阳在奔跑途中听见这句话，眼眶湿润了。
<br/>　　白无常等人也是面面相觑。
<br/>　　“老七，真打？”牛头小声嘀咕。
<br/>　　“问我，好像是我要来似的，上峰有令，我们能怎么办？”
<br/>　　黑无常道：“这下不了手啊。”
<br/>(本章完)
