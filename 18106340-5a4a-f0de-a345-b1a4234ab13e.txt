　　第653章
<br/>　　“大当家，你过来看……”
<br/>　　看到程大雷走过来，和珅分开人群，冲程大雷打了个招呼。
<br/>　　程大雷走近了，见面前有一个深洞，底下黑洞洞的，也看不清楚里面藏着什么。
<br/>　　“下面是什么？”
<br/>　　“刚刚发现，把其中的恶气放一放，等下便派人进去。”
<br/>　　周围的人都在交头接耳的议论。听他们谈话，程大雷便听明白了大概。
<br/>　　和珅着手管理，大家一起修整破旧的朔方城，无非是将坍塌的房屋修好，派给迁入朔方城的流民。然而，目前朔方城的情况是房多人少，一家一户总能分到合适的房子，所以这项工作才完成得慢些。
<br/>　　也是程大雷再三催促，大家才加快工作进度。关键还是城墙的修补，诸人准备采房中废墟修补城墙时，发现了这个地窖。
<br/>　　等了也有些时间，大概地窖中的恶气已经散尽，放了蜡烛进去，再提出来时，蜡烛并未熄灭。
<br/>　　“大当家，可以让人下去了？”和珅道。
<br/>　　程大雷点点头，望着脚底的深洞。
<br/>　　和珅唤了两个兵丁过来，冲他们指了指：“你们两个下去看看？”
<br/>　　二人面露难色，神情犹豫不敢下去。这些人总有些畏惧鬼神，谁知这地窖中会不会藏着些稀奇古怪的物事。
<br/>　　关鱼在一旁立着，表情十分难看。他现在带领鱼字军驻扎朔方城，而这两个兵是刚刚从流民中招募的，若是他原先的手下，当二话不说便跳入地窖中。
<br/>　　“有令不行，该斩。”关鱼已经摁住自己的佩刀，想着要杀人立一立军威了。
<br/>　　程大雷看着这一幕，蛤蟆城的人是看着一座城从无到有建起来的，所以人心都聚在一起，但朔方城却不同。
<br/>　　人心呐人心，人心要怎样才能凝聚，岂不就是现在。
<br/>　　想到此处，他冲关鱼摆摆手，制止他的行动。自己开口道：“好了，我下去看看。”
<br/>　　“大当家，下面不知有什么，还是让手下人下去吧。”和珅一惊拦住程大雷道。
<br/>　　“什么话，本大家的命是命，他们的命就不是命么。”程大雷道：“危险就让本大家先闯一闯。”
<br/>　　周围看热闹的诸人都难以置信的看着程大雷，乱世之中，帝国诸多诸侯无不是将手下的性命当做筹码，谁会说出这般话。
<br/>　　程大雷瞧见了这一幕，心中暗暗得意：这便是人心呐。
<br/>　　况且，一个破地窖会有什么危险，本就是系统奖励的，就算有什么所谓的凶险，程大雷自量也撑得住。
<br/>　　和珅与关鱼见程大雷心意已决，也都没再阻拦。关鱼抽出自己的佩刀，双手递给程大雷：“大当家小心。”
<br/>　　“好说，好说。”
<br/>　　程大雷倒提钢刀，腰里拴着绳子，被一点点放进去。这地窖竟然颇深，有三四丈左右，也不知道当初修地窖那人是怎么想的。
<br/>　　其实程大雷并不了解，凉州这边多有戎族侵犯，几乎家家户户都有地窖，只盼着戎族过来时能够躲过一劫。如此，自然是越深越好，毕竟活命要紧。
<br/>　　双脚触底之后，程大雷将绳子解了。好大一会，才适应眼前的黑暗，见地窖中的摆设很是简单，铺着一堆稻草，还有一些已经风干的食物。待他走近了看，忽然吓了一跳：只见稻草堆上有一具白骨。
<br/>　　除此之外，倒也没有什么。墙角摆着十几口箱子，程大雷打开看发现里面是一些金锭银锭，其他箱子里想必也是这些东西。
<br/>　　程大雷略略一想，便大概明白是怎么会事。当是某年某月戎族打过来时，这财主躲进了地窖。只不过后来不知发生怎样的事，他没能活着出来。
<br/>　　程大雷对金银珠宝的兴趣不大，沿着绳子回到地面上，命人将地窖中的东西搬出来。
<br/>　　程大雷已经走过一趟，自然也没什么凶险，只是见到程大雷这等身份还亲自冒险，诸人对他的信任又多了几分。
<br/>　　接下来的事情便交给和珅去做，这些财物搬出来后，派人送到蛤蟆城。在那里，程大雷已经有一座藏宝库。
<br/>　　这些年走南闯北，程大雷还颇收集到一些这样的东西。金银珠宝，翡翠玉器，这些东西花也花不出去，留着也没太大的用意，于是便用一座宝库收着，钥匙在苏樱手中。到如今，里面还颇有些世上的奇珍异宝。
<br/>　　程大雷回到了城主府，窝在大厅里，想下一步该做什么。群雄割据的大势下暗潮汹涌，说不得以后还会发生怎样的事，目前关键是发展自己的实力。只有足够强的时候，才能撑过这个乱世。
<br/>　　现在要做的还是扩充朔方城的人口，靠眼前的一两万人，都算不得一座标准城池。
<br/>　　可想要招揽流民毕竟不是一件容易的事。程大雷看着刚得到的广播技能，想了想选择使用。
<br/>　　之后，也没有肉眼可见的变化。程大雷也不知道这个技能使用后，会带来怎样的后续反应。
<br/>　　当然，系统一直就是这个尿性，程大雷早已习惯。他喝了杯茶，觉得有些困意，回屋睡觉去了。
<br/>　　……
<br/>　　徐州，琅琊城。
<br/>　　一场登陆战正在此地进行，大船从海上驶来，载着全副武装的精锐士兵。面对对方的船只，刘顶天根本没办法形成有些阻拦，只能在沙滩上与对方展开白刃战。
<br/>　　李行哉一马当先，手中握着霸王点金枪往前冲。
<br/>　　“王上，小心些，小心些……”戚继光心急如焚，这样的战斗李行哉本不用亲自参战，可他就像是疯了一般，拦也拦不住。
<br/>　　“你管我做什么，指挥，去指挥。”李行哉大吼。
<br/>　　戚继光有些无奈，只好令亲卫保护好李行哉，自己挥舞着令旗，命令士兵进攻。
<br/>　　一队队士兵早已经过千锤百炼，或者说是戚继光的折磨，互相配合无比默契。此刻他们五人一组，将匪贼各个击破，然后继续向前进攻。
<br/>　　如今这个时代的战争，总之还是很简单的，无非就是用人命往上堆。沙滩上横尸遍野，血流成河，来自落凤城的士兵前赴后继，向前进攻。
<br/>(本章完)
