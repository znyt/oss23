　　第356章
<br/>　　风雨飘摇，帝国这艘大船已经是千疮百孔，如宋游渠，薛问婷这些人还天真的以为：他们生来富贵，高人一等，而他们的富贵还将千世百世永远的传承下去。
<br/>　　可帝国并不缺乏有识之士，如宋伯康这类人，很早就遇见到，帝国这艘大船撑不了多久了。要么你就做着自己的春秋大梦，当这艘船沉没时，与他同归于尽。要么你就是在之前，选好新船，再搏一场富贵。
<br/>　　在这种状态下，有人的野心在滋长，想要逐鹿天下，搏一个帝王基业。有人想要将生平所学卖给天下英雄，以期待未来的封侯拜相，比如和珅。有人心怀苍生，想结束这乱世，还天下一个太平，帝国并不缺这种人。当然，更多人还是想活下去，单纯的只是活下去，比如程大雷。
<br/>　　当乱世到来时，绝不会有体面的存身之地，大家将会像野兽一样厮杀，然后不停的死人，死人，死人，直到死的人够多时，重新制定规则。
<br/>　　如果，在此时还有人叫嚣着祖上阔过，宣扬体内流淌的血液天生高人一等，几乎等同于蠢货。
<br/>　　宋伯康就是预见到这一点，才会觉得宋游渠留在程大雷之处，未必是一件坏事。哪怕双方是敌人，但对于程大雷的本事，宋伯康都是佩服的。宋游渠如果能从程大雷身上学到一二分东西，在乱世就多一二分活下去的可能。
<br/>　　而宋游渠能在短短一年时间能成长得这么快，肯定也少不了程大雷忽悠的功劳。
<br/>　　世界常常为青年人所改变，同时，青年人也是最好忽悠的一批人。
<br/>　　他们干净单纯，单纯的善或者单纯的恶，心里还是相信美好、善良、正义这些字眼的。
<br/>　　宋游渠等十几个纨绔二代被关在琴川关，程大雷怎么会放过这个忽悠洗脑的机会。
<br/>　　宋游渠等很早就被放了出来，有自己独立的住处，他们并没有受到歧视，当然，在这个山贼窝子里，谁歧视谁还说不定呢。
<br/>　　他同样与大家一起劳动，甚至参与士兵的训练中，身上那点纨绔子弟的做派很快就被洗刷干净，程大雷抽时间给他们做做思想政治工作，从底子改变他们的固有观念。
<br/>　　集体可以降低一个人的智商，想改变这些人并不困难。
<br/>　　一开始，宋游渠、薛问婷都是畏惧程大雷的，而畏惧又何尝不是一种崇拜。年轻人么，谁不想成为英雄，那年夏天的武科场，程大雷毫无疑问是最耀眼的存在。
<br/>　　若有可能，这些纨绔公子，未必不想成为另外一个程大雷。如果做不到他做到的事，那就只好崇拜。
<br/>　　一年足以改变太多事情，现在的程大雷，俨然已经成为众多青年人的偶像。
<br/>　　甚至于说，把宋游渠放回去，都有可能成为己方间谍。
<br/>　　而这也是程大雷的目的之一。
<br/>　　秋收之后，便是农闲，大家为将来的冬天做准备。村镇之中，时不时发出高歌之声，这算是劳动过后，为数不多的消遣。
<br/>　　当然，大家能闲着，程大雷可不能闲着。
<br/>　　这一日，程大雷骑着黑牛，悠哉悠哉的出了琴川关。
<br/>　　他身后银眸佩剑背斧，神情依旧冷得像块冰。
<br/>　　二人一前一后，倒成了荒野上一道靓丽的风景线。在如今的琴川，不认识程大雷的有，但不认识这头黑牛的实在是太少见了。
<br/>　　关键是这头黑牛长得……大家不好意思说这黑牛丑，只好说这黑牛长得有特点。
<br/>　　既然认出了这黑牛，也就认出了牛背上的程大雷。
<br/>　　程大雷经过之处，纷纷有人跪倒。
<br/>　　这些人来自全国各地，有些是被地主欺压无家可归，有些是在战火中失去了家人，当然，其中也不发杀人犯、赌徒，但无论如何说，来到琴川后，因为程大雷的缘故，他们都能端上一碗饱饭。
<br/>　　如此，这跪拜却也是诚心诚意的。
<br/>　　程大雷一开始对这种事情不是很适应，在琴川关内部大家也不是跪来跪去，程大雷总觉得这种事情很折寿的。
<br/>　　还是和珅与程大雷说，身为上位者，要有神秘感，要高高在上……
<br/>　　程大雷很满意上位者这三个字的形容，于是也就接受了这个建议。
<br/>　　黑牛缓步而行，程大雷却也并不着急，甚至有意压着速度。
<br/>　　他这次去的是落玉寨。
<br/>　　如今的程大雷有了人，也有了粮，马上农闲，也有了大把大把的时间，程大雷决定将落玉寨升到三级。
<br/>　　当然，一套良好可执行的制度会避免很多麻烦，可是生产力的发展才能推动集体向前进步。
<br/>　　只可惜程大雷不掌握任何科学技能，就只能把一切寄希望在系统上。
<br/>　　如果能把落玉寨升到三级，该可以给系统商店开出新的商品。
<br/>　　这对程大雷来说不算怎样的困难，他已经有三次升级经验，甚至都耗不了太多时间。
<br/>　　真正令程大雷觉得为难的，却是不知如何面对樊梨花而已。
<br/>　　有些事情该解决的时候不解决，拖下去，只会变得越来越麻烦，当你想要解决的时候，却发现已不知该如何解决。
<br/>　　这句话正应了程大雷现在的状况。
<br/>　　可再远的路，终有一个终点，在日落黄昏时分，程大雷和银眸到了落玉寨。
<br/>　　如今的落玉寨和先前自然不同，一些老弱已经迁到琴川，哪里更适合生活。留下的，大部分是精壮年轻人，在山谷内部，骏马奔腾，都是从戎族购来的上好战马。
<br/>　　“大当家，您来了。”
<br/>　　守门的小卒见到黑牛就认出了程大雷，立刻向门内禀报，程大雷由人牵着牛进入落玉寨，见一排排人在门口迎接。
<br/>　　他们都是统一的黑衣黑甲，背上有狰狞丑陋的***案。
<br/>　　程大雷挥挥手，让他们各忙各的，自己走进聚义厅，自顾自在中间那张太师椅上坐下。
<br/>　　目光转了一圈，却是没看见樊梨花，程大雷奇怪问：“人呢？”
<br/>　　“病了。”徐灵儿道。
<br/>　　“病了？”程大雷怔了怔：“什么病？”
<br/>　　“也不知道怎么会事，知道你要来就病了。”徐灵儿道：“大概得的是不想见你的病吧。”
<br/>　　
<br/>(本章完)
