　　第138章
<br/>　　清早，随着第一声鼓响，落叶城的城门缓缓打开。
<br/>　　如往日一样，城门口挤着诸多进城的人。有挑着担子进城做小买卖的，有家里揭不开锅去城里富亲戚那里借钱的，有食不果腹想在城里奔门营生的。
<br/>　　“都别挤，谁往前挤的，本军爷的鞭子可不认人。”
<br/>　　申屠大喝一声，手中的鞭子在空中发出一道爆响。
<br/>　　今天和往常一样，没什么出奇之处。等着进城的人里，也没有那个像可以挤出油水。而那些真正的富户，申屠也不敢压迫得太狠，说不得他们和城里的贵人有什么瓜葛，一道条子递过去，自己的差事就丢了。
<br/>　　从远处过来三个做小买卖的，一个卖草鞋的，一个做卖绿豆的，一个卖猪肉的。
<br/>　　“这是你编的草鞋。”申屠问了一声，在看到对方脸的时候，一股莫名的悲怆涌上来，自己坎坷的仕途，被上官欺压的岁月，此刻都想了起来。
<br/>　　“手艺不错，好好干。”
<br/>　　本来申屠还想拿对方两双草鞋，不知怎的就打消了这个年头，还鼓励了对方两句。
<br/>　　三个做小买卖的混进城。
<br/>　　不多时，从北边来了一个公子哥，座下高头大马，马上搁着一柄长枪，马前还有一个牵马的女人。
<br/>　　最讨厌这样的富家子弟，仗着有几个凑钱就四处花天酒地，兴风作浪。而这种人，却也是申屠最不敢得罪的。他连盘问都不敢，便将对方放进了城。
<br/>　　又来一个挑担卖柴的壮汉，对方真是生得好相貌，立在自己面前就像一座铁塔一般。
<br/>　　“你干嘛？”
<br/>　　申屠还未盘问，就遭到对方一声吼。他吓了一跳，随后壮着胆子道：“我问问你怎么了，你凶什么凶，那蛤蟆大王怎么样，还不是昨夜被抓了。好啦好啦，进去吧。”
<br/>　　随后，城门口又来了进城卖把式的兄弟，猎户小赵……申屠擦了擦额头的汗水，忽然觉得有些累。
<br/>　　今天一切如常，可为何总感觉有些不对，照道理来说，冬天不该有这么多人进城的。
<br/>　　做小买卖的，骑马的公子哥，担柴的壮汉，卖把式的兄弟，猎户小赵……这些人进城之后，转了一圈，最后陆陆续续都从后门进了湖底捞。
<br/>　　“军师！”
<br/>　　做小买卖的刘关张，公子哥林少羽，牵马的小白狼，担柴的秦蛮，卖把式的高飞虎高飞豹，猎户赵子龙……
<br/>　　徐神机摆摆手：“山寨来了多少人？”
<br/>　　“包括我们在内，今天共有五十余人混进城。”刘悲道：“黄总管坐镇山寨，山寨两千能打的弟兄都准备好了，以烟火为号，只要咱们决定动手，大队随时可以杀来。”
<br/>　　“好！”徐神机亢奋道：“今夜咱们就动手，砸破大牢，救出大当家。”
<br/>　　“大当家肯为我们冒死借粮，我们就能为他赴汤蹈火。”熊大熊二道。
<br/>　　“慢，慢。”刘悲多看了熊大熊二一眼，他是没想到这二兄弟还能说出『赴汤蹈火』这样的词：“这只是最坏的打算，想那大牢必定防卫森严，咱们就算能救出大当家，也必定有很多兄弟送命。况且那大牢什么情况咱们一无所知，咱们不能莽撞动手。”
<br/>　　“那你说怎么办？”徐神机昨夜一宿没睡，光顾着磨刀了，那句话怎么说来着，他的小刀早已饥渴难耐。
<br/>　　“来的路上我已想过，咱们先要派出人打探大牢周围的环境，还有那苏家小姐，也要派人保护起来。当然，我想到的这些，军师也一定早想到了。”
<br/>　　“……”徐神机顿了顿，扶着胡须道：“那是自然，我正要安排人去做。”
<br/>　　“最最重要的是……”刘悲道：“可不能让大当家在牢内受委屈啊。”
<br/>　　蛤蟆寨的山贼中，并不缺住过牢的人，据他们所说：落叶城的大牢号称活人进，死人出，里面十八般刑具，样样齐全，到了里面不死也得脱层皮。不提其他，里面吃的东西连猪食都不如，刷锅水加把糠就算是牢饭了，其他什么沙子，铁钉，鼻涕都能在牢饭里找到。
<br/>　　想到这里，徐神机忽然觉得鼻头有些泛酸：“大当家那里受得了这个，咱们一定要早早把他救出来啊。”
<br/>　　……
<br/>　　今天是苏四海出殡的日子，城内有很多人来吊丧，大家看到灵前身穿孝衣，梨花带雨的李如，心里都想：这苏四海生前好艳福啊。
<br/>　　不过大家都有些奇怪，像苏四海这样的富户，死后停灵至少该七七四十九天，请和尚道士做三堂法事，怎么苏四海头七没过，便被送出了门。
<br/>　　当然，这念头大家也只是在心底想想而已，树倒猢狲散，没人会在这个时候为苏四海出头。
<br/>　　一番忙碌后，夜来李如回到家中，刚回到自己屋中他就迫不及待将孝衣脱了。
<br/>　　身上只穿着贴身的亵衣，臂膊大腿露在空气中。这样的天气，是有些寒的，可李如却觉得浑身清凉。
<br/>　　生人三十六年，从未体验过这样的清凉。
<br/>　　少小贫贱，那不成事的父母对自己自然无多少偏爱。别人家的女儿穿新，自己穿旧，别人戴花，自己扎绳，也只有大哥会将辛苦赚来的钱为自己买一块糖点。
<br/>　　从小李如便知道，自己身后一无所有，老天唯一给自己的便是这身好相貌。
<br/>　　十三岁被卖给花子，当年就被破了身，之后在几个富户手中流转。
<br/>　　看她花容月貌，比三月梨花还有娇嫩，谁知她背后早已千疮百孔。
<br/>　　十八岁落到苏四海手中，为妾便是为奴，便是下人厨子也敢给自己脸色看。自己肚子又不争气，没给苏四海留下一个儿子，不能母凭子贵。
<br/>　　如此活到三十六。
<br/>　　如今，在整个苏家人人都要管自己叫一声夫人，万贯家财，千亩良田再无人能和自己争。
<br/>　　她杏目内若有明月，从未如今日一般迷人。
<br/>　　或许是因为此刻她瞳孔里装着整个苏家……
<br/>　　自己再不是一无所有，自己再不用仰人鼻息。
<br/>　　生人三十六，她可以把头抬起，深吸一口这清凉刺骨的空气。
<br/>　　这是三十六年第一次。
<br/>(本章完)
