<br/>    这个家伙别看外表斯斯的，但是斗起狠来，根本是一个亡命之徒。三寸人间那天在匪巢傅华差一点要了许公子的小命，当时傅华那个凶神恶煞的样子，也证明冯葵所言不虚，现在傅华被欺负成这个样子，绝对不会这么罢休的，她相信傅华一定会想办法报复馨园集团和许公子的。现在傅华和万家可谓同仇敌忾了，这让她觉得万家报复的机会来了。<br/>　　<br/>　　    万家虽然是没有足够的实力打倒馨园集团和许副市长，但万家胜在是岭南这边的土著，是扎根很深的地头蛇，许多傅华不掌握的情况，万家掌握，万家完全可以把这些提供给傅华，跟傅华联合起来，利用这些情报一起跟馨园集团和许副市长好好的斗一斗。<br/>　　<br/>　　    万红梅开始密切注意傅华的动向了，她发现傅华去了香港，去找了一个乔玉甄的女人。对这个女人万家是多少知道点根底的，乔玉甄身的事情发生过很多事情，还跟一些较神秘的人物有过交集，反正是一个背景模糊不好惹的人物。然后是那个带傅华出现在匪巢里面的余芷青。余芷青在香港深圳一代有着黑寡妇的外号，算是凶名卓著，也是个不好惹的女人。<br/>　　<br/>　　    刚看到这三个家伙凑到了一起，万红梅还替冯葵好一顿的不值，冯葵一个人在美国正因为相思而备受煎熬，傅华这坏蛋却在香港左拥右抱，风流快活，大享齐人之福。<br/>　　<br/>　　    不过万红梅很快觉得事情不是她想的那么简单，这一段时间这三个家伙行踪诡秘，凑在一起好像是在密谋什么，虽然还没有什么大的动作，但是暗地里却是好像在酝酿着什么大事件一样。万红梅仿佛都能嗅到风那种浓郁的阴谋味道。<br/>　　<br/>　　    而且万红梅所了解到的这三个人的性情而言，他们密谋的事情肯定是要对馨园集团造成极大的打击的。万红梅身体内继承自她爷爷军人嗜血的因子开始躁动起来，她觉得这是一次为万家出气的大好机会。<br/>　　<br/>　　    当然了，目前的形势而言，万红梅认为这三个人的目标当肯定没有许副市长。这三个家伙目前的身份都是商人，是老百姓，民不与官斗许副市长一直也没跟他们直接发生冲突，他们应该不会主动把目标对准许副市长的。许副市长只是跟他们万家交恶，目前来时是万家要针对的主要目标。<br/>　　<br/>　　    在针对目标这一点，傅华和万家并不是一致的，但是万红梅却有信心让傅华和乔玉甄、余芷青三个家伙跟她一样，把目标对准许副市长的。因为许副市长是馨园集团最大的后台，许副市长不倒，馨园集团还是伤不到根本的。<br/>　　<br/>　　    接下来的问题是怎么去把这个联盟建立起来了，正在这个时候，万红梅发现傅华回了北京，相对来说，许副市长和馨园集团的触角还伸不到北京去的，于是她踩着傅华的脚步也追到了北京来。<br/>　　<br/>　　    听万红梅讲完这些之后，傅华觉得万红梅说的要对付许副市长和馨园集团的第一个理由是有些牵强的，这个女人要是真的那么为他着想，早主动来找他了。估计万红梅是想引起她同仇敌忾之心，才把他的事情拿出来作为第一个理由的。<br/>　　<br/>　　    对万红梅说的第二个理由傅华是相信的，这其实是没落世家跟当权派之间的一场利益之争，他看到的许公子和馨园集团张扬跋扈的做派，他们在跟万家发生利益之争的时候，下手肯定是毫不容情地。<br/>　　<br/>　　    但是他却不能因为万红梅轻飘飘的几句话，把他和余芷青、乔玉甄这件商量了那么长时间的计划，全盘端出来跟万红梅分享。即使这个万红梅曾经跟冯葵关系很密切也不行。因为他们的计划最关键的一步是不能让对手知道他们想做什么，如果对手事先知道了他们的计划，计划也功亏一篑了，甚至他们还会为此付出惨重的代价。<br/>　　<br/>　　    傅华看了万红梅一眼：“你的故事我听完了，我也相信你跟我一样，都对馨园集团恨之入骨，但是我不知道我能帮你做些什么？”<br/>　　<br/>　　    “做什么啊，一起对付许副市长和馨园集团啊。”<br/>　　<br/>　　    “等等，你首先要搞清楚一点，许副市长从来不是我的目标，你不要把我和你的目标扯为一团。起馨园集团来，许副市长这个目标太大，我目前的能力，还没办法对付这个大家伙。除非你们万家能够出手对付他。”<br/>　　<br/>　　    “这个你不说我也是要这么做的，万家这几年手里也是掌握了一些许副市长的违法乱纪的证据的，我可以把这些都提供给你，你拿出这些可以对付许副市长了。”<br/>　　<br/>　　    这大概是以前当权的官员不愿意轻易得罪在地的士绅的原因吧，因为在地的士绅对地方熟悉，或多或少都会掌握一些当权者的劣迹的，这也是士绅们能够保持对当权者影响力的一部分因素。<br/>　　<br/>　　    但是仅仅是提供出证据是不够的，一方面傅华对深圳的地面并不熟悉，他没有办法去查证万红梅提供的证据是否真实。二来，要举报一个官员，举报人也是需要承担很大风险的，万家只提供证据并不负责将证据交到相关部门，所有的风险也转嫁到他的身了。<br/>　　<br/>　　    另外一方面，他对万红梅所有的信任都是建立在万红梅跟冯葵的关系之的，而且这些还都是万红梅自己跟他说的。但是冯葵跟万红梅之间的真正关系是怎么样的，他实际并不清楚。现在人心险恶，谁知道万红梅今天来找他，会不会是个圈套呢，如果她是在跟他玩反间计呢？<br/>　　<br/>　　    傅华笑笑说：“我觉得如果万家想要参与这个游戏，仅仅是提供证据出来是不够的。我拿到了证据也无法查证真假，没办法查证真假，我自然不敢贸然的去使用这些。”<br/>　　<br/>　　    “那你想万家怎么做？”<br/>　　<br/>　　    “简单啊，万家完全可以自己冲在前面的。”<br/>　　<br/>　　    “你想要万家冲在前面打头阵？”万红梅的脸色变了，因为万家的根基在政界，现在的政界是一个完整的生态系统，其的每一个细胞都跟其他细胞相互关联，互为因果的，往往是牵一发而动全身。<br/>　　<br/>　　    如果万家站出来揭发许副市长的话，也等于是跟整个生态为敌。万家实际也是依赖这个生态系统生存的。跟整个系统为敌也意味着万家是在自掘坟墓。万红梅的大伯之所以会对许副市长和馨园集团的欺凌哑忍，实力不足只是其一方面，更重要的一方面是万家和对手之间其实是生存在一个系统之的，不到万不得已，他们是不想毁灭整个系统的。<br/>　　<br/>　　    万红梅说道：“不行，万家给你提供子弹可以，绝对不能出头露面的。”<br/>　　<br/>　　    “其实万家也不需要自己出头露面的，你们应该养过一些死士之类的人吧？现在正是需要这些人效力的时候，让他们出来把这件事情担起来，不会危及到万家的利益了。”<br/>　　<br/>　　    自春秋战国开始，国的世家大族有养士的传统，他们养的士种类很多，有出谋划策的谋士，有以命搏富贵荣华的战士……其最神秘最诡谲的那种是傅华所说的死士。<br/>　　<br/>　　    春秋战国时期，那些著名的刺客要离、专诸、荆轲之流的，都是死士。但死士并非只有刺客这样一种类型。还有一种死士刺客更为神秘。这一种死士会跟豢养他们的人保持距离，甚至大多数人都不会知道死士跟豢养他们的人还有关联。这样子才能够保证别人不把死士所做的事情联系到豢养他们的人身。<br/>　　<br/>　　    在豢养他们的人需要做某件事情而又不能出面的时候，这种死士会挺身而出为他们的主人牺牲自己，这种牺牲包括生命、家庭、事业等等之类的。基本死士挺身而出了之后，他们原本拥有的一些东西可能都会失去，他们对于主人来说也没有了存在的价值。<br/>　　<br/>　　    这种死士还有一种名称叫做弃子，也是可以被舍弃的棋子。从一开始成为这种死士开始，他们的命运已经注定了，注定是要在对主人最关键的时刻被牺牲。但真正获取最大利益的往往却是他们的主人。<br/>　　<br/>　　    万红梅看着傅华苦笑：“你把我们万家想象的也太牛了，我爷爷可没那么长远的打算。”<br/>　　<br/>　　    傅华有些无语了。基本每个时期都是有当权者的，这些当权者在台的时候，威风八面，一言九鼎，所有人的表现基本是大同小异。差别在于他们不再当权了之后。<br/>　　<br/>　　    那些长于计谋，懂得这世界是风水轮流转的人，会在当权的时候做好布局，这样子即使将来不在继续当权了，也能保证他们安享晚年。而有些当权者却在这个时候只顾着去威风了，根本没想不当权了怎么办。<br/><br/><br/>
