　　“叮！轮回者庞博死亡！”
<br/>　　一道淡漠的声音突然在混沌道君耳边响起。
<br/>　　“咦？！”
<br/>　　混沌道君那一直淡漠的目光中，终于多出了几分兴趣，他轻咦一声，道了一句“有趣！让我来看看究竟发生了什么事。”
<br/>　　庞博是主角叶梵的伙伴，身上多少沾了一些大气运，具有妖族血脉，遇见妖族大能夺舍都没能死亡，到最后叶梵成帝也都没死，现在却死了，真是让人好奇。
<br/>　　混沌道君心念一动，神念一瞬间扩散到灵墟洞天，因为那里是叶梵与庞博所在之地。
<br/>　　瞬间，在他眼前便出现了一副有趣的画面。画面中，一个黑衣男子正手持一柄黑色匕首，迅速刺向叶梵，叶梵向旁边快速闪躲而去，险而险之的躲避开来，而旁边地上却躺着一人，倒在血泊中，正是庞博。
<br/>　　混沌道君一眼便看出，那个黑衣男子有大约接近金丹期的实力，而现在刺杀比凡人强一点的叶梵，手持一柄法器，竟然没能一击必杀，真是奇怪。
<br/>　　而且最让混沌道君感兴趣的是，那个黑衣男子的修炼体系与这个世界格格不入，他的出现，竟然没有引来天道的关注，也只有在刺杀叶梵之时，才有一股莫名的威压出现在上空，天上雷霆闪烁，汇聚而来，似乎是要毁灭什么东西。
<br/>　　“不好！真是该死！”
<br/>　　黑衣神秘人忽然大叫一声，手中匕首不停，连忙向着叶梵杀去，速度之快，迅若闪电，讲究一击必杀。之前估计是有庞博的阻拦，才没能让他杀死叶梵。
<br/>　　“有趣，竟然不是此界中人。”道君仿佛遇到了什么非常感兴趣的事情，他一步踏出，消失不见。
<br/>　　“时间——静止！”一道淡淡的声音响起，在黑衣神秘男子耳边，宛如雷霆炸响，在他心中掀起惊涛骇浪。
<br/>　　随着话音落下，整片天地，彻底安静了下来。空中飞鸟静止不动，草丛中虫子安静了，被风吹动的树木，竟然还倾斜着……
<br/>　　世界变得彻底静止了，好似银幕中的画面被暂停了，每个人都保持着原有的动作不变，他们的思想，也在这一刻静止了。
<br/>　　“灵魂——聚！”
<br/>　　道君口出道音，瞬间一股无形的力量涌现，庞博那刚刚已经散去的灵魂，在这一刻汇聚而来，凝聚成一个人的模样，钻进庞博体内。
<br/>　　“生命——现！”
<br/>　　又一声道音响起，可惜谁也没能听见。却见地上的庞博那被刺杀的伤口正以一种肉眼可见的速度恢复起来，那洒在地上的血液逆流而上，重新进入他的体内，那原本已经停止的心跳，又重新开始缓缓跳动起来。
<br/>　　“唔，真是麻烦！”
<br/>　　混沌道君轻轻道了一声，向着庞博与叶梵轻轻一拂，之后抓着黑衣神秘人，消失不见。
<br/>　　在道君消失后，那静止的状态，又瞬间恢复原状，该飞的鸟依旧在自由自在地飞翔着，该摇的树，依然按原来的轨迹摇动着。
<br/>　　谁也没有发现什么不对，仿佛世界本该如此。
<br/>　　“呃，庞博，怎么回事？你怎么会倒在地上？还有，我怎么也倒在了地上？”叶梵从地上爬了起来，看着地面上的庞博，有些迷糊地问道。
<br/>　　“我们好像是……睡着了？”庞博有些迷糊道。
<br/>　　“好像真是？”叶梵有些不确定道。
<br/>　　“管他怎么了？赶紧修炼才是硬道理，主神任务可不简单啊，小心被抹杀了，特别是你那坑人的荒古圣体。”庞博大大咧咧道。
<br/>　　“也是。”
<br/>　　于是，两人便不再管其他，赶紧回去继续修炼，毕竟，混沌神殿一直是悬在他们头顶的一柄刀啊，随时可能落下。
<br/>　　……
<br/>　　混沌神殿中。
<br/>　　混沌道君面前多了一个神秘的黑衣人，正是之前刺杀叶梵之人。
<br/>　　“你是什么人，为何会出现在遮天界？”道君开口问道。
<br/>　　这声音听在男子耳中，好似是他最亲近的人在问他话，让他完全没有一点防备，情不自禁地开口答道：“我叫王谊，代号“亡谕”，是万界杀神阁的一员，来这里是因为我接了刺杀叶梵的任务。”
<br/>　　“万界杀神阁是什么？”混沌道君目光微凝，心中有些猜测，出声问道。
<br/>　　“万界杀神阁是一个培养杀手的地方，谁也不知道它在什么地方，这是一个遍布诸天万界的神秘存在，专门接受任务，刺杀人物。我只是刚刚进入其中，知道的不多。”
<br/>　　“培养杀手？遍布万界么，有趣！”道君轻笑一声，在心中却是暗暗猜疑，这是系统流还是别的什么。于是，他继续问道：
<br/>　　“如何进入万界杀神阁？”
<br/>　　“不知道，每次任务完成是由神阁自行接引。啊……”
<br/>　　说着，还不等他将话说完，便只听那人大叫一声，身体竟然在一瞬间爆炸开来。
<br/>　　“咦，在我的领域内，竟然还能控制自爆，万界杀神阁，还真是不简单！有趣，真是越来越有趣了。哈哈！”
<br/>　　……
<br/>　　与此同时，在一个遥远的不知何处的地方，屹立着一座高大壮丽的宫殿，宫殿外无尽混沌，但都不能侵蚀宫殿分毫，它仿佛亘古便屹立于此。
<br/>　　宫殿金碧辉煌，宝光闪耀，“太辰宫”三个鎏璃大字闪闪发光。
<br/>　　宫殿之中，却是鸟语花香，色彩艳丽，彩霞飞舞，神瀑飞落，龙凤呈祥，神光璀璨，一片人间仙境的模样。
<br/>　　在宫殿深处，两个四五十岁模样的男子正在下棋论道。一男子身穿青色道袍，神色不变，自有一番威严。另一人身穿白袍，模样俊俏，脸上带着淡淡笑意，好似春风拂面。
<br/>　　“咦！”
<br/>　　突然，青衣男子轻咦一声，好似发现了什么。另一边的白衣男子说道，“怎么了？”
<br/>　　“有几个小家伙跑到我的世界里乱搞。”
<br/>　　“又不是什么大事，那算什么，你的世界一向如此，会吸引许多外来之人。我的世界也会如此。世界乱了也好，一尘不变可不是什么好事。”白袍道人听后，微微一笑，毫不在意道。
<br/>　　“也是，那只是我众多世界中的一个，也不是主世界，而是主界衍生出来的附世界。世界毁了，我在重新造一个便行了，到了你我这个境界，只要主世界存在，创造一个衍生世界只是随手而已。”青衣男子也毫不在意。
<br/>　　“听说你最近正在创造另一个主世界？”
<br/>　　“你是说圣墟界！”
<br/>　　“不错。听说起源之地要开启了，何时去碰碰运气，吾等已经不知多少年没突破了。”
<br/>　　“到了吾等这个境界，要突破可不是那么简单的事。”
<br/>　　“是啊，吾近来欲新创一界，我等论道一番，如何？”白衣人说道。
<br/>　　“固如所愿，不敢辞！”
<br/>　　……
<br/>　　于是，两个神秘大神继续论道，而混沌道君此时还不知道他已经被人发现了，他还是继续开始他的主神生涯。
<br/>　　……
