　　“大哥哥，前面那扇门，就是小鹿的家了。”
<br/>　　众人刚要出发，先前被藏在楼梯拐角处的小鹿突然开口，指着前方不远处的一闪防盗门，语气有些欢喜。
<br/>　　太小的年纪，根本考虑不到太多事情，当看到壁虎怪物的时候，众人都已经猜到，小鹿妈妈活下来的机会已经不大了。
<br/>　　除非是在感染的时候跑出去，但如果是这样，也该早些寻找小鹿才对。
<br/>　　要么就是在路上死掉了，要么就是感染开始的时候躲在楼内被吃掉了，无论是哪一个选择，都很残酷。
<br/>　　“我进去看看。”
<br/>　　向白小飞传递了一个眼神，刘哔上前几步，试着推动了一下，防盗门很轻易的就被打开了。
<br/>　　并没有关。
<br/>　　谨慎的走近屋内，里面没人，家具摆放都十分整齐，没有打斗过的痕迹。
<br/>　　随手抹了一把门口的鞋柜，摸到了一层薄灰，这个地方，已经有一些时间没人居住过了。
<br/>　　“进来吧，里面安全。”
<br/>　　确认四周没有什么危险之后，刘哔对外面招了招手。
<br/>　　率先跑进来的，是从白小飞怀中挣脱的小鹿，此时这个孩子，满脸迫不及待的表情。
<br/>　　迫不及待听到你妈妈的死讯么？
<br/>　　漫不经心的看了小鹿一眼，刘哔走近卧室，随手翻动着房间内的物品。
<br/>　　“妈妈……妈妈！”
<br/>　　“妈妈……小鹿回来了，你在哪里啊！”
<br/>　　门外传来小鹿焦急的喊叫声，搅的人心烦意乱。
<br/>　　放下了手中的薄皮本，刘哔走出了卧室，懒洋洋的依靠在客厅的沙发上。
<br/>　　“屋内没有打斗痕迹，东西摆放也很整齐，不过我在我始终发现了一些散乱的计划书，估计小鹿的妈妈，应该是在梳理工作的时候正好爆发病毒，然后匆忙逃了出去。”
<br/>　　简单的分析了一下，刘哔便不再理会此事，这个房间很普通，并没有发现异常的地方。
<br/>　　那个给他威胁的尸兄，并不在这里。
<br/>　　会在哪呢？
<br/>　　又会是谁呢？
<br/>　　在心中逐步推敲着，刘哔的眉头微皱，他感觉自己现在仿佛进入了一个困境。
<br/>　　线索还是太少了。
<br/>　　而且原著他看的时候已经太过久远，很多东西，都已经没有印象了。
<br/>　　“大哥哥，我妈妈呢？她在哪里啊？”
<br/>　　眼角含着泪花，小鹿跑到刘哔面前，表情十分委屈，语气听起来，让人感觉可怜。
<br/>　　“不知道，我不清楚在这之前发生了什么，所以无法给你答案。”
<br/>　　歪头想了一下，刘哔也没能想起小鹿母亲的信息，故此也没有想法再去思索了。
<br/>　　反正这些无关紧要的事情，让白小飞他们去搞吧，他没精力理会这些。
<br/>　　当然，也是他懒得寻找，这种事情，实在是太麻烦了，和他又没什么关系。
<br/>　　“叔叔，小鹿的妈妈呢？怎么看不到呀，妈妈……是不是不要小鹿了？”
<br/>　　在刘哔这边没有得到任何答案，小鹿转身来到白小飞面前，眼中的泪花，此时却再也控制不住，如同决堤之水一般落了下来。
<br/>　　白小飞低着头，不言不语，刘哔不知道，他这里难道又有答案了吗？
<br/>　　事实上，在进入这个房间之前，他们就已经猜到，小鹿妈妈已经离开了这里。
<br/>　　至于离开之后去了哪，是死是活，这也是他们无法知晓和控制的。
<br/>　　他又不是全知全能的神，充其量，他也只是走了狗屎运，变的稍微强大一些的人类罢了。
<br/>　　“叔叔说会找到妈妈的，叔叔骗人！小鹿不要和叔叔好了！”
<br/>　　见所有人都不说话，小鹿的心态瞬间崩溃，整个人躺在地上，本能的做着一些肢体动作。
<br/>　　在她单纯的思想里，只是觉得自己现在很伤心，而伤心，自然要有个发泄的地方。
<br/>　　小孩子的发泄，无非只是哭泣和无理取闹罢了。
<br/>　　保安小哥在旁边擦拭着枪支，没有说话，但目光却十分不忍。
<br/>　　身为大人，他想到的自然更多，在这样危险的情况下，小鹿的妈妈，基本没有可能幸存下来。
<br/>　　“小鹿……叔叔答应你的事情，就一定会做到！”白小飞抬起了头，眼神坚定，将小鹿抱在怀里：“不哭了，你的妈妈，我们一定会找的的！”
<br/>　　小鹿没有说话，小小的身体依旧在微微颤抖着，繁杂的啜泣声，哭的人心中如同织起了一团乱麻。
<br/>　　“走吧，在楼里找找，说不定，会有什么线索。”
<br/>　　刘哔站起身来，眼中闪过一丝不耐，在这里，他觉得已经耽搁了足够长的时间。
<br/>　　暗处还潜伏着一个强大的敌人，身处这样危险的地界，他可不认为呆在这里是一个明智的选择。
<br/>　　“也对，这样的话，我们出去寻找一下吧，对了小鹿，你妈妈有没有车？”
<br/>　　白小飞沉吟了一下，抬起头来，他倒是觉得，刘哔提出的建议十分不错。
<br/>　　就算没有找到，再不济也会找到一些线索，总比傻坐在这里等候要强很多。
<br/>　　“小鹿知道，我们家的车在地下二层，叔叔是想要开车吗？”
<br/>　　停止了啜泣，小鹿泪眼朦胧，声音中还带着一丝残留的哭腔。
<br/>　　“先去停车的地方看看吧，如果你妈妈还活着，汽车应该会被开走，这是一个很重要的线索，我们需要去确认一些。”
<br/>　　白小飞抱着小鹿，站起身来，随后，刘哔和保安小哥也纷纷起身。
<br/>　　这个想法不错，他们的确应该先去查看一下，这样会省下很多时间。
<br/>　　门外的电梯已经彻底停止运作了，众人想要下去，只能走步行楼梯，但是楼道内的那些变异虫卵，先前已经都被打碎，里面的变异蚊虫也都提前孵化了出来。
<br/>　　虽然刘哔放了一把火，但谁知道能不能将那些恶心的东西全部烧死。
<br/>　　这些东西攻击力并不高，可毕竟基数太过于庞大了，如果形成了蚊群，也是一个不小的麻烦。
<br/>　　更何况是在楼道的那种环境，一旦被包围住，逃走都很困难。
<br/>　　“诶，你们看，这个壁虎怪物的尸体！”
<br/>　　来到众人之前战斗过的地方，白小飞顿住脚步，指着地上那具庞大的褐色尸体，脸上满是震惊。
