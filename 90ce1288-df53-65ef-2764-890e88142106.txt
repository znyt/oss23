　　到了这个时候叶帝手中已经是法穿鞋、炽天使、时光之杖、法穿仗、冰心还有振奋这几件装备在手。
<br/>　　可以说这一波是叶帝状态最好的一波，不仅仅他的装而比做齐了，就连他的减冷却也是达到了峰值百分之四十五。
<br/>　　此刻他的所有技能的冷却时间差不多都只有一半，而这样下来的话，他释放起技能来也会更加的得心应手。
<br/>　　他现在的q技能的冷却时间仅仅只有2.6秒，每释放一个技能的话，去掉中间释放其他技能的间隔时间，理论上来讲他的流浪的q技能也就只有一秒钟而已。
<br/>　　一秒钟一个q技能，可想而知这个时候的流浪会有多么的恐怖。
<br/>　　在无数观众期待的目光中，下一刻扇子妈已经是开启了自己的大招，如同之前一样再一次来了一个群体加速，叶帝五人直接放弃大龙，向着对方的人群中冲了过去。
<br/>　　与此同时叶帝手中的流浪也是开启了自己的疾步，飞快的向着对方那边跑去。
<br/>　　一个闪现冲到了圣枪游侠的面前，这一次叶帝没有在利用q技能起手，而是直接w技能起手，随后qr两个技能近乎同时开启。
<br/>　　在大招过后，叶帝手中的e技能再次开启，几乎一瞬间他手中的q技能便是再次转好。
<br/>　　没有任何的间歇叶帝手中的第二个q技能在转好的一刹那便是朝着林萧这边的圣枪游侠的身上释放了过去。
<br/>　　从叶帝释放出第一个技能，到他这边第二次q技能释放出来，中间的时间没有超过0.5秒，而在这短短的一瞬间，林萧手中的圣枪游侠已经是倒地身亡。
<br/>　　甚至临死林萧这边都没有释放出来自己的水银饰带，叶帝的操作太快了，快到林萧这边都没有反应过来。
<br/>　　而且刚刚叶帝的那个跑速，让林萧有些跟不上来，他这边刚刚的画面一直停留在流浪那里，只不过流浪跑的太快了，短短的一瞬间便是消失在了镜头中。
<br/>　　等林萧将镜头调准到自己这边的时候，他的血线已经很低了，而且这个时候叶帝这边的最后一个q技能已经释放出来了。
<br/>　　“我的天，圣枪游侠被秒杀了，这流浪的输出也太恐怖了吧，这个速度也太快了！！！”看着这一幕，解说席上的米粒顿时震惊的张大了嘴巴，随后才开口喊道。
<br/>　　“后台刚刚给出了一个数据，大魔王从释放第一个技能，到最后一个q技能释放出来秒杀圣枪游侠，只用了0.46秒！！！”就在无数观众震惊的时候，林雅楠给出来的一个数据则是再次让无数观众瞠目结舌。
<br/>　　这虽然不是叶帝操作最快的一次，不过论华丽程度却是丝毫不比之前的那些操作差。
<br/>　　而且看到这样一个已经偏肉型的流浪还能够打出这么高的输出来，这也是这些观众震惊的原因。
<br/>　　“卧槽，流浪的输出这也太恐怖了吧。”
<br/>　　“666666666666，0.46秒秒杀圣枪游侠，这个流浪可是出了两个肉装的啊！！！”
<br/>　　“你们看圣枪游侠旁边的蜘蛛和娜美，他们居然也都被打掉了三分之一的血量，这这这...”
<br/>　　“这也太逆天了吧，估计霄神现在特别后悔，后悔自己这一局为什么不带一个虚弱过来制裁一下大魔王。”
<br/>　　“这下子大魔王真的起飞了，天赋风暴骑手被打出来了。”
<br/>　　直播间里的无数观众看到这一幕后，纷纷震惊的开口。
<br/>　　这一波叶帝倒是没有选择那一套法术机关前的打法，他是选择了最快速度秒人的这一套连招，因为叶帝知道，只要这一波自己秒掉对方的圣枪游侠，那么接下来五打四对面完全不会有机会的。
<br/>　　在叶帝冲过来秒杀掉林萧的时候，这一边秦霄的娜美也是反应过来，连忙在流浪的身前释放出了自己的q技能。
<br/>　　知道流浪的速度很快，所以秦霄特意向着流浪身前的位置释放的水泡，可是即使这样他还是小瞧了流浪的速度。
<br/>　　在娜美的水泡落地前叶帝手中的流浪已经是跑过了对方的水泡。
<br/>　　而蓝队这边其他几人的输出打在叶帝身上后，都是显得有些不痛不痒的，这个时候的叶帝真的是太肉了，不仅装备肉，而且他的大招还为他附带着吸血的效果。
<br/>　　加上振奋还能投提升吸血的能力，现在的叶帝简直就是打不死的小强，如果这个时候还有远古意志那件装备的话，那么这个流浪才真的叫做无敌。
<br/>　　在对方率先被击杀一人的情况下，而且在叶帝这边的q技能冷却时间非常短的情况下，最终这一波团战的结果显而易见，白队这边成功打出了一波1换5。
<br/>　　唯一的一个人头还是蒋念这边收下来的，只不过在这种后期的团战当中，他们这边少了圣枪游侠这个收割英雄，已经是无力回天了。
<br/>　　团灭对手后，看着对方的复活时间，这个时候叶帝等人也没在去选择攻击大龙，而是带领着中路的兵线直接向着对方的高地进攻过去。
<br/>　　对方的还有40多秒的复活时间，这些时间已经足够他们将蓝队的基地给击破了。
<br/>　　最终这一局比赛在时间到达40分钟出头的时候，白队一鼓作气直接推掉了对方的基地。
<br/>　　“让我们恭喜白队获得了这场bo3的胜利，这一场比赛我们再次见证了大魔王的传奇神话。”米粒激动的大声喊道。
<br/>　　“不愧是电竞鬼才，这份能力真的是厉害，大魔王用他的套路一次又一次的征服着这个赛场，相信用不了多久，流浪法师这个英雄肯定会成为全球最为火热的一个中单。”林雅楠跟着开口。
<br/>　　而场下的无数观众在看到最终的比赛已经结束后，十万名观众也是再一次齐齐的发出了一阵欢呼声、呐喊声，这欢呼不仅是给白队，也是在给蓝队。
<br/>　　为他们能够打出这么两场精彩的对弈个喝彩。
<br/>　　当叶帝一行人走出比赛室后，这欢呼声越发的热烈了起来，甚至几人的耳膜这一刻也是被震的嗡嗡作响。
<br/>　　现场观众热情可见一斑！
