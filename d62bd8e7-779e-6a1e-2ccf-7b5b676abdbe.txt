　　西域与漠北差不多，自古就是苦寒之地，到了冬天如果不在身上套一件皮裘，冷的简直能冻死人。
<br/>　　李道宗不是第一次来西域，薛仁贵在西域也不是第一年，所以他们还都能适应。但尉迟宝林和王玄策、白文墨以及那些新来的学生则是被冻的苦不堪言。
<br/>　　“叔，这帮混蛋啥时候能来齐啊，我这都快冻硬了。”尉迟宝林缩在李道宗的帐篷里面烤着火，喋喋不休的唠叨着。
<br/>　　大唐的军队来了，薛仁贵也交卸了自己的责任，把白文墨和那些学生统统交给尉迟宝林，让他给李道宗送过来。
<br/>　　必竟马贼的生活这些文人暂时还适应不了，尽管有一大批的学生想要留下，但考虑到他们在荒漠上独自生存的能力，薛仁贵最后还是没有答应他们的要求。
<br/>　　“西域这帮家伙平日里散漫惯了，你想让他们准时那是不可能的，所以等着吧，也许一个月之后就来齐了。”李道宗坐在火堆的另一面，慢条丝理的说着，眼睛在火焰的炙烤下微微眯着，看上去像是要睡着了一般。
<br/>　　“还要一个月？还要一个月就开春了吧，这帮混蛋这也太能磨叽了。”尉迟宝林惊讶的跳起来，想了想又坐回原来的位置：“不行，我得通知仁贵一声，让他那边给这些混蛋加加压力，让他们快点。”
<br/>　　“那么着急干什么，慢工才能出细活。”被火烤的有些发蔫儿的王玄策不急不缓的说道：“你总得让他们彼此之间有个沟通吧？不让他们商量个路数出来，只怕他们会团结起来对付我们了。”
<br/>　　“哎，我就说你们这些读书的花花肠子多吧，有啥弯弯绕绕的你就不能直接说出来，干嘛非得含含糊糊的打马虎眼。”尉迟宝林对王玄策自然不会像对李道宗那么客气，一时听不明白瞪起眼睛就开始嚷嚷。
<br/>　　“你小子消停点吧，不跟你是说是因为说了你也不懂。”李道宗一边示意亲卫往篝火中加些干木头，一边说道：“你小子过几天就回去，仁贵那边正缺人手，可没你闲着的份。”
<br/>　　王玄策也借机说道：“是啊，尉迟小将军，西域的事情需要几个方面互相好好配合，最后才能达到让太子殿下满意的局面，万万大意不得。”
<br/>　　这下尉迟宝林没脾气了，瞪着一双大眼珠子看看李道宗又看看王玄策，最后央央说道：“你们就是看俺人笨好欺负，得得得，一会儿我就回去成了吧，真是……”。
<br/>　　王玄策有些好笑的和李道宗对视一眼，好言相劝道：“一会儿可不行，这边有些事情还需要小将军来镇镇场子，所以三、五天之闪，小将军还真走不得。”
<br/>　　“又咋啦？让俺走的是你，让俺留的还是你，你到底想要让俺怎样！”尉迟宝林被越绕越懵，有些火大的从地上再次跳起来，气鼓鼓的问道。
<br/>　　“宝林！”李道宗见尉迟宝林已经快要有暴走的趋势，连忙将他喝住：“你先回去休息一下，晚上的时候有一个宴会，到时候记得过来，就这样吧。”
<br/>　　“知道了！”尉迟宝林摇摇大头，心有不甘的瞪了王玄策一眼，又冷哼一声，意思好像是：算你命大。
<br/>　　一直等到尉迟宝林出去之后，李道宗笑着转过脸：“王剌使，宝林性子憨直，还请不要见怪。”
<br/>　　“无妨，郡王客气了，玄策还要谢过郡王解围之恩呢！”王玄策对李道宗略一欠身，以示敬意。
<br/>　　刚刚如果不是李道宗把尉迟宝林打发出去，以王玄策的秉性定然不会相让半分，而最后吃亏的也一定会是他，所以他对李道宗道谢也是该当。
<br/>　　“你我同殿为臣，恩不恩的就不要提了。”李道宗大度的摆摆手，仿佛这对他来说只是一件小事，并不值得一提。
<br/>　　王玄策也是豁达之人，见李道宗满不在乎，也就不在多言，笑着转移的话题：“郡王，今日晚间，还请多加注意，能不开口最好不要开口，一切看玄策眼色行事。”
<br/>　　“没问题！某家一切听王剌使安排就是。”李道宗笑着点头答应。
<br/>　　从理论上说，他只是过来打个前站，要不了多久还是要回去的，西域这地方实在没什么搞头，就是要打仗也要等到三、五年之后，现在他在这里完全没有必要。
<br/>　　而之所以让他过来，实际上也不过是作作样子，让西域诸国看到大唐对他们的重视，至于任务……除了配合一下王玄策、薛仁贵这些小辈之外，其他也就没什么了。
<br/>　　作为一个郡王，李道宗知道，他这一辈子基本上也就这样了，也没有什么百尺竿头更进一步的可能，再大的功劳对他来说都已经没什么用与，与其争功不如把这些功劳拿出来大家分，落个照顾小辈的好名声。
<br/>　　王玄策也正是看出这一点，所以才敢在西域放手施为，打算把西域搅个天翻地覆。
<br/>　　高昌、龟兹、焉耆……，这些西域三十六国中的强国，都在必须被剿灭的名单之上，大唐太子不想让他们继续存在下去，那么他们就没有存在的必要。
<br/>　　王玄策和包龙图一样但也不一样，老包的‘天下大同’是指想让大唐所有人都有受教育的权利；而王玄策的‘天下大同’是指让大陆上所有国家共沐大唐光辉。
<br/>　　所以王玄策并不排斥李承乾的安排，能在异国他乡搞风搞雨，又没有后顾之忧正是他喜欢的。
<br/>　　是夜，高昌王城显得分外热闹，提前到达的各国使节纷纷现身于此，为迎接大唐使者做准备，但是似乎每一个人脸上笑的都是那么虚伪，甚至可以说是狰狞。
<br/>　　马贼抢他们的东西，杀他们的子民，这让他们感到恐惧，所以他们想到了大唐，想到了东面那个庞然大物，所以他们请大唐请来了，想让大唐这头‘猛虎’把家里的‘狼’赶走。
<br/>　　但是，最后他们发现，似乎一切和想的并不一样，当初盘算着利用大唐和马贼拼个你死我活，到时候大唐与马贼两败俱伤，他们就可以坐收渔翁之利。
<br/>　　可是马贼太不争气了，在被大唐军队抓了几个人之后，竟然调头跑了，远遁千里。
<br/>　　这下问题就尴尬了，如果让唐军回去吧，西域诸国担心马贼再杀回来；可如果不让唐军回去吧，这特么养活三万唐军，似乎比以前养活一万五千马贼还要破费。
