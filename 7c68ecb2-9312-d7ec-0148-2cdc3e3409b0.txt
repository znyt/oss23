　　第576章
<br/>　　时间真的能改变很多事情。孟玄青自己也记不清有多长时间未与别人动过手。
<br/>　　这么多年过去了，他用毒的功夫还没放下，可与人交手对战，不管是体力还是反应都下降许多。
<br/>　　所以，一交手他便被李莫愁完全压制，一杆拂尘甩开，可柔可刚，打得孟玄青只有招架之力，而无还手之功。
<br/>　　夫子在一旁压阵，看到这一幕吓了一跳，果然是唯女子与小人难养也。他在琢磨着自己要不要插手，或者是如何寻找一个合适的机会插手。
<br/>　　老夫子遇上刁婆娘，是完全没有道理可讲的。
<br/>　　孟玄青被压制得无法还手，不得已只能用阴损的招数了。作为一个职业是巫医的高手而言，与人近战动手的确不是他所擅长，他的阴损招数就是下毒。
<br/>　　袍袖轻甩，从袖口中窜出一条银色小蛇，呲呲吐着蛇信，向李莫愁飞来。
<br/>　　这些日子，孟玄青又养了一些毒物，身上藏得暗器，不仅这条毒蛇。假若被毒蛇咬中，李莫愁一条命也就交待在这里了。
<br/>　　她的反应是大骇，立刻抽身后退，守在李婉儿身边。
<br/>　　李婉儿如今仍然昏迷不醒，却怎么知道周围已打得难解难分。
<br/>　　与李莫愁分开后，孟玄青也同样松口气，他这把老骨头可经不起这样折腾。
<br/>　　那银蛇呲呲向李莫愁游来，李莫愁用一枚银针送它走了。立在那里站定，目光警惕的盯着周围。
<br/>　　有几个书院弟子向李莫愁扑来，连环几枚毒针送他们见了阎王。夫子心忧不已，请孟玄青出手医治。孟玄青用毒是个好手，救人也不在话下。可那里还救得活，这银针见血封喉，死者脸黑得如焦炭一般，根本不给孟玄青展现本事的机会。
<br/>　　李莫愁站定，也不惧有谁攻来。她目光遥遥望着程大雷，心里其实还不明白今日是怎么会事。
<br/>　　程大雷看见了高处的李莫愁，心里也松了口气。毕竟也是个绝世高手，这样算下来，己方就有三名绝世高手了。
<br/>　　当然，燕不归是敌是友还说不好。
<br/>　　这个时候，燕不归已经在与张六的战斗中，完全取得下风。纵然张六有多名好手助阵，可也敌不过复仇心切的燕不归。
<br/>　　仇人就在眼前，怎能放过他。
<br/>　　“杀呀。”
<br/>　　口中大吼一声，双手握着重剑狠狠劈下。张六刚才已经被逼到绝地，手中的大棍早被斩成两段，现在他退无可退，挡无可挡，只能看着头顶的重剑扑面而来。
<br/>　　咔嚓。
<br/>　　骨头碎裂的脆响，张六的身体被从中间劈开，血肉流了一地。
<br/>　　周围有不少人都吓得胆战心惊，他们虽然也杀人，但终究还是有些帝国读书人的体面。这样残忍霸道的杀人方式，却是第一次亲眼目睹。
<br/>　　燕不归溅了一身血，然后他仰头大笑，只感觉一腔快意。
<br/>　　小师妹，你的仇我终于为你报了，现在你可以安心上路了。
<br/>　　“杀呀。”
<br/>　　一柄长剑捅入了他的肩膀，出手的只是书院一不知名的小弟子。这一剑本不该伤到燕不归，可处于癫狂中的他，竟没有任何防备。
<br/>　　啊呀。
<br/>　　口中大吼一声，一剑将这人的脑袋斩了下来。燕不归已经处于疯狂中，只觉得眼前人人可杀，人人都该为他的小师妹抵命。
<br/>　　一人拼命，十人难敌，用一柄重剑开路，当真是没人拦得住他。
<br/>　　然后，他看到了被剑阵包围的程大雷。
<br/>　　想不注意到程大雷也并不容易，他骑着一头黑牛在剑阵中左冲右撞，嘴里还骂骂咧咧。
<br/>　　然后，燕不归的旧恨又想起来了。既然天下人人都该死，那程大雷自然是一万倍的该死。不杀他不能告小师妹在天之灵，不杀他不能消我心头之恨。
<br/>　　于是他开始闯阵，看上去像是要破阵。
<br/>　　主持大阵的宋寒之皱起眉头，果然，这两个魔头是一伙的。他下令大阵运转，放开一条路让燕不归进去，既然今天两个魔头都来了，那就一块儿杀了吧。
<br/>　　程大雷心头感激莫名，什么叫敌人的敌人就是伙伴，他可不歧视燕不归是个戎族，只要对方不歧视自己的职业就好。
<br/>　　二人都很强，如果联起手来，未必不能破阵呐。
<br/>　　“狗贼，去死吧。”
<br/>　　闯入剑阵的燕不归，大吼一声向程大雷砍来。
<br/>　　程大雷愣住了，所有人都愣住了。二人不该是联手破阵么，怎么互相打了起来。
<br/>　　宋寒之稀里糊涂，但这终究是好事，他下令按住阵脚，向让二人狗咬狗，最后再送他们一块儿下黄泉。
<br/>　　程大雷感觉有些头疼，随之他确定一件事，这燕不归当真疯了。
<br/>　　如果不疯，怎么有可能单枪匹马闯书山，如果不疯，怎么明知送死还来这白沙岛。
<br/>　　要知道，如果有得选的话，这两个地方，程大雷是一个也不想来。
<br/>　　可惜他没得选，就如同现在，他同样没得选。燕不归要杀他，他便只能杀了燕不归，这便是所谓的以杀止杀。
<br/>　　二人在剑阵中央厮杀，重剑对扛程大雷的匹夫剑，打法完全不同。程大雷占了伶俐快捷，燕不归占了势大力猛。
<br/>　　那本也是程大雷擅长的打法，只是手中的兵器实在不趁手。
<br/>　　打起来便知道，程大雷半点不输燕不归。燕不归之前便受过重伤，虽然如今勉强能支撑着战斗，可是那样严重的伤势，肯定不似从前。况且，他现在俨然有走火入魔的趋势，疯狂令他悍不畏死，但不代表他不会死。
<br/>　　程大雷轻而易举便取得了上风，在连环快剑的攻击下，燕不归根本施展不开自己的本事。这就是高手过招，哪怕一丁点微弱的优势，也能滚雪球般形成完全碾压的姿态。
<br/>　　然后，燕不归开始逃。
<br/>　　这一幕，倒是大大出乎程大雷的预料。因为看燕不归的样子，不该打不过就逃啊。
<br/>　　燕不归本来没准备逃，他是要去杀人。
<br/>　　今天岛上为何有这桩事，他已打听明白。不就是书院擒了程大雷女人，借此要挟么。
<br/>　　所以，他也要去杀掉那个女人，让程大雷明白痛失心爱之人的滋味。
<br/>　　如此说来，燕不归确实疯了，他报复得不再是某个人，而是整个世界。
<br/>(本章完)
