　　周宋捷张了张嘴，最终还是没有说什么，只是用手拍了拍刘东来的肩膀，离开了行动简报室。周宋捷不是军人出身，他也不了解军队，所以他没办法多说什么，也没资格多去评论什么。
<br/>　　两者都是国家的暴力机器，只能说，他们各自的领域不相同。
<br/>　　杭山市市局那边替龙隐部队战术中队安排了一间独立的房间给他们使用，从行动简报室离开的时候，罗昊就注意到慕容飞有心事，之前不方便细问，罗昊也就没有挑破，现在房间里除了战术中队，也没有其他人，罗昊便问起慕容飞情况，今天晚上就会对李霄力采取行动，慕容飞作为重要的远程火力点需要给突击小组提供火力支援和压制，在行动中，罗昊不希望出现任何的意外，因为任何的意外都有可能导致自己身边的兄弟牺牲。
<br/>　　“刚才神秘失踪的钱鑫我见过。”慕容飞说道：“他就是千雪那个小区的保安，我回来那天他还拦着我，质疑我的身份；后来几次我陪千雪出去散步的时候，也见过他几次，我总觉得他看千雪的眼神有些奇怪，在他的眼睛中藏着东西。”
<br/>　　“会不会是你多疑了？”罗昊说道：“这个钱鑫的背景很普通，父母都是工薪阶层，虽然在社会上混过，但那些社会关系很简单，并不具备什么威胁。”
<br/>　　“也许是吧。”
<br/>　　慕容飞摇了摇头，甩开自己脑子里那些乱七八糟的想法，不再纠结这件事情。
<br/>　　“多疑，看谁都是恐怖分子，职业病。”
<br/>　　司徒空用磨刀石打磨着自己手中的飞刀，嘿嘿一笑。司徒空手中的飞刀，每把长约五寸，两指宽，刀柄末端带有一个指环，可以用做暗器，也可以用作近战武器。司徒空的刀囊固定在双腿外侧，左右各三把，总共六把；而原本应该出现在大腿外侧的自卫手枪则是被他以斜四十五度角固定在了战术背心上。
<br/>　　“行动结束后，给你两个小时的自由时间，不放心就回去看看。”罗昊对慕容飞说道。
<br/>　　“谢谢。”
<br/>　　“说下这次行动。”罗昊把一张港口的平面图铺在桌子上，说道：“港口外围，市特警队会进行布置，会封锁住所有路口，我们要做的就是防止李霄力跳海逃跑，以及注意货轮上船员的安全，尽可能避免误伤。行动中，我跟冰刃各带一支攻击小组，分别从船头和船尾接近，狐狸和无常注意监视周围情况，公主负责情报支援。这次行动没有什么难度，但是我需要重申一点，一定要留下李霄力的活口，他是我们这次行动的目的，他可以被打伤，但是绝对不能被打死！”
<br/>　　……
<br/>　　杭山市，李霄力名下一家酒吧地下室。
<br/>　　桌子上摆放着三把土制猎枪，三把“化隆造”，若干子弹。
<br/>　　刘三从桌子上拿起一把七成新，套筒被拉开，没有弹匣的“化隆造”扣动扳机，“啪”，拉开的套筒被复位。
<br/>　　“准备的怎么样了？”
<br/>　　李霄力走进地下室，对刘三问道：“人员和家伙都准备好了吗？”
<br/>　　“放心吧，力哥，一切准备妥当，只要货轮进入港口，我们就能进行接货。”刘三举起手里的“化隆造”对着刘三晃了晃，显得非常有信心。
<br/>　　李霄力勾住刘三的脖子，把他拉到一边，问道：“三儿，最近条子那边什么情况？”
<br/>　　“还在盯着我们那几个场子，就连健身中心那边都在盯着呢。”刘三有些不解地对李霄力问道：“有什么问题吗？”
<br/>　　“不知道。”
<br/>　　李霄力揉了揉自己的眉心，说道：“我这两天眼皮总是跳个不停，心里总觉得不踏实。这次接完货，我先出去避避风头，等我回来以后再处理这批货。我不在的这段时间，你帮忙盯着点，我觉得条子早晚要对我动手，我跟他们合作过，多少对他们有点了解，最近可能不太平。”
<br/>　　“放心力哥，我会注意的。”刘三点头回答道。
<br/>　　李霄力对着刘三挥了挥手，让他继续去忙，扭了扭脖子，发出“嘎啦嘎啦”的声响，心中的不安更甚。他当过警方的内线，帮助过警方打掉了自己原来老大的势力，所以他对警方那套路数还是熟悉的，这也是为什么他在杭山市待了这么多年，做了这么多事都没有被警察抓到线索的关键所在。
<br/>　　……
<br/>　　进入秋季之后，天也黑得比夏季的时候早，刚过六点钟，天色就已经暗了下来，马路两侧的路灯亮了起来，拥堵在马路上的车辆也纷纷亮起了汽车的示宽灯，红色的灯光连成一串，构成一幅大城市夜间最常见的画面。
<br/>　　环形高架上，汽车排成长队向上挪动着，如果坐着直升机从高处往下看去，能看到环形高架上的汽车呈螺旋状向下延伸，一直延伸到地面路段，即便是没有驾驶着汽车挤在这看不到头的车流中，也会在心中升起一股无力感。
<br/>　　天空中一架杭山市特警支队的直升机，呼啸着飞过，直奔港口的方向而去。此时坐在直升机中的，不是杭山市的特警支队，而是齐凡所带领的第二攻击小组；罗昊所带领的第一攻击小组，则是坐着一辆用轻客改装而成的指挥车，从地面朝着港口驶去。
<br/>　　指挥车周围并没有开着红蓝爆闪灯，威风凛凛的特警车开道，指挥车车厢内，罗昊正在往自己脸上涂抹着伪装迷彩，云熙双手则是在键盘上快速敲打着，过了一会儿，云熙停下手上的工作，说道：“港口附近的监控探头已经被全部接管，现在接入画面。”
<br/>　　云熙在键盘回车键上轻轻一敲，港口附近的画面立刻传送到了指挥车中。罗昊抬起头看向指挥车中被分割成好几块的屏幕，他的注意力被其中一块监控探头的画面所吸引，指了指引起自己注意的监控探头拍摄到的画面，对云熙说道：“把那辆五菱宏光面包车放大。”
