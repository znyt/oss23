　　大竹峰驻地内，苏茹盯着一众弟子没让他们乱跑。山下灵尊明显是冲着自己两个小徒弟来的。田不易没回来之前她不放心。
<br/>　　等田不易御剑飞来，苏茹立刻就把众弟子召集起来。
<br/>　　“不易，灵尊到底因何异动，掌门师兄可是有了定论。”她怕牵连到自家徒弟。
<br/>　　“没事，掌门师兄也没说出个所以然。”田不易现在可不像刚才一样轻松。灵尊异动，他也犯嘀咕。
<br/>　　看到站在角落出生的苏重，田不易心中一动：“尔旦，庚金剑气没什么问题吧。”
<br/>　　他突然狐疑起来。庚金剑气他当时确实研究过，也没看出什么问题。但关键是它的出身有问题啊。
<br/>　　这东西来自魔门，如果真有什么诡异之处，那不是坑自己徒弟吗？田不易不由紧张起来。
<br/>　　苏重面色如常：“没问题，一点儿问题都没有。”
<br/>　　没问题？问题大了去了！
<br/>　　苏重现在知道灵尊为什么围着他打转了。他修炼的庚金剑气，吸收的可不是异种金气，而是实打实的凶戾气。是天地间凶邪能量。同时也是诛仙剑阵的驱动能量。
<br/>　　灵尊作为诛仙剑阵一部分，不止一次见识过阵法启动。对凶戾之气最熟悉不过。
<br/>　　苏重练得是凶戾气，用的是天机印。一切都和诛仙剑阵大有关系，水麒麟怎么可能无动于衷。
<br/>　　“没问题就好。”田不易心下一松，突然想到一个问题：“对了。我怎么没见你吸收金气啊？”庚金剑气的不是吸收金气入体的吗？那可得消耗大量金属。可苏重从未向他索要过啊。
<br/>　　苏重浑身一僵，自家这胖师傅可真够敏锐的。
<br/>　　“原本功法那是人能练的吗？”苏重佯装抱怨：“如果不做修改，我能不能站在这里还是两说呢。”
<br/>　　田不易顿时讪讪，也忘了纠结金气的事情。自家徒弟不能修炼玄清道，自己就给他找了这么一门折磨人的功法。饶是田不易脸皮厚如城墙，依旧感觉赧然。
<br/>　　“那什么，明天你们就要参加七脉会武。一定要全力以赴，打出我们大竹峰的气势来。以前你们打不过龙首峰我不怪你们，但现在你们有聚元功练体，有黑竹林阵法辅助修炼。要是还打不过龙首峰，什么后果你们自己去想吧！”田不易摆起脸色狠狠呵斥一众弟子。
<br/>　　刚才在大殿上怼苍松怼的那么爽，还不是因为自家徒弟争气。徒弟们这次要是能力压龙首峰，那怼起苍松来岂不是更爽！
<br/>　　“尔旦啊，明天好好发挥。你没有法宝不能御物，明天一定不要勉强，保全自身为重。这次来啊，主要就是涨涨见识哈。”田不易对着苏重和蔼可亲的嘱托。
<br/>　　一众弟子看像苏重眼神满是幽怨，师傅这区别对待越发严重了。
<br/>　　“小凡也是，比赛第二，参与第一。长见识长见识哈！”
<br/>　　田不易笑呵呵，最近他也发现了，自家两个小徒弟每一个都不简单。
<br/>　　小徒弟妖孽他早知道，没想到木讷寡言的张小凡也资质不凡。有黑竹林帮助，他比以前更加关心徒弟修炼。这一关心就发现，张小凡竟不知不觉就能御物了！比老六杜必书快了不止一筹！
<br/>　　“这草庙村真是人杰地灵啊，三个娃娃一个比一个妖孽！”
<br/>　　又耳提面命训斥了众人一顿，在众弟子怏怏不乐的表情中才收住好为人师情绪，打发众人各自休息。
<br/>　　……
<br/>　　第二天，众人吃过早饭离开驻地，前往通天峰大殿抽签。
<br/>　　青云七脉会武弟子众多，用抽签来选择比赛对手。
<br/>　　苏重跟着中师兄上前，随便抽了个签子也不在意。给众位师兄送上祝福后，苏重就独自一人前往自己的擂台。
<br/>　　不像其他峰弟子众多，除了参加比赛的人，还有大票凑热闹弟子。一个人比赛，总有其他师兄弟跟着打起加油。
<br/>　　可大竹峰上下大猫小猫两三只，不只是苏重，他们所有人都得独自去擂台比武。
<br/>　　按照抽签决定，苏重顺着沿途各种标志，很快就到了自己的擂台。
<br/>　　这时候擂台上已经开始打起来了。
<br/>　　两个弟子都是御物境界，操纵飞剑隔空劈砍，倒是打的似模似样。
<br/>　　太极玄清道是青云根本道法，没达到御物前，只能改善体质增强精神，可一旦能御物。实力立刻就会发生翻天覆地变化，配合各种法诀威力无穷。
<br/>　　台上两人也就刚刚会御物，可配合各自飞剑，竟也打的有声有色。
<br/>　　很快就轮到苏重上场，对手是一个白衣青年，对方遁光一闪就飞上擂台。
<br/>　　这是青云弟子达到御物，并找到法宝后学习的第一法诀——御剑飞行！
<br/>　　一道遁光飞遁千里，端的潇洒迅捷。可惜苏重不会，他只能发力跳上擂台，就和大蛤蟆似得。
<br/>　　地下一阵哄笑，只这一下就显示出了苏重境界。
<br/>　　“师弟，在下朝阳峰段珪。”白衣青年彬彬有礼温和笑道。
<br/>　　苏重伸手不打笑脸人也笑着拱了拱手：“大竹峰李尔旦。”
<br/>　　底下又是一阵哄笑。实在是尔旦的谐音太接地气。
<br/>　　“师弟请出手吧。”段珪却没有嘲笑的意思，依旧不急不缓。他也看出来了，眼前这位师弟修炼时间不长，明显没达到御物境界。
<br/>　　而且大竹峰弟子是出了名的少，这大概也是来长见识的。
<br/>　　“师兄误会了，我虽然没达到御物境界，但修炼功法比较特殊。师兄还是先出手吧。”苏重也不想搞突然袭击。
<br/>　　段珪脸上依旧带着笑，只当年轻人傲气作祟，拔出腰间长剑笑呵呵道：“好好好，咱们也别谦虚了，一起动手吧，你看我剑都拔出来了，来吧。”他还是存着照顾师弟的意思。
<br/>　　苏重无奈，这不是自己要搞偷袭啊。都告诉你自家法诀奇特了，怎么还这么大咧咧。他是看出来了，对面这个温和青年，估计就是个好好师兄。
<br/>　　不然也不会有那么多师弟来给他加油，想来人缘不差。
<br/>　　“师兄小心，我这法诀攻击速度可不慢。”
<br/>　　“没事，来吧来吧。”段珪不以为意。
<br/>　　苏重无奈一笑，伸手一点，一道乳白剑气骤然飞出好似瞬移般突兀出现在段珪眼前。
<br/>　　不等对方反应，剑气骤然炸开，一股狂风突起。刚才还老神在在的段珪，一个跟头就给掀翻下擂台。
<br/>　　被众多师兄接住后，段珪还有点儿反应不过来。
<br/>　　“我怎么下来了？”他有点儿迷糊。
<br/>　　“嘿嘿，师兄你的当然是被人家打下来的啊。”一个小个子师弟笑嘻嘻调侃自家师兄。
<br/>　　段珪这时候也回过神来：“师弟，你可真是深藏不漏啊。”段珪拱拱手无奈苦笑。
<br/>　　苏重还能说什么，只能露出一个抱歉笑容。他这还没用聚合剑气呢！
<br/>　　庚金剑气除了锋锐无比，还沾着一个快字。锋锐迅捷，正是庚金剑气特征。
<br/>　　段珪不出手，他就在没有机会出手！
