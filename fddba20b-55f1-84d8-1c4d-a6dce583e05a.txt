        &lt;h3&gt;第三章第四十四章  马总与夏颜&lt;/h3&gt;
<br/>
<br/>        马总挂了电话，看了一眼对面的夏颜。
<br/>
<br/>        他和夏颜的母亲是认识的，之前有过工作上的交集，夏颜的母亲帮过他。一来二去他们的关系也不错。
<br/>
<br/>        夏颜调到西城，靠的是一个姓冯的关系。
<br/>
<br/>        夏颜的母亲和姓冯的关系不错，据说是校友。而夏颜当初考大学的时候都是他们几个利用关系促成的，至于夏颜取代了谁，他们几个都不知道，知道的是班主任和当初在现场的人。
<br/>
<br/>        唐红莲是个人精，不但自己的工作干得出色，连女儿夏颜的工作也安顿得妥妥的。如今的夏颜，也是西城一个信息部门的老总，工作能力有没有没有关系，只要关系好就行，再说，夏颜当好老总就行，专业知识有工作人员也就可以了。
<br/>
<br/>        “颜颜，我马上就要调到西城了。”搅动着手中的咖啡，马总说。
<br/>
<br/>        马总快五十岁的人了，看起来也就四十多一点的样子，身体结实，精明能干，一头曲卷的头发茂密的覆盖在头顶，眼睛透着精明的光，只是眼圈发黑，一副麋x的模样。
<br/>
<br/>        “恭喜你啊，高升了，你的那个陈子乔也跟着一起过来吧？”
<br/>
<br/>        “嗯，她也过来呢。”
<br/>
<br/>        “你们俩个人形影不离啊。还真是好搭档。”
<br/>
<br/>        “工作上的事情，颜颜你别胡想。”
<br/>
<br/>        “那这一次怎么没有跟着你一起来？”
<br/>
<br/>        “你老公傅海生单位不是要做信息化建设吗？她弟弟和妹妹就是做这个生意的，这几天在洛城呢。”
<br/>
<br/>        “傅海生单位要搞信息化建设？”
<br/>
<br/>        “啊，你不知道？”
<br/>
<br/>        “没听他说，我们最近都太忙，没有时间联系的。”
<br/>
<br/>        “你们这种生活也很辛苦啊。”
<br/>
<br/>        “习惯了，对了，陈子乔的妹妹叫什么名字？是哪个公司的？”夏颜知道陈子乔是陈子昂的四姐。
<br/>
<br/>        “妹妹叫陈子昂，弟弟叫陈子霖，说是从北京回来，在西城也有分公司。”
<br/>
<br/>        “你说什么？陈子昂？”夏颜站起来。
<br/>
<br/>        “怎么了颜颜？这么激动？”
<br/>
<br/>        “陈子昂是我同学。”夏颜又坐下去了，夏颜看着窗户外面的太阳，有些刺眼。
<br/>
<br/>        陈子昂已经这么多年没有消息了，傅海生在梦里都会叫着她的名字，如今他们见面了会是一种什么样的情况？
<br/>
<br/>        陈子昂就是她和傅海生心里不可触碰的一根刺，谁想起来都难受。
<br/>
<br/>        她也曾经在梦里梦到过陈子昂那阳光明媚的笑脸，还有陈子昂和她在一起的快乐时光，她们在一起度过了一段愉快的时光。
<br/>
<br/>        陈子昂善良，没有心机，是学霸，人也好，走哪里都是浑身闪着光，耀眼得像外面的阳光，让她感觉到刺眼。
<br/>
<br/>        和陈子昂在一起，她感觉她就是一个丑小鸭，就像是电视剧里面的配角，她想当主角，她也要像她一样光芒四射。四班有个男生暗恋陈子昂，让她给陈子昂捎信，她就将信藏起来，并且偷着拆开看。
<br/>
<br/>        她经常给四班的那个男生捎信，但是信都到不了陈子昂的手中，她看着写给陈子昂的信，竟然莫名的喜欢上了四班的那个男生。
<br/>
<br/>        她就借口陈子昂约会那个男生，陈子昂什么都不知道，对那个男生就冷冷淡淡的，她和那个男生好/上了。本来学习不怎么样，俩个人早恋了更是影响学习，被妈妈发现了。
<br/>
<br/>        “颜颜，想什么呢？”马总打断夏颜的回忆。
<br/>
<br/>        “想我同学呢，陈子昂你见过吗？”
<br/>
<br/>        “见过，很漂亮的一个美女呢。”
<br/>
<br/>        “她一直都很漂亮，不但漂亮，学习也好。”
<br/>
<br/>        “难怪能去北京发展，你们是大学同学？”
<br/>
<br/>        “高中同学，她没有考上大学。”
<br/>
<br/>        “那还是学习不好，比你们还差点啊。”
<br/>
<br/>        “比我们都好。”夏颜低着头使劲的搅动着手中的咖啡，她现在的人生其实应该是换陈子昂的。
<br/>
<br/>        “你妈妈还好吧？”马总并没有在陈子昂的学习上纠结，转变了话题。
<br/>
<br/>        “还行，就是一门心思想往上爬。”夏颜突然就笑了一下，她妈妈心里只有两件事，工作和女儿，现在都多大年龄了，还在不停的折腾着。
<br/>
<br/>        “哈哈，现在正是黄金好时机啊。”
<br/>
<br/>        “女人五十五岁就退休了。”
<br/>
<br/>        “那人家还能奋斗几年的。”马总的年龄其实和唐红莲差不多大小。
<br/>
<br/>        “忘记了，你和我妈妈年龄相仿。”夏颜嘲讽的一笑，她怎么都想不到她堕落得和一个可以当他爸爸的人有过纠结，而且这个人还是她妈妈的朋友。
<br/>
<br/>        记得那一次和傅海生吵架，扔了傅海生写给陈子昂的心之后，心了觉得轻松了很多，这么几年背着那个秘密，压得她透不过气来。
<br/>
<br/>        她全心全意的和傅海生在一起，可是傅海生心里一直想着陈子昂，让她恨得牙根痒痒，却又无可奈何。
<br/>
<br/>        陈子昂有什么好啊？
<br/>
<br/>        可是傅海生和他妈妈的态度真是令她伤透了心。毕竟她和他们生活了几年了，怎么在他们心中比不过一个陈子昂呢？
<br/>
<br/>        她一气之下就回了娘家，妈妈不在家，她却迎来了马总。
<br/>
<br/>        马总看着她红红的眼睛关切的问：“颜颜，你怎么了？谁欺负你了？”
<br/>
<br/>        马总关切的话语使夏颜更是泪如雨下。
<br/>
<br/>        马总拿起纸巾给夏颜擦着眼泪，夏颜扑在马总的肩膀放声痛哭。
<br/>
<br/>        她哭哭啼啼的说：“我和傅海生吵架了。”
<br/>
<br/>        “夫妻之间吵架是常事情呢，哪有不吵架的。乖，别哭了。”马总轻拍着夏颜的背，柔声说。
<br/>
<br/>        “一句：乖，别哭了。”让夏颜沉沦了。
<br/>
<br/>        她自小没有了爸爸，和妈妈生活在一起，妈妈其实是很强势的，每次摔跤都是严厉的让她自己爬起来，从来都不问她疼不疼。
<br/>
<br/>        自从没有了爸爸，妈妈也不再婚，但是会经常换男朋友。
<br/>
<br/>        她吃穿不愁，花钱不愁，可是她总觉得缺什么。
<br/>
<br/>        突然之间，马总的肩膀是那么的可靠，她就想那样靠着不离开。
<br/>
<br/>        夏颜后来才知道，她是缺少安全感，缺父爱。
<br/>
<br/>        她对马总的心理产生了微妙的变化，她知道这样不好，可是在马总这里她得到了以前得不到的东西。
<br/>
<br/>        隔三差五的他们都会见一次面。
<br/>
<br/>        马总就像是一个父亲一样对她嘘寒问暖，偶尔也会给她惊喜，刚来西城的时候，马总就送了一套小户型的房子给她，房子是她的名字。
<br/>
<br/>        在西城这个寸土寸金的地方，她一个从小城市里出来的，有房子就是身份想象征。
<br/>
<br/>        原来在泉城，俩个人见面还要躲躲藏藏，来了西城之后，便也没有了那么多顾忌，俩个人会一起看电影，进咖啡厅，逛商场。
<br/>
<br/>        她想要买的东西马总都会给她买。
<br/>
<br/>        夏颜看了一眼对面坐着的男人，心里莫名的涌起一丝悲哀。
<br/>
<br/>        她爱了傅海生很久，和她竟然不及这个男人亲密。
<br/>
<br/>        人都是会变的！
<br/>
<br/>        总有一些人，在道德的边缘试探着，扭曲的人性散发着腐朽的气息隐藏在看不加的犄角旮旯里。
<br/>
<br/>        夏颜深深的叹了口气！
