　　第385章
<br/>　　夜深人静，欢宴散去。
<br/>　　众人在酒足饭饱之后，来到给他们安排的住处。琴川关有联排的民居，想要在短时间内给一百余人安排好屋子，在任何地方，都不是一个简单的事情。好在如今的琴川关非比从前，执行力还是蛮高的，只用一个下午时间，便布置好一切。屋内有崭新的被褥，简单的陈设，甚至水缸里都装满了水。
<br/>　　从长安一路过来，每个人都已是筋疲力尽。今天大家又喝了太多酒，熬到好晚。可是，诸人却没有任何睡意，心事重重的等着阿喜三人回来。
<br/>　　这帮人看上去是大老粗，不拘小节，豪横得紧。可如今毕竟是生存问题，诸人不能不小心。不远千里来到琴川关，程大雷究竟是什么态度，这一切只有等阿喜回来，才能又最终答案。
<br/>　　阿喜背着双手，踱步回来，推开自己的屋门，见有客厅、卧房、厨房。这样的布局，自然称不上豪华，可却已经是简单干净。
<br/>　　他给自己点火烧茶，一壶水还没有烧开，门就被别人推开了。
<br/>　　“怎么样，老大怎么说的？”
<br/>　　“急什么？”阿喜轻咳一声：“我陪老大说了很多话，口干舌燥，现在喉咙还上火呢，你们能不能等我先润润嗓子。”
<br/>　　屋子里已经挤满了，粗略有十几个，阿喜的小屋子都快装不下。
<br/>　　“您就别端架子了，老大究竟是什么态度，打算怎么安排我们的？”
<br/>　　“老大说了，让大家先住下来，以后做什么，就看各人本事。咱们过去是兄弟，老大能够不管我们么，呵，笑话。”
<br/>　　“老大真是这么说的？”
<br/>　　“怎么，你们信不过我，还信不过老大么。”阿喜清了清嗓子道：“来的路上我就说过，让你们把心都放进肚子里。老大是什么人，他从前现在，又有那次对不起我们。”
<br/>　　众人无语，心道，来的路上，好像是你最担心吧。
<br/>　　“嗨，我就知道老大，不是能忘了兄弟的人。”
<br/>　　“咱们跟着他，还能错的了，我说你们也真是，瞎担心。”
<br/>　　“好啦，好啦，都回去睡觉吧。”阿喜摆摆手：“对了，还有一件事，今时不同往日，老大如今是家大业大，不能再像以前，究竟能不能被重用，还要看大家的本事。到这里，咱们只有真正出得上力，其他人才能看得起我们。”
<br/>　　……
<br/>　　书房内，程大雷可以说是一团乱麻。
<br/>　　阿喜带过来的这些人，足有上百个。程大雷对他们是熟悉的，其中大部分人，都是江湖上的旁门左道，有小偷、骗子、赌徒、盗墓贼等等之类的。即便有几人身强力壮，会几招江湖上的把式，但真正战场上的厮杀，却未必适合他们。
<br/>　　他们毕竟不是战士。
<br/>　　阿喜有谋略之才，倒是可以放到和珅手中历练，飞天蛤蟆白元飞的轻功不错，能够做一些打探情报之类的事情，像阿苦有些蛮力，可以交给关鱼调教……其他人的本事，山寨也不是用不上。
<br/>　　所以这些人的投奔，对程大雷来说不算什么大事，真正令他烦恼的，还是这些人带来的情报。
<br/>　　林少羽竟然真的敢打长安城。
<br/>　　当然，以他的性格，做出这种事情也不奇怪。林少羽的父亲当初是被明帝冤杀，最后尸骨都没能落下来。推己及人，如果这件事发生在程大雷身上，程大雷不一定能比林少羽更理智。
<br/>　　毕竟，林少羽已经忍了三年。
<br/>　　都说君子报仇，十年不晚，可这世上究竟又能有多少君子。
<br/>　　现如今，林少羽在号召天下绿林豪杰，明帝发布勤王令，要诸侯动兵。
<br/>　　这场庙堂和江湖的对决，究竟谁胜谁负？
<br/>　　如今的程大雷是一头雾水，关于未来什么都看不破。
<br/>　　当然，他的思路也是极其清晰的，就是一个字：忍。选择按兵不动，冷眼旁观，只等纷纷乱象落了踏实，自己再做出更正确的选择。
<br/>　　第二天上午，阿喜、阿苦和白元飞三人一起来见程大雷。
<br/>　　“大当家，你找我们？”入乡随俗，现在他们对程大雷的称呼，也和山寨的所有人一样。
<br/>　　程大雷点点头，吩咐三人落座。
<br/>　　“昨天我一直在想对你们的安排，现在我有一个重要的任务交给你们，你们敢不敢接？”
<br/>　　阿喜眼睛一亮，他们这些人不怕死，怕的就是被小看。毕竟是一穷二白过来投奔程大雷，现在他们急切需要一件事，证明自己的价值。
<br/>　　所以，他们现在不怕危险，怕的恰恰是不够危险。
<br/>　　“大当家尽管吩咐，如果完不成，我们提头来见。”阿喜郑重其事道。
<br/>　　“好，我等的就是这句话。”程大雷道：“阿喜、阿苦、白元飞听令。”
<br/>　　“属下在！”三人同时起身答应。
<br/>　　“命你们三人组成一情报队伍，打探各地的情报。人员就是长安城带来的弟兄，这支队伍以阿喜为首，一个月训练时间，怎么训练我不管，一个月后，这支队伍要发挥出作用。”程大雷道：“这支队伍就以阿喜的名字命名，叫做喜字队。”
<br/>　　阿喜一怔，瞬间陷入狂喜之中。他可知道，能以自己名字命名的，都是山寨的中坚力量。比如鱼字军，蛮字军，龙字军……自己初来乍到，就可以有一支自己的队伍，而且是重要的情报人物，足见程大雷对自己的重视。
<br/>　　原先的担忧，烟消云散，现在程大雷提供了给自己施展本事的舞台，自己可以尽情施展平生所学。当然，于此同时，带来的也是巨大的压力，如果自己搞不好玩砸了，也就真需要提头来见程大雷了。
<br/>　　“属下定不负所托。”阿喜郑重其事的向程大雷行礼。
<br/>　　程大雷摆摆手道：“别太紧张，今天叫你们来还有别的事。你们是什么时候从长安城出发的？”
<br/>　　“算下来已经有一个月了。”
<br/>　　“也就是说，正义教已经围城一个月，情况如何？”
<br/>　　“双方互相见过几仗，互有损伤，却也没能分出胜负。”
<br/>　　程大雷微微点头，阿喜说的是来之前的情形，如今已过去月余，哪里的情况如何，目前还无法知晓。
<br/>(本章完)
