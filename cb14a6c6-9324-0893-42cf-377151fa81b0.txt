陆寒霆在文件下面刷刷的签上了自己的大名，龙飞凤舞的，他的俊脸上没什么表情，陆茵茵一再的挑衅，还在他的地盘上，其实他将陆茵茵请出海城易如反掌，但，陆茵茵终究是他的姑姑，他不想奶奶夹在中间为难，奶奶年纪已经大了。
<br/>
<br/>        “知道了。”陆寒霆掀了掀薄唇。
<br/>
<br/>        “总裁，刚才得到消息，陆茵茵已经开车前往圣得堡酒店了，今天李文清院长的记者发布会就在圣得堡酒店举行。”
<br/>
<br/>        陆寒霆拿着笔的大手突然顿了一下，夏夕绾也去圣得堡酒店了，原来李文清院长之所以选择圣得堡酒店不单是商讨厉老夫人的手术方案，更是为了这一场媒体记者发布会。
<br/>
<br/>        现在夏夕绾在网络上就是一个谜，她虽然是天才少女，但是学历还没有公开，李文清院长在酒店里正式召开记者会是要真正揭开夏夕绾的神秘面纱，让她璀璨绽放了。
<br/>
<br/>        看得出来李文清院长是十分喜爱夏夕绾的，陆寒霆还没有听说过这位得高望重的李院长如此重视过一个人。
<br/>
<br/>        陆寒霆有点烦闷，他将手里的钢笔丢了下来。
<br/>
<br/>        严毅看得出来自家总裁心情不佳，他开口道，“总裁，要不你也去圣得堡酒店看一看吧，今天过后，太太就不再是以前的太太了，太太要翻开人生崭新的一页，开启新篇章了，这个时刻怎么能让陆子羡陆茵茵他们陪在太太的身边呢，总裁你应该亲眼去见证一下这个时刻。”
<br/>
<br/>        陆寒霆比谁都清楚这一场媒体发布会之后，夏夕绾就会像一颗明珠，无论走到哪里都会是瞩目的焦点。
<br/>
<br/>        他突然想起那一日他跟夏夕绾在火车上的初次遇见，女孩澄亮黑漉的翦瞳看过来的时候，他就知道她不一般。
<br/>
<br/>        但他也从来没有想到她会这样的惊才绝艳。
<br/>
<br/>        陆寒霆抬手扯了一下衬衫的纽扣，这时一串悠扬的手机铃声响起了，来电话了。
<br/>
<br/>        他抬眸看了一下，竟然是夏夕绾打来的。
<br/>
<br/>        陆寒霆迅速拿起手机，按键接通，“不是走了吗，怎么给我打电话了？”
<br/>
<br/>        “陆先生，我是走了啊，可是我到楼下打车，打了好久的车都打不到，你们这里是不是不给打车啊？”夏夕绾轻软的嗓音传递了过来，还佯装叹息了一声。
<br/>
<br/>        陆寒霆深邃的眸底瞬间拨开了层层寒雾，露出了霁月般的笑意，小骗子，他这里可是海城最繁华的城市中心，门口车水马龙，怎么可能打不到车？
<br/>
<br/>        陆寒霆缓缓勾起薄唇，嗓音低柔含着笑意，“所以呢陆太太？”
<br/>
<br/>        “陆先生，能不能请你帮个忙啊，你可以开车送我去圣得堡酒店吗？”
<br/>
<br/>        陆寒霆剑眉微挑，心尖荡起了阵阵涟漪但语气如常，“哦，你等着。”
<br/>
<br/>        他起身，单手拎起了车钥匙健步离开。
<br/>
<br/>        严毅站在原地，这么一刻他好像看到了所谓的爱情，这世上最好的爱情大概就是你情怯不前时，她正好伸出了手。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆出了陆氏集团的大门，他站在劳斯莱斯幻影车边，寻找那道纤柔的身影。
<br/>
<br/>        这时身后有两只小手抱了过来，从后面一把将他给抱住了。
