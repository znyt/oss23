不是也没有发挥全力吗？”李振邦击退一头暴龙兽之后，声音平静的回道。<br/>　　<br/>　　    观众们听到两人的对话不禁有些愕然，擂台上已经斗的昏天暗地了，他们看的也十分过瘾，结果这竟然还不是双方的全部实力，这让他们有些骇然。<br/>　　<br/>　　    要知道擂台上的人最大也不过才二十五岁而已，这群人的身体内到底还隐藏着何种力量？这些人现在表现出来的实力已经比多数四五十岁以上的人还要强了，他们到底还要强到何种地步？<br/>　　<br/>　　    “以你们现在表现出来的实力，还想要我们发挥全力？你就不要做梦了！”苏白珊的语气中充满了讥讽，显然并没有把卡罗皇家学院的众人放在眼里。<br/>　　<br/>　　    作为一名年轻的七级光系大魔法师，苏白珊确实有傲视他们的资格。她不但有傲视卡罗皇家学院学员们的资格，甚至有傲视一切人的资格。放眼整个大陆，在她这个年龄段能超越她的人，历史上也没有几个，而那几个人无一例外都在历史上留下了浓墨重彩的一笔。<br/>　　<br/>　　    “既然你这么想看到我使用全力，那我就再卖卖力气好了。”李振邦说完，一只六阶的沼泽巨鳄出现在了擂台上。<br/>　　<br/>　　    这只六阶的沼泽巨鳄，正是李振邦之前在自由角斗场的战利品。当时它已经重伤濒死了，被李振邦救治了过来，成为了他的一只契约召唤兽。<br/>　　<br/>　　    沼泽巨鳄出现以后，变得很是兴奋，张开大嘴发出嗬嗬的声响。尽管这里不是最适合它战斗的地方，但是对很久没有战斗的它来说，都已经不重要了。<br/>　　<br/>　　    根本不用李振邦吩咐，沼泽巨鳄就主动对上了一头暴龙兽。沼泽巨鳄身体内也有着一丝巨龙的血脉，所以面对同样有着巨龙血脉的暴龙兽怡然不惧。<br/>　　<br/>　　    两头魔兽互相对峙着，发出低沉的嘶吼声恐吓着，脚步缓缓移动，都在寻找着对方的破绽。同是六阶强力输出的魔兽，都有着不低的智慧了。<br/>　　<br/>　　    沼泽巨鳄率先发动了攻击，作为会使用法术攻击的魔兽，要比暴龙兽这种纯物理输出的魔兽有一定优势。<br/>　　<br/>　　    一道道水箭从沼泽巨鳄的大嘴中喷吐而出，带着破空声朝着暴龙兽的身体射去。<br/>　　<br/>　　    水箭射在暴龙兽的身上发出“笃笃笃”如同敲击皮革的声音，并没有对暴龙兽造成什么实质性的伤害，不过却激起了暴龙兽的怒意。<br/>　　<br/>　　    暴龙兽怒吼一声，迈开大步朝着沼泽巨鳄冲了上去。<br/>　　<br/>　　    有了沼泽巨鳄的加入，光明学院和卡罗皇家学院的战斗天平发生了微弱的偏移，这一次轮到卡罗皇家学院占据一丝上风了。<br/>　　<br/>　　    苏白珊的辅助功能是强大，但毕竟有极限，不可能无限度的提升每个人的实力，这是神都做不到的事情。<br/>　　<br/>　　    “呀！不好意思，我稍微用一点儿力，你似乎就要求饶了。”李振邦冲着苏白珊挑了挑眉毛，语气说不出的轻佻。<br/>　　<br/>　　    “你……哼！”苏白珊咬了咬牙，冷哼一声，不再理会他了。<br/>　　<br/>　　    苏白珊不理会不代表其他人不理会，圣女在苦修士们的心中，地位仅次于教皇陛下，又岂容他人亵，渎。两名苦修士已经忍耐很久了，此刻再也忍不住了，同时放弃了自己的对手，朝着李振邦冲了过去。<br/>　　<br/>　　    龙淼淼只见面前的苦修士虚晃一棍，然后调转方向朝着李振邦冲去。不由得大惊失色，急忙想要追上去，却突然停下了脚步，抿了抿嘴，转身朝着苏白珊冲了上去。<br/>　　<br/>　　    慕容冰的私自行动让她变的很狼狈，苦修士的实力可不是用外在的等级来衡量的，她的大部分攻击都被苦修士给抵挡了下来。<br/>　　<br/>　　    并不是说慕容冰实力不如苦修士，只不过慕容冰并不是力量型战士，有小玉在一旁伺机攻击，让她根本不敢全力出手，小玉的攻击给她带来的痛感实在是有些苦不堪言。<br/>　　<br/>　　    此刻苦修士的突然离开，让慕容冰的压力大减，慕容冰对小玉早已经恨的牙根痒痒了，此刻哪里会放弃这个报仇的机会。<br/>　　<br/>　　    慕容冰将自己的速度发挥到了极致，手上的指虎如狂风暴雨一般朝着小玉攻击过去。<br/>　　<br/>　　    小玉本来不过是一名五级圣殿骑士，实力相当于黄铜战士而已，就算是有苏白珊的加持，离普通白银战士也差着一大截，更不要说白银战士中的佼佼者慕容冰了……<br/>　　<br/>　　    （17K后台正在升级，作者后台已经完全大变样，还在适应中。马上就要母亲节了，别忘记和妈说一声节日快乐！）<br/><br/><br/>
