　　秦琼战死的消息李承乾知道的并不比老头子晚多少，当初给老秦续命的事情是他找崔钰办的，现在秦的时间到了，会得到通知。
<br/>　　不过李承乾真正在乎的并不是老秦的生与死，十二年时间足够老秦安排好自己的身后事，他已经不再需要再为老秦操心。
<br/>　　所以崔钰说完事情的经过之后，李承乾只是嗯了一声，随后便瞪着崔钰说道：“说说我妹妹的事情吧，你不是说要向上面请示么，事情办的怎么样了？”
<br/>　　“陛下，您，您就不要为难臣了，长公主殿下的头都被打碎了，就算是臣能把人给带回来，也没办法恢复不是。”一听李承乾提起长乐的事情，崔钰顿时变成了一张苦瓜脸。
<br/>　　脑袋被开了两个洞，脑浆几乎流干的情况下，崔钰真的想像不到应该如何让人再活过来，长安又不是丧尸横行的浣熊市，怎么可能会允许这样的存在到处晃荡。
<br/>　　而且，现在时间已经过去两个多月，长乐的身体虽然外观保持的很好，但只要是个人都知道，这只是一具皮囊而已，身体里面所有的器官早就已经不能用了。
<br/>　　李承乾自然也知道这样的情况，可他现在也没有任何办法，长乐的事情他是真的不知道应该如何向老头子还有老娘交待，所以只能把一切责任都推给崔钰，让这位身兼双职的属下来想办法。
<br/>　　有过地府之行的经历，李承乾已经不再是一个无神论者，但是一些最基本的物理特性却不会因为那些神神鬼鬼的东西存在而发生改变。
<br/>　　也就是说，在大唐不可能存在人头明明都已经掉了，但尸体还会到处蹓跶的情况，如果真有那种情况出现，只能说是有人在装神弄鬼，绝不可能是借尸还魂。
<br/>　　“朕不管你用什么办法，总之朕需要长乐回来，如果你们办不到，那就别怪朕与其他人合作。”经过了这么长时间，李承乾也不是什么都不知道，与那几个被他抓来的道门人物聊天的过程中，他多多少少知道了一些事情，所以才会如此威胁崔钰。
<br/>　　“陛下……”崔钰的脸色变了变，似乎想要解释一些什么，但李承乾迅速摆手打断了他：“你可以把朕的话原封不动的带回去，两天之内朕要一个满意的答复，否则的话，你就不用再回来了。”
<br/>　　……
<br/>　　太原府，李家发迹之地，也可以说是龙兴之地。
<br/>　　在太原府城外的一座小小道观之中，独孤玉凤双眼有些失神的望着天空，看上去好像在犹豫天会不会掉下来。
<br/>　　舒天和程华安两个穷极无聊的坐在一旁，有一搭无一搭的聊着天。
<br/>　　“老程，你说那小道姑说的是不是真的？我们就这样隐秘不出，长安那边会不会出什么问题？”舒天挖着鼻孔，随手将鼻屎弹向院子里某个房间的方向。
<br/>　　“你还想出什么问题，长公主都没了。”程华安没好气的说道，显然他们虽然躲在这间道观里面，但并不是与世隔绝。
<br/>　　舒天叹了口气：“可我们也是为了陛下的安全，那小道姑不是说了，如果我们敢回去，他们就敢动手么。”
<br/>　　“谁会信？如果你没有亲自参与这件事情，我说给你听，你会相信？”程华安翻了个白眼继续说道：“要我说当初就不应该信了她的邪，直接将她抓起来，问出那两枝枪的下落就什么问题都解决了。”。
<br/>　　舒天拍拍屁股站起来：“算了吧你，你也不看看当时是什么情况，还抓住那小道姑，咱们三个当时什么情况你又不是不知道。”
<br/>　　提起当初，程华安立刻跳了起来：“当时的情况怎么了，王成虎那个败类，老子没想到他竟然会与道家的人走到一起，亏得当初陛下对他那么好。”
<br/>　　“现在说什么都没有用了，我们现在是有心杀敌，无力回天，事情已经发展成现在这个样子，就算是回去了也没有什么用处，不如继续留在这里，看看他们还能玩儿出什么花样。”舒天一边说着，一边来到独孤玉凤的身边：“夫人，您的意见呢？”
<br/>　　独孤玉凤依旧是有些忧郁的看着天空，就像身边的舒天并不存在一样。
<br/>　　程华安无可奈何的摊开手，对着舒天摇了摇头。
<br/>　　独孤玉凤这个样子已经整整有一个多月了，自从收到长乐公主的死讯，她除了吃饭、喝水、上厕所，基本上都会保持着同样的一个姿势。
<br/>　　而就在此时，院子里一间厢房的门被人从里面打开，玄寻雪从里面缓缓走了出来，脸上带着一丝憔悴。
<br/>　　舒天见到玄寻雪出来，立刻转移了目标，从独孤玉凤的身边离开：“喂，玄寻雪，你一直把我们留在这里到底有什么目的，我们三个就那么招人稀罕？”
<br/>　　玄寻雪轻轻带上身后的房门，同时温声说道：“把你们三个留下自然是有我的目的，不过眼下时机未到，还不能告诉你们。”
<br/>　　舒天似乎早就知道玄寻雪不会告诉自己，耸了耸肩膀继续问道：“那你呢？你每天都把自己关在里面，到底在搞些什么名堂？”
<br/>　　“推演天机！”玄雪寻淡淡说道。
<br/>　　面对玄寻雪例行公事般的回答，舒天打了个哈哈说道：“算了吧，你都推了好几个月了，也没有推出个结果来，所以还是不要再白费力气了。”
<br/>　　玄寻雪用不屑的目光看了舒天一眼：“天机虽然难测，但却不是绝对，你们非我道之人，焉知其中玄奥！”
<br/>　　“哎，你……”舒天打蛇不成反被咬，顿时有些觉得下不来台。
<br/>　　倒是程华安在旁边拉了他一把：“行了，你少说几句，留着力气想想晚上吃什么好了，何必与这道姑一般见识。”
<br/>　　“哼，一群不明是非之辈！”玄寻雪侧目看了程华安一眼：“一场危机近在眼前，可笑尔等命不久矣却还想着吃喝。”
<br/>　　“什么？你说什么危机？”舒天的注意力瞬间被玄寻雪吸引了过去，皱着眉头问道：“你最好把事情说清楚，否则别怪我们对你不客气！”
