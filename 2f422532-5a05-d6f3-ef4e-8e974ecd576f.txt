　　第556章
<br/>　　“公主是怎么死的？”
<br/>　　夫子慢吞吞的开口，最后还是问出了声。
<br/>　　“被人杀死的。”孟玄青道。
<br/>　　“什么。”夫子一惊，强压住眼中的情绪，道：“哎，罢了，人活一世，草木一秋，到最后终究是个死。公主性格刚硬，树敌又多，死在对头手里，也不算什么稀罕事。”
<br/>　　孟玄青抿了口茶，看了夫子一眼，道：“你为什么不喝茶，要不我给你倒上？”
<br/>　　“多谢，我还不渴。”夫子口中说得平静，心里想得明白：在你这毒师面前，我哪里敢喝茶，谁不知道你下毒得手段神不知鬼不觉。
<br/>　　他沉默了片刻，道：“公主死得痛苦嘛？”
<br/>　　“这我就不清楚了。”孟玄青抿着茶道：“反正就是在火海里活活烧死的，临死前嗓子都喊哑了，最后万箭攒心。至于痛不痛苦，其实我也不知道，那你得到九泉之下问公主殿下。”
<br/>　　夫子的眼皮剧烈跳动，强压着心头火，良久才恢复平静。
<br/>　　“那公主的遗骨，可有妥妥安葬？”
<br/>　　“哪还有什么遗骨。”孟玄青道：“一把火烧成灰了，刚才我不是给你说了么，公主被万箭攒心，最后尸体里烧出好几斤箭头。”
<br/>　　说罢，孟玄青将茶杯倒满，目光盯着夫子的脸。
<br/>　　夫子一张脸由白转青，由青变红，脸颊微微跳动。
<br/>　　咚。
<br/>　　他的手重重拍在桌子上，原地站起，口中哇呀呀大叫：“公主千金之躯，就算该死，也不该下这样的毒手。当真是气煞我也，气煞我也。”
<br/>　　孟玄青抿口茶，眼观心，心观口，闭口不言。
<br/>　　夫子年轻时有个外号，叫赤面书生。满口子仁义道德，是一肚子杀人放火。动辄把刀相向，一言不合就灭人满门。也就是如今年纪大了，修身养性，外表看上去慈眉善目许多。
<br/>　　当然，也只是外表看上去。所谓江山易改本性难移，骨子里的东西却是不好改的。
<br/>　　而他是鲁国王子，白衣老尼是西蜀公主，王子和公主之间，却还发生过一段缠绵悱恻的爱恋。只不过两个人的性格都属于急火金刚，虽然也算郎有情妾有意，可相处时却是争执更多些。所以这桩事，也就不了了之。夫子终身未娶，未必没有这方面原因。
<br/>　　而的确，在当初那伙人中，孟玄青算是脾气比较好的。
<br/>　　“是谁杀了公主？”夫子咬牙切齿，双目喷火，过往赤面书生那股劲儿又出来了。
<br/>　　“哎，人生一世，草木一秋，谁还没个死。你……要看开些啊。”孟玄青又端起茶杯。
<br/>　　啪。
<br/>　　孟玄青手中的茶杯被夫子夺过来摔得粉粉碎，口中骂道：“我与那贼人，一天二地仇，三江四海恨。不杀他为公主报仇，我何以为人。”
<br/>　　“你这个人得脾气太急躁了。”孟玄青道：“那人得来历可不简单，本事也是不弱。”
<br/>　　“哈哈哈。”夫子仰天长笑：“我万卷山立足东海，弟子满天下，山中高手如云，又怕过谁来，快快快，你与我说，究竟谁杀了公主。”
<br/>　　“你当真要知道？”
<br/>　　“当真。”
<br/>　　“好好好，我便说与你听。杀公主的贼人，就是那琴川之主，江湖上恶名昭彰的蛤蟆大王程大雷。”
<br/>　　“啊，是他。”夫子一惊，坐回到椅子上。
<br/>　　“怎么，你不敢了，提起他的名字，便吓破了胆？”
<br/>　　夫子冷哼一声：“我正要杀他，只不过西北与东海相隔太远。虽然有消息说他已来了东海，但东海之大，想要寻到他怕是不太容易。”
<br/>　　“这个你不用担心，要找到他轻而易举。”
<br/>　　“什么，你知道他在何处？”
<br/>　　“当然。”孟玄青冷冷笑道：“此人不在天边，近在眼前，此时此刻，他就在你的书院。”
<br/>　　夫子一怔，呐呐道：“他是……”
<br/>　　……
<br/>　　燕不归陷入三大高手的包围中，虽然他没有程大雷的能力，但行家一出手，便知有无有。三人一出现，就封锁了他逃走的路线，人人摆出防御姿态，站在那里毫无破绽。
<br/>　　啧……
<br/>　　燕不归叹了一声，脸上也无太多变化。他握住剑柄，拔出了佩剑护在身前。
<br/>　　“谁来送死？”
<br/>　　“小贼，上次你我没有分出胜负，这次与我战个痛快。”
<br/>　　张六同样是火爆性子，拎着大棍就冲了上来。二人战在一起，打得难解难分。张六一杆顶门杠，专打燕不归得腰眼，小腿，肩膀，脑袋。木棍像一条闹海蛟龙，来势汹汹，威力当真不容小觑。
<br/>　　燕不归也是存了必死决心，重剑术重攻不攻守，一柄五十一斤得重剑，速度半点没有被拖慢。二人都是以快打快，强攻对强攻。
<br/>　　当真是棋逢对手，将遇良才，互相都没讨到便宜。如宋寒之和马猛二人，一时间竟没有插手的机会。
<br/>　　宋寒之是找不到插手的机会。马猛却是没办法插手。他是起夜出来的，总不能带着把刀起夜。如今赤手空拳，他可不敢去接燕不归的重剑。
<br/>　　宋寒之见二人的战斗一时之间分不出胜负，于是也并不着急上前助战。口中呼喝连连，是传讯给书院的众弟子。
<br/>　　等人到齐了，布下书院七十二剑阵，哪怕燕不归是大罗金仙，也得留在此地。
<br/>　　燕不归也明白这一点，此刻的他已不敢有任何轻视书院的心。如果现在再不走，等书院的高手到了，想要再走就走不成了。
<br/>　　从张六这里突围几乎已是不可能的事情。燕不归目光转了一圈，盯住了马猛。
<br/>　　马猛站在那里太奇怪了，脸皮抖动，两腿紧紧夹着，似乎正忍耐着什么。
<br/>　　上看下看，左看右看，无论怎么看，你都是个破绽啊。
<br/>　　燕不归大喝一声，强攻逼退张六，举剑攻向宋寒之。这却是虚招，引得宋寒之和张六防守后，他身体突然一折，一剑砍向马猛。
<br/>　　马猛当真已到了关键时刻，好巧不巧，燕不归把突围的重点放在他身上。如果往常时候，马猛也不怕燕不归，可今天的事情对他而讲，当真有些不利。
<br/>　　也是人到绝境，逼出潜力，他双掌一合，硬生生接住了燕不归的重剑。
<br/>　　“贼人，留下吧。”
<br/>(本章完)
