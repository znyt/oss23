　　也许，对道风来说，对与错其实都不重要，他要的，是一个结果。
<br/>　　一道人影，飞上城楼，落在他的身边，是陈露。
<br/>　　陈露现在跟他的关系很微妙。一开始，道风是强烈反对她跟随自己身边的，不过陈露十分执着，一直留在风之谷不走，而且她对杨宫梓和他的关系，并不嫉妒，用她自己的话说，她只要能留在道风身边，可以不需要什么名分，只要能在一起就行。
<br/>　　她不在乎做小老婆。
<br/>　　道风对她，也是完全没有办法。
<br/>　　“风哥，我就知道你在这里。”陈露飘过来，笑嘻嘻地说道。每一次见到道风，她都很开心。
<br/>　　“什么事？”道风问。
<br/>　　“有个人找你，被小马他们拦住了，是一个老道士，小马出手阻拦，根本打不过他，不过他也不伤人，我们问他是谁，从哪里来，他都不说。”
<br/>　　道风皱了一下眉头，本想问什么，最后还是让她把人带来。
<br/>　　陈露走后，道风继续出神地望着城外，等了有一会，一个声音在身后响起来：“这里的风景，比人间可是差远了。”
<br/>　　道风听声音有点陌生，转头看了一眼，两眼顿时睁大，惊道：“是你！”
<br/>　　站在自己身后的这个人，居然是徐福！
<br/>　　饶是一向历尽万事、宠辱不惊的道风，在见到徐福的第一眼，也是情绪大变，一只手临空抓出了一道黑光，附身冲了过去。
<br/>　　“切莫动手，道风子，我却是找你有事相商。”
<br/>　　“我先擒住你再说！”
<br/>　　道风身影一晃，化形消失，几乎是下一秒钟，徐福立刻感到一股无形的气流绕着自己旋转起来，产生了一股极强的压迫，并且不断飞快地旋转着，寻找着进攻的突破口。
<br/>　　徐福不敢怠慢，双手一翻，将两只判官笔拿在手中，同时闭上眼睛，在心中算起风暴进攻的方位：“乾、坤、坎、兑、离……”
<br/>　　猛然双目暴睁，手中两只判官笔朝着一个方向扫了过去，几乎在同一瞬间，围绕他旋转的无形之气也汇聚成了一个圆点，然后爆发开来，在一时间将攻势燃烧到了极致。
<br/>　　徐福手持判官笔，交叉在一起，死死顶住了这股风暴，衣服和头发胡须都被余波震得飞了起来，身体也晃了两下，后退半步，脸上却展开了笑容，道：“道风子，你而今有这惊世骇俗的手段，我找你却是没有找错。”
<br/>　　风暴散去之后，道风的身影显现出来，站在徐福的面前，一只手擒着番天印，道：“你可与我一战。”
<br/>　　徐福道：“我修炼千年，却不是好欺负的。”
<br/>　　道风道：“我若用心一战，鹿死谁手，有未可知。”
<br/>　　徐福摆了摆手，“我知道你为什么想擒住我，你怕我跑了是不是？”
<br/>　　“少废话，把小师弟给我送回来！”
<br/>　　“我今日既然来风之谷找你，却是为了这件事，道风子，你先别动怒，听我慢慢与你讲来。”
<br/>　　“少阳呢？”道风心底最关心的就是这件事，“这件事不讲清楚，我跟你没得谈。”
<br/>　　徐福点头说道：“你们兄弟情深，我是知道的，你放心，少阳子现在民国，暂时还好好的。”
<br/>　　道风长出了一口气，之前瓜瓜来报信，说叶少阳从民国来信，当时他还觉得有点不可能，如今听徐福亲口这么说，这才真正确定了，一直压在心底的石头总算放下了，随即问道：“为什么是暂时？”
<br/>　　“民国很危险，比你们这个时代更危险，他会遭遇什么，谁也不能确定……”
<br/>　　“你为什么送他去民国？”
<br/>　　徐福叹道：“道风子，少阳喜欢的那姑娘，是转世鬼童，现在尸族手中，若少阳在这里，必然杀去尸族，后果……你必然也想的到，就算九死一生，救出鬼童，又怎么跟天下交代，转世鬼童，人人得而诛之，到时候少阳会因为她得罪天下，如何自处，又当如何自保？”
<br/>　　道风微微颔首，道：“所以，你送他去了民国？”
<br/>　　“星宿海一战，他丹田损坏，实力大减，在人间无法自保，我若留他在这里，必然只有死路一条……送他离去，却能暂时保他周全。”
<br/>　　道风放下了番天印，道：“那你为何送他去民国，而不是别的时代？”
<br/>　　“那自然是有原因的，我要帮他历劫。”
<br/>　　“历劫？历什么劫？”
<br/>　　徐福笑道：“这个，却是只有他自己知道了，我也只参透一二，天机昭昭，我却不能告诉你。”
<br/>　　道风也没有逼问他，问道：“他什么时候能回来？”
<br/>　　“若是成功历劫，他的实力会超过从前，必然可以回来堪当大任，应对天劫……”
<br/>　　道风冷笑一声，道：“你所作所为，也是为了应对天劫，那何以违背阴司，成为阴司首要追捕的逃犯？”
<br/>　　徐福冲他也笑了一下，说道：“你呢，你所作所为，难道是阴司所同意的。”
<br/>　　道风也无话可说，想了一下道：“你来找我，是为什么事。”
<br/>　　“一件大事，我想来想去，只有你能帮我。”
<br/>　　道风疑惑地看着他，等着他说下去。
<br/>　　“帮我救出白起的元神。”
<br/>　　道风震惊。
<br/>　　“白起元神，被关押在枉死城昭狱之中，我上次试过一次，没能得手，因此，我需要一个帮手。”
<br/>　　道风说道：“你不是疯了吧，去枉死城昭狱抢人，这根本不可能。”
<br/>　　“很有可能。”徐福心平气和地说道：“在之前，那是不可能，不过现在，阴司与太阴山开战，又为青冥界的战局所牵动，内部空虚，你我合作，必然能救出白起。”
<br/>　　道风凝视着他，说道：“我为什么要帮你？”
<br/>　　“因为，我们需要人手。”
<br/>　　道风眉头皱起来，道：“我们？”
<br/>　　徐福走近他，手一翻，拿出了一方玉印，在空中一点，玉印下方的金字亮了起来，好像烟花一样向四周绽放，无数流光，将两人罩在中间。
<br/>　　道风环顾左右，感受到了一种真空般的存在，知道他是用了法术生成了一个完全与外界隔绝的空间，在这里说话是绝对安全的。
