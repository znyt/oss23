　　两人踏上神秘的浮空小岛，想象中的阻拦或者攻击并没有出现，好似里面并没有人似的，但两人都不敢放以轻心，谁也不知道这个小岛里面会有什么，小心驶得万年船。
<br/>　　而且他们也知道，此事的最后的幕后黑手也都没有找到，那恐怕就是所谓猎食的“天”，面对如此存在，他们又怎敢大意？
<br/>　　两人都用法宝护身，玄光垂落，防护这他们的身体，入眼所见，是一座已经坍塌大半的建筑群，建筑暗淡无光，但依稀可以看见一些，完好那半，有种恢弘大气的气势，而且那建筑所用的材料，都不是普通之物，应该是炼制至宝所用的器材，连瓦片都是黄金色的，有股神圣而缥缈的感觉。
<br/>　　那半边建筑明显是一座雄伟的殿宇，有一股奇异的力量，殿宇内暗淡无比，此处到处都充斥着那神秘的黑色物质，而且殿宇内的黑暗最为浓郁，好似那里才是黑暗的源头。
<br/>　　那半块残破的门匾，是一种神秘树木所制成的，竟然能够不被黑暗完全侵蚀，上面的大字乃是一种神秘大道之文所书，字里行间透露着玄之又玄的奥秘，好似一座高峰屹立在人前，让人完全感受不到它的峰顶，又似一汪大洋，人只是大洋中的一个小黑点，完全不起眼。
<br/>　　它就好似大道，无边无涯，又处处透露着玄妙，蕴含着无上奥义，远超一般混元层次。
<br/>　　“太玄”两字，不只是一个名称，更好像是象征着什么，那是一种大道。
<br/>　　元始只是看上一眼，便在其中找到了好多熟悉的大道，还有更多他不熟悉的，没听说过的，只是一眼，他便沉浸其中，不可自拔。
<br/>　　种种大道奥义被他领悟，他的心中的大道逐渐被完善，越是感悟，他越是震惊，书写这两字的人物，修为完全到了一个不可思议的境界，至少远超元始前世因果之主的修为，元始不知道那是天道境还是什么，因为他还没有达到过天道境。
<br/>　　这两个字中，他好似看到一个身穿银边花纹白色长衫的男子，他剑眉挺翘，双眼傲世，全身上下有一种独特的气质，他在奋笔疾书，写下太玄两字，两字竟然化形成妖，但立刻又被那男子打散，重新变为字形，虽然如此，但两字其中的韵味，依然浩瀚无边。
<br/>　　“当——”
<br/>　　元始脑海中突然响起一道悠扬钟声，如晨钟暮鼓，让他有种醍醐灌顶的感觉，他一下子清醒过来了。
<br/>　　“好厉害！”
<br/>　　睁开眼睛，元始突然有些心悸的看着眼前的神秘匾额，只差一点，若是没有混沌钟将他敲醒，他今日恐怕就要沉沦进去了，恐怕要不了多久，他便会死去。
<br/>　　就算如今，元始都已经很麻烦了，因为他全身上下都被神秘的黑暗物质给侵染了，而且这种物质还在入侵他的神魂，若是他再迟一步，恐怕连神魂都要被污染了，那时他将彻头彻尾地变成一个黑暗怪物。
<br/>　　醒来后，他便知晓原因。那太玄中所蕴含的道理是真的，但它却也被黑暗给污染了，平时还看不到，但一旦领悟，那种黑暗的气息便会随着自己的领悟，入侵到自己的大道道韵之中。
<br/>　　所以元始才会感觉到厉害，这黑暗物质竟然能够融入到道韵之中，从而入侵别人的大道。这一刻，元始想到了许多，若是以后遇到这样的人，在与之大战过程中，自己该怎么办，不用大道吗，若是不用大道法则，那又如何战斗，但是用了大道，若是被入侵了，就好似你的大道被夺舍了，那时的大道还是自己的吗。
<br/>　　若是不想被入侵，那就只能让自己的大道比别人的强，到时候，他还怎么入侵。
<br/>　　可是，如今自己的大道道韵在刚才的武道过程中已经被入侵了，而且自己的身体也被入侵了，怎么办？
<br/>　　“我明白了，这黑暗物质并不是某种生物，而是一种大道所衍生出来的物质。”
<br/>　　元始喃喃低语道。同时他心里也在暗暗吃惊，到底是什么修为的人，才能凭借一道大道神通衍生出来的物质就将大罗金仙甚至准圣给污染了。元始不知道这人到底有多强，但至少不会比那位书写“太玄”大字的人差。
<br/>　　想到此处，元始就暗暗心惊，宇宙果然广阔，不走出去，永远不知道宇宙的危险。
<br/>　　他转头看向旁边，却见此刻道祖还沉浸在无边大道之中，而且他也被黑暗迷雾给入侵了，却丝毫不知。
<br/>　　对此，元始丝毫不意外，自己的修为比道祖高，都被入侵了，道祖被入侵也很正常，若是没有入侵才奇怪，奇怪他身上到底有何至宝，能够抵御这种大道入侵手段。
<br/>　　“醒来！”
<br/>　　元始一声大喝，在道祖的识海之中响起，那已经被黑暗入侵的黑色识海中，荡起圈圈波纹，道祖一个激灵，缓缓醒了过来。
<br/>　　感受到自身的情况，道祖心中一片苦涩，他没想到自己竟然会被黑暗给污染了，这次他的宝贝竟然没有抵抗住。他心神一动，顿时便看到在他神海最核心处，一颗棱锥形的光明晶体正在散发着光明的力量，驱散着他识海内的黑暗，但已经被污染的身体，却是无能为力。
<br/>　　这颗散发着光明的神秘晶体是他一次游历混沌时偶遇的，当时他便觉得此物不简单，处于混沌之中，竟然没有被混沌之气给化为虚无，而且还能一闪一闪地散发出出强烈的光芒。
<br/>　　道祖如获至宝的收了这块光明晶体，但是却发现这东西自己竟然不能炼化。而且它看起来并不完整，当时道祖并没有多想，认为这东西是残缺的，并不能炼化，所以他一直放在识海之中，上次意外进入此地，他发现这东西竟然能够驱逐黑暗，他顿时喜出往外。可惜，他只能被动使用，无法主动出击。
<br/>　　此次进入这神秘的地方，道祖一直依仗这光明晶体来抵挡黑暗，之前也一直没有出事，可这次他自己沉浸于悟道状态，自己将黑暗物质卷入道韵之中，导致黑暗入侵。
<br/>　　真是自作孽，不可活，道祖有些欲哭无泪。
<br/>　　还好没有被完全侵染，而且看元始的样子也被入侵了，他心里才多少好受些。
<br/>　　“道友，如今该如何是好？”道祖看向元始，有些无奈问道。
<br/>　　“我等都已被黑暗感染了，如今也不好后退，只好进去一看，若是找到了原因，说不得能够解决吾等身上之难。”元始沉吟片刻，说道。
<br/>　　他自是知道，恐怕里面更加危险，但如今已无退路，必须前进。
<br/>　　道祖自然也知道，他只好同意，虽说那光明晶体能够克制黑暗，但他不能调用，谁也不知道它能够撑多久，他可不想用自己的性命去赌。
<br/>　　元始看着眼前书写太玄的残匾，心中知道这是个好东西，但因为被黑暗入侵了，导致自己不敢用，若是能够去除那种黑暗的影响，绝对算得上一件至宝。
<br/>　　“哎！”
<br/>　　难道只能放弃，但他心中又有些不甘，元始叹息一声。
<br/>　　突然，光华一闪，元始不再犹豫，收起那残匾，将之收进了混沌钟空间之中，混沌钟内蕴一方星空，自然可存储东西。
<br/>　　旁边道祖看着有些羡慕，他自己是因为没有合适的东西来收，若是有，他早收了。这东西绝对是件至宝，仅仅感悟片刻，他就感觉自己快要突破了，若是能天天感悟，那还了得。可惜，没有先天至宝以上的灵宝，贸然收取，只会给自己找麻烦。
<br/>　　收好残匾，两人继续向前走去。
<br/>　　远处，黑暗物质更加浓郁了，向着外面扩散开来，仿佛那翻滚的潮汐，一浪接着一浪，永不停息。
