　　如果一定要说看到李承乾时的心情，长乐只能说很温暖吧，心里热热的，有种想哭的冲动。至于其他，那都是扯蛋，什么激动啊、什么热血啊、还有什么后悔啊，这些都没有。
<br/>　　长乐知道，当时她心里真的什么都没想，看李承乾的那一瞬间，她真的只是觉得特别委屈，特别想哭。
<br/>　　黑暗中，小姑娘想到了很多东西，她并不想死，但是她知道，如果只能活一个的话，哥哥活着才是最好的选择。
<br/>　　然而实际上的情况却是两个人要么都死，要么都活，只有一个活下来的话，结局并不一定就比死了的要好多少，这就是现实。
<br/>　　李承乾看的很清楚，也很明白，所以祈祷称心能够及时搬来救兵的同时他也在想着如何进行自救。
<br/>　　“丽质，哥不会死的，你也不会死，放心的睡一觉好不好？哥哥有些事情要做，等弄完了，我们就一起出去了。”打起精神安慰着妹妹，李承乾试着将身体离开长乐，他要去石头那里试试，看看有没有办法把它弄开。
<br/>　　“不，我不要，哥，要不我们死在起好了，这样下一辈子我们还可以作兄妹。”长乐倔强的抓住李承乾的胳膊，死活不肯松手。
<br/>　　“傻丫头，你听谁说死一起下辈子还作兄妹的，说不定你成了姐姐，我成了弟弟呢。”李承乾故意逗长乐，作为哥哥这是他的责任。
<br/>　　“那我照顾你啊，而且你要听话，如果不听话我就打你屁股。”长乐似乎完全沉浸在幻象之中，说着说着竟然笑了起来。
<br/>　　“行行行，以后你照顾我，但是现在不行，现在我是哥哥，如果你不听话我可是会打你屁股的。”
<br/>　　“那你打好了，反正我不让你离开，你一离开就不会回来了。”长乐执着的认为，李承乾离开她是想要去自杀。
<br/>　　“那你跟我一起过去看看那块石头，我们一起想办法。”李承乾被长乐整的没招了，只能拖着她一起走。
<br/>　　摸索着，兄妹两个将堵住洞口的石头摸了一遍，结果发现似乎没什么好办法，乌漆嘛黑的什么都看不到，光靠摸根本解决不了问题。
<br/>　　“这里太黑了，什么都看不到，如果有点光线就好了。”再次摸索了一会儿，李承乾有些恼火的说道。
<br/>　　“有啊，这里有蜡烛，还有火折子。”长乐缩在李承乾的身边小声说道。
<br/>　　“蜡烛？这里怎么会有这种东西？”
<br/>　　“昨天晚上那些坏蛋就是在这里休息的，有一些没有用完，就放在洞里了。”长乐解释道。
<br/>　　“在哪里，带我过去找找。”李承乾一只能找到照明的东西，立刻来了兴致。
<br/>　　现在虽然洞里的空气不多，但应该足够两人再用一会儿，蜡烛那东西就算是再能烧，估计也烧不掉多少空气。
<br/>　　再说如果能找到一些缝隙什么的，只要能捅开一些，那么也就等于有了空气，尽管不能满足洞里的空气循环，但是趴在缝隙那里喘喘还是可以的。
<br/>　　摸索蜡烛又费了不小的功夫，主要是长乐这个小丫头记不准方向，一会儿东一会儿西，最后差不多要把洞里全摸遍了，才从山洞中间刚刚长乐坐的那块石头的下面摸到蜡烛和火折子。
<br/>　　蜡烛点燃，长乐看着李承乾怪异的目光，自己也觉得这事儿办的有些差劲，不由心虚的说道：“哥，你干嘛那么看我，如果不是我记起来有蜡烛，你现在还在摸黑呢好吧。”
<br/>　　李承乾翻了个白眼，没有再理会长乐，而有了光，长乐也不再缩在李承乾怀里，只是跟屁虫一样走在他身后。
<br/>　　等走到洞口石头跟前时，李承乾随手从背后将三棱军刺抽了出来，吓了长乐一跳，兔子一样跳到一边：“哥，你要干嘛啊？”
<br/>　　“傻丫头，不用工具怎么把这里捅开。”李承乾试着把军刺插进一个个四周的裂缝里面，扭头对长乐说道。
<br/>　　“这东西你天天都带着啊？不怕伤到自己么？”看着李承乾捅来捅去的样子，长乐也凑过来帮忙找缝隙，找到一个就指给李承乾看。
<br/>　　就这样兄妹两个一路试下去，直到最后失望的发现，缝隙的确是有，但是位置不对，军刺太短了，够不到那些被堵住的位置。
<br/>　　“哥，我们是不是死定了？”被困在一个地方，时间往往都会过的很漫长，单单是找蜡烛和试验缝隙，长乐觉得像是过了一天一夜一般，慢慢开始失望起来。
<br/>　　“现在说死还是太早了一些，办法还有最后一个，如果不行……下辈子你当姐姐吧。”李承乾用手中的军刺在石头上戳了几下，苦中作乐的说道。
<br/>　　“什么办法？”长乐精神一震。
<br/>　　“你说，这个缝隙能有多深？”李承乾指指石头本身左下角位置，离地大概两尺左右的位置。
<br/>　　“不知道，怎么了？”长乐问道。
<br/>　　“赌一下吧。”李承乾没有理会长乐的问题，自言自语的嘀咕一句。
<br/>　　接着，他就在长乐疑惑的目光中开始往外掏东西，时间不大，地上已经放着三十来个小小的纸包。
<br/>　　“这是什么？”长乐拿起一个问道。
<br/>　　“火药。”李承乾从长乐手里把纸包拿回来，轻轻打开，露出里面黑色的火药，然后将纸轻轻折出一条线，顺着线将火药灌进了石头的缝隙里面。
<br/>　　长乐没有再说什么，只是在边上静静的看着，当看到李承乾打开第四个小包时，她也学会了打开的方式，便帮着哥哥，把一个个纸包打开，然后放在一边备用。
<br/>　　此时洞里的空气已经有些浑浊，兄妹两个已经没有了说话的兴趣，一阵阵若有若无的窒息感将他们笼罩。
<br/>　　一包又一包火药被灌进缝隙，便缝隙却始终没有被灌满的趋势，长乐终于忍不住自己的担心：“哥哥，要不……”。
<br/>　　“闭嘴，现在说什么都晚了，剩下这一点空气根本不够一个人活下来。”李承乾将长乐的话头打断，头上不住的往外冒汗，看着地上仅剩的八个打开的小包，一阵阵绝望开始充斥脑海。
