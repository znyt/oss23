　　这一天，帝俊身穿金黄色九龙帝袍，站在一个高大的祭坛之上，旁边还站着手托一卷妖文写成的祭文的鲲鹏，台下站着太一、女娲、伏羲等人，以及妖族的众多高手，全场一片安静。
<br/>　　“吉时已到，祭天开始。”
<br/>　　鲲鹏大声喊道，同时双手托起祭文来到帝俊面前，待帝俊托起祭文，便退到一边。
<br/>　　帝俊接过祭文，开口念道：
<br/>　　“自天地开辟以来，洪荒混乱不堪，众生皆苦……今有妖族愿代天管理洪荒，还众生一片安宁。今吾帝俊、太一、女娲、伏羲、鲲鹏，共创妖族，以三十三天为妖族根基，混元剑、河图、洛书为镇族之宝。天庭，立！”
<br/>　　一篇华丽的祭文，由妖帝帝俊口中念出，声音宏大，传遍妖族每个成员耳中，传遍洪荒大地，让无数生灵都能听到。
<br/>　　随着妖帝帝俊的声音，字字如轰雷般落下，天地之间，仿佛有一股玄妙之音，回荡而起，让众神通者脸色阴沉无比，他们知道，妖族立天庭，是天命所归。
<br/>　　紧接着，却见三十三天之上。帝俊陡然又是一步踏出，他大袖一挥，脸色威严无比，大声喝道：
<br/>　　“今吾妖帝帝俊，整合妖族，创立天庭。吾为天帝，太一可为东皇，伏羲为羲皇，女娲为娲皇，鲲鹏为妖师。此后，妖族共尊之。”
<br/>　　“拜见天帝，拜见三皇，拜见妖师……”妖族众强者纷纷扣首膜拜。
<br/>　　帝俊不停，继续念道：“白泽、计蒙、英召、飞廉、九婴、商羊、钦原、毕方、穷奇、诸犍可为十大妖神。大妖有鬼车、饕鬄、飞诞……”
<br/>　　“拜见妖神！”众小妖叩拜道。
<br/>　　轰隆隆！
<br/>　　就在此时，轰隆隆间，九天之上，滚滚仙音飘摇，一道道瑞气彩云浮空而起，万丈祥云布满苍穹，整个三十三天天界意志，仿佛承认了帝俊、太一、女娲、伏羲、鲲鹏等五大妖族首领的地位一般，道韵缭绕。
<br/>　　下一瞬，更有大量的璀璨功德金光汇聚而来，此光璀璨无比，一分为六，大约六成被帝俊、太一所得，三成直奔女娲、伏羲、与鲲鹏，剩下一成，洒向在座的众多妖族成员。
<br/>　　一时之间，五大妖族首领周身光芒万丈，在功德的加持之下，道行一路飙升，气势势不可挡般崛起，轰隆隆间，帝俊与太一借助天庭的气运与功德，再斩一尸，成为斩二尸的准圣中期高手。
<br/>　　而，女娲、伏羲、鲲鹏三人，也是道行高升，最终停在准圣初期巅峰！鲲鹏因为先前创妖文得到功德，此次更是只差一步，便可斩去第二尸。
<br/>　　“哈哈哈！”几人兴奋不已。
<br/>　　“混账！岂有此理？立天庭竟还有如此功德气数，帝俊二人，居然借此再斩一尸，如此一来，除了三清之首元始外，他俩岂不是鸿钧道祖之下，洪荒最强？”众多洪荒大神在此刻全都嫉恨不已。
<br/>　　而与此同时，抵达盘古殿之外，正要踏入巫族地盘的万劫魔尊，突然间似有所感，抬头看向三十三天方向，瞳孔一缩，冷笑道：“借助功德斩尸，可惜，这是最弱的方式。对于本尊来说，你们还是很弱。”
<br/>　　接着，大袖一甩，不再理会，毫不犹豫地踏入祖巫地盘。看着眼前古老沧桑的盘古殿，蓦然开口道：“贵客临门，各位祖巫，可否出来一见。”
<br/>　　声音透过宫门，传入大殿深处。这让正在里面商量事情的十二祖巫一惊，顿时一同走了出来。
<br/>　　“阁下是谁？为何会来我盘古殿？”领头的是祖巫之首空间祖巫帝江，开口说话的也是他。
<br/>　　“哈哈，万劫不灭邪圣仙，万道森罗魔至尊。本尊乃万劫魔尊！至于为何来盘古殿，当然是为你巫族而来。”万劫魔尊大笑道。
<br/>　　“万劫魔尊？没听说过。阁下为我巫族而来？此话怎讲？”帝江谨慎地看了一眼他，慢慢说道。
<br/>　　万劫魔尊一点也没有因对方没有认出他而尴尬生气，别人若是认识他，那他才会奇怪呢。他毫不忌讳道：“妖族如今立天庭，声威大涨，而作为他们对头的巫族，你们如今恐怕不好过吧。”
<br/>　　“你这魔人，怎么说话的？”祝融大声喝道。
<br/>　　“我巫族如何，好像与阁下无关吧。阁下是不是有些多管闲事了。”帝江听见此话，眼中闪过一丝愤怒，沉声道。妖族立天庭，他如何不知，可是他又有什么办法呢，眼前此人来此，又是为了什么。
<br/>　　“本尊可以帮你们巫族对付妖族。”万劫魔尊一点也不生气，微微一笑，淡淡道。
<br/>　　“什么？阁下不是来消遣我们的吧。”
<br/>　　“本尊可没那份闲心。消遣你巫族，你巫族算什么？”万劫魔尊毫不留情地打击道。
<br/>　　“什么？”听了此话，大多数祖巫脸上出现愤怒之色，人家都将大话在自己家门前说出来了，这如何能忍？
<br/>　　不能忍，便直接动手。这就是祖巫的直接，粗狂。更何况。若是眼前这人真有本领，还需要试过一番才知道。
<br/>　　“接我一拳，火神降临。”
<br/>　　祝融最为直接，说不成，那就打。他一拳轰出，便是无尽火海蔓延而来。火海之中，站着一尊火神，庞大的身躯，全身缠绕着焚天烈火，带着巨大的力道，向着万劫魔尊轰来，仿佛要将他轰成碎片。
<br/>　　“吸——”
<br/>　　万劫魔尊对于那滔天火海完全不在意，他张开大口，将那满天神火吞没，如今他修炼的是是《万劫魔典》，火一样是劫，火劫之力，他一样可以吸收。
<br/>　　之后，他同样是一拳轰向祝融，这一拳，他并没有使用什么神通，完全是靠自己的肉身力量，而且还不是全部的肉身力量。
<br/>　　轰！
<br/>　　两人对撞，祝融被轰出数米之远，而万劫魔尊，却是站在原地，丝毫不动。
<br/>　　“祝融，看来你不行啊。有点意思，看我的，接我一拳，水神漫天。”共工见祝融被轰走，顿时打击一声祝融，接着，对着万劫魔尊亦是一拳轰来。
<br/>　　这一拳中，包含着海浪的叠涌之力，叠浪一重紧接着一重，绵延不绝，力量却是一倍一倍地在叠加起来，越到最后，力量越强。
<br/>　　“这招有点意思。”万劫魔尊眼中闪过一丝兴趣，轻声笑道，当然，也仅仅是感兴趣而已。之后，他同样的一拳轰出，这一拳朴实无华，并没有什么技巧，而是简简单单的一拳，非常直接，此拳与之前的一拳一样，只是力量却大了许多。
<br/>　　“轰——”
