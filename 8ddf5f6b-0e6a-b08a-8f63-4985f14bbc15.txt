她还有很多事情要做，这一次来帝都，就是为了科学院，现在选拔在即，她更不能分心。

<br/>        这时有一个工作人员突然走了过来，“你好，请问是夏小姐吗？”
<br/>
<br/>        夏夕绾点头，“我是。”
<br/>
<br/>        “我们院长有请，请夏小姐到隔壁一叙。”
<br/>
<br/>        在这个工作人员出现的时候，夏夕绾就已经猜到了七七八八，科学院院长夜老在选拔之前要找她私下聊一下。
<br/>
<br/>        夏夕绾坦诚的点头，“好，请带路。”
<br/>
<br/>        ……

<br/>        夏夕绾来到了隔壁的办公室，夜老已经在里面等着她了。
<br/>
<br/>        “夜老，你找我？”
<br/>
<br/>        夜老看着眼前这个从容大方的女孩，在他面前，她没有丝毫的紧张和局促，唇角带着笑意，她静静的看着他，直奔主题，等待他先开口。
<br/>
<br/>        能让厉嫣然和他的孙女夜明珠同时感觉到不安的这个女孩子，果然非同凡响。
<br/>
<br/>        夜老好好看了夏夕绾两眼，突然就觉得这个女孩似曾相识。
<br/>
<br/>        其实刚才在门口他看到夏夕绾的第一眼，就有一种很熟悉的感觉。
<br/>
<br/>        这种感觉让他排斥，不喜。
<br/>
<br/>        “你就是夏夕绾？”夜老开口。
<br/>
<br/>        夏夕绾点头，“是的，我就是夏夕绾。”
<br/>
<br/>        “我不喜欢你，所以不管你怎么努力，我都不会让你进科学院的，打消这个念头吧。”夜老直接宣判道。
<br/>
<br/>        夏夕绾就知道不会顺利，她没想到的是这位院长直截了当的拒绝了她，断了她进科学院所有的路。
<br/>
<br/>        “为什么？”
<br/>
<br/>        “听说你一个月前刚来帝都，已经在帝都混的风生水起了，你的大名都已经传到了我的耳朵里，每个地方都有自己的圈子，就像帝都名媛圈有帝都名媛的圈子，你这个外来人突然侵入，已经打破了一种平衡，如果让你进了科学院，我有一种预感，你也会打破科学院现有的平衡，我作为科学院的领导者，绝对不会允许这种事情的发生。”
<br/>
<br/>        夏夕绾听明白了，她挺直了自己纤柔的美背，澄亮的翦瞳里绽放出了锋锐的光芒，“夜老，你是一个封建而固执的人，科学院是什么，科学院是为了医学而生，而不是你个人的领地，你想将科学院掌控在你自己的手里，不允许任何人挑战你的权威，你这样固步自封，不进则退。”
<br/>
<br/>        夜老面色大变，他豁然起身，双目震惊的看着夏夕绾，“你…你究竟是什么人？”
<br/>
<br/>        这些话，二十多年前就有人跟他说过！
<br/>
<br/>        夏夕绾看着夜老猝然大变的神色，双眼亮了亮，不过她不动声色道，“怎么，你觉得我是什么人？”
<br/>
<br/>        夜老陷入了自己的回忆里，“这些话，二十多年前就有人跟我说过，那个人就是…我的师傅！”

<br/>        什么？
<br/>
<br/>        夏夕绾刚才已经猜到夜老这么大的反应肯定是跟她妈咪有关，但是她没想到的是…妈咪是夜老的师傅？
<br/>
<br/>        天哪。

<br/>        太不可思议了。
<br/>
<br/>        “大概二十多年前，一个神秘姑娘降临了帝都城，她收了几个关门弟子，其中大弟子就是我，她传授我医术，教我建立科学院，一手创下了如今帝都医学界的秩序，迎来了繁华盛世。”
