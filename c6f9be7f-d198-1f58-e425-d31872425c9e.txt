　　“唔！看来，准圣之后，还有路可走。吾此刻的境界，已经不算准圣了，但也还没有到圣人，那么，便称它为‘亚圣’，或者‘伪圣’吧。”说着，他眼前一亮。
<br/>　　“吾如今已斩去‘逝我’与‘道我’，这算是亚圣中期巅峰了吧。”
<br/>　　元始话音刚落，只见九天之上，一道金黄色功德之光正在汇聚，风云变色，乾坤倒转，酝酿已久之后，一道庞大的功德降落下来，落在元始体内。
<br/>　　“唔？还有这等好事？”
<br/>　　元始微微一愣，他想不到开创出一个境界后，竟然使得天道有感，降下功德。他毫不犹豫便将这些功德收了起来。他估计，若是自己直接吸收了这些功德，然后再加上盘古开天留在他身上的功德，说不定可以直接成圣了。
<br/>　　但，这种成圣的方法，不是他想要的。他的证道方法是，绝不依靠任何外物证道。因为他始终坚信，最强的证道方式就是不假外物，靠自身证道。
<br/>　　同时，就在天空降下功德后，一道神秘的声音，响彻在整个洪荒宇宙生灵的脑海之中：
<br/>　　“今有玉清元始突破准圣极限，开创准圣之后圣人之前新境界，天道有感，特此嘉奖！”
<br/>　　啊？
<br/>　　这怎么可能？众修士的第一反应就是不相信，开创出新的境界，这怎么可能。要知道，每一个境界都是他人走过无数遍的，固有的境界，这种境界，早有定数，冥冥之中已经注定。如今，你却说有人打破了这个惯例，开创出新的境界，这是多么的滑稽的事，简直是滑天下之大稽。
<br/>　　若是一般情况下众修士听到此话，他们绝对会把它当作是一个笑话，一个狂妄的修士说的一个笑话。但，如今却是天道之音径直传入众人脑海之中，这就相当于天道亲自承认，这话就是真话中的绝对真话，因为天道是不会骗人的。那就是说，这是真的了，玉清元始的确是开辟出了准圣之后的新境界。
<br/>　　轰！
<br/>　　得到这一结论之后，众修士全都沸腾了。这是要逆天啊，这简直令人难以置信，教人不敢相信这一切都是真的。
<br/>　　“这怎么可能？玉清真人简直太牛了！”
<br/>　　“这是要上天的节凑啊，要与太阳肩并肩啊！”
<br/>　　“什么与太阳肩并肩？太阳很了不起吗，这是要将所有的璀璨压倒身下，这已经比太阳还要炙热，比星光还要灿烂啊！”有人这样称赞道。
<br/>　　“元始，这怎么可能？可恶！”也有看他不顺眼，羡慕嫉妒恨的。
<br/>　　“元始，是个劲敌！今后怕是元始的天下了。”有人脸色沉重无比，这样评价道。
<br/>　　当然，也有人觉得这与自己相距太远，自己顶多就是羡慕尊敬一下，便无所谓了，他牛与不牛与我何干。
<br/>　　其他人怎么想，元始无所谓，反正自己此时修为突破，距离大道又进一步，心情颇好。他神念一扫，便看到外面的老子与通天二人，于是，他打开宫门，邀请两位兄弟进来，此刻他心情好，想找两人分享一下，请他们喝茶论道。
<br/>　　“来来来，二弟，三弟，进来喝杯茶。”
<br/>　　“大兄邀请，不敢辞尔。”两人行礼做状，向着元始一拜，走了进去，更何况他们还有很多事要问一问元始呢。
<br/>　　“大兄，你刚才……”刚坐下，通天便开口问道。
<br/>　　“慢说，慢说，先喝杯茶，之后我们再论道一番，岂不美哉！”元始摇摇头，打断他的问话。接着取出茶杯，为两人斟了两杯茶。
<br/>　　之后，三人开始喝茶论道，主要是元始在讲，老子与通天在听，当然，两人有时也会提出自己的观点，三人论道，好不自在。
<br/>　　却说那被元始突破的惊天异象打断的正在大战中的巫妖两族，此时他们一个个全都阴沉着脸，脸色极为难看，倒不是因为元始的突破，而是他们发现自己两族的损失实在是太严重了。之前在大战还没有发觉，或者说，就算发觉了，当时正在大战，被煞气冲昏了头脑，放任不管，可如今被元始这一打断，他们才有机会查看各族的损失。
<br/>　　这一查看，其结果另他们心中惊颤不已，妖族百万大罗金仙十不存一，十大妖神死了三个，妖王、大妖死伤更多，更不用说金仙、玄仙之类的强者了，那更是数不尽数。
<br/>　　至于巫族，也是相差不多，十二祖巫都受了大小各自不同的重伤，大巫死伤无数，中巫、小巫更是死了不知多少。
<br/>　　这一战，谁也没有占到便宜，两族都是损伤无数，至少数十万年内，两族难以恢复元气。
<br/>　　看着洪荒之中堆积如山的巫妖强者的尸体，还有那如同河流一般的鲜血，天地破碎，湖海断流，高山崩碎，无尽生灵，遭遇不幸，天地同悲！
<br/>　　两族成员眼中流露出强烈的悲哀，他们看着对方，眼中流露出无比强烈的仇恨，满脸尽是疯狂，恨不得将对方杀后，食其肉喝其血。但，首领没有发话，他们此时也不敢妄动。
<br/>　　虚空之中，祖巫帝江与妖帝帝俊相互对峙着，他们眼中的仇恨不比其他人差，甚至更强，但他们此刻都全部忍住了，忍着不动手。由不得他们不停手，若是再如此战下去，到时两族恐怕就要退出洪荒舞台了。沉默了一会儿，妖帝帝俊开口道：
<br/>　　“还想再战吗？”
<br/>　　“要战便战，我巫族何惧之！”帝江祖巫大声喝道。
<br/>　　“恐怕你巫族损失不起吧？”妖帝帝俊沉声道。
<br/>　　“哼！”帝江祖巫冷哼一声，便不再说话，因为他知道帝俊说的是实话。巫族与妖族不同，巫族的培养极难，不仅需要祖巫精血，而且出生率也极低，故而巫族成员并没有妖族那般庞大。妖族那就不同了，只要修炼之路不断绝，便会有无数的妖族强者崛起。
<br/>　　“这事不算完，巫族，尔等全都给本帝等着，下次再战，定要斩尽尔等头颅，来祭奠吾妖族死去的无尽英灵！”
<br/>　　妖帝帝俊深深看了帝江一眼，转身带着妖族成员便要离开。因为他知道，这一战不能再打下去了，如若不然，会出现不可预测的后果。就算结果妖族胜了，那也是惨胜，更何况，妖族也不一定会胜利。妖族的兴衰此刻全系在他一妖身上，他不能赌。
<br/>　　“大哥，怎么就走了，我要将这些巫族全部杀了！”就在此时，东皇太一一脸不爽的走了过来，对帝俊质问道。此番征战，妖族损伤无数，不杀巫族，他誓不罢休。
<br/>　　“你以为我不想吗？二弟，若是我妖族再战下去，损失的就不只是这些了。”帝俊沉声道。
<br/>　　“更何况，道祖第三次讲道就要开始了，你难道想在道祖讲道期间开战？”
<br/>　　“大哥，可是……”东皇太一急声道。
<br/>　　“没什么可是的，洪荒就这么大，他们还能跑了不成，以后有机会再战。此刻，听我的，走！”妖帝帝俊打断东皇太一的话，带着妖族退回天庭，却是准备回去休养生息，等待时机，下次再战。
<br/>　　“大哥，我们怎办？”在妖族走后，其他祖巫也来到帝江祖巫身旁，问道。
<br/>　　“下次再战，我们先回去修炼，争取早日达到巫神境。走！”
<br/>　　由帝江祖巫带头，带着剩下的巫族儿郎们回到了各自的部落，之后，各族无也回到了盘古殿，回去疗伤了。
