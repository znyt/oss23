　　混沌之中，鸿钧道人与杨眉老祖对峙在一起。
<br/>　　“要如何做过？”鸿钧道人问道。
<br/>　　“贫道站在原地不动，任凭道友进攻，以此做过一场。”扬眉老祖对自己很有信心，淡淡道。
<br/>　　鸿钧道人听后点了点头，表示同意。
<br/>　　“道友请！”
<br/>　　鸿钧道人也不再说话，直接取出盘古幡一摇，顿时，无数混沌剑气迸发而出，划破混沌，地水火风涌动，打向杨眉老祖。
<br/>　　但，剑气打在扬眉老祖身上，却是一点感触都没有，好似凭空消失一般。鸿钧不信邪地再次打出几道混沌剑气，结果依旧。随后，他抡起盘古幡着实打去。
<br/>　　盘古幡却是一碰到杨眉老祖，就消失不见了，鸿钧道人大惊，这是怎么回事，盘古幡竟然不见了？
<br/>　　随后，他接连取出诛仙四剑，杀向杨眉老祖，结果依旧，四剑一靠近扬眉，便会消失不见。
<br/>　　之后，鸿钧道人取出太极阴阳图，阴阳流转，裹向扬眉，扬眉老祖此时却是单手向着虚空一划，顿时虚空裂开一道巨大口子，将太极图陷了进去，消失不见。
<br/>　　而后鸿钧取出各种宝物，全都落入了后尘，他手中除了防御的灵宝，其他的都落入了对方的手中。
<br/>　　“道友的空间神通果然强大，贫道不如也。”鸿钧道人感慨道，到了此时他若是还不知道杨眉老祖用的是空间法则，他就不是鸿钧了。
<br/>　　“哈哈！道友，如此咱们就算过了，这些还是还给道友，贫道也该去混沌中了。”杨眉老祖将宝贝全部取出，然后还给鸿钧道人，也不解释，以此来了结因果。
<br/>　　“道友客气了，此次是贫道技不如人，这些宝物应该归还道友所有才是。”
<br/>　　“不必了，这些东西对我而言并不算太重要，在道友手中比我作用大，贫道就此告辞，走也！”
<br/>　　扬眉道人说走就走，毫不犹豫，已经前往界外的混沌世界了。
<br/>　　混沌界海，才是混元大罗金仙的舞台。
<br/>　　在杨眉老祖走后，鸿钧道人手中乳白色光芒闪烁，时间停顿，岁月流转。显然，鸿钧道人不可能只有那么一点神通，他亦是有所保留。扬眉道人有空间神通，他亦有时间神通，而且，时间比空间更加神奇玄妙。
<br/>　　若不是为了早点结束争斗，以此了结天道因果，谁输谁赢还不一定呢。
<br/>　　鸿钧看了一眼混沌深处，眼中闪过一丝羡慕之情，混沌之海，他也想去啊，只是如今他已经成了天道圣人，除非超脱，否则这一辈子都只能待在洪荒宇宙之中了。
<br/>　　鸿钧道人伫立原地，久久不语。过了一段时间，才转身向着紫霄宫而去。
<br/>　　……
<br/>　　混沌之中，无数洪荒修士在艰难地行走，寻找那紫霄宫。有些没有灵宝者，只能靠自身修为抵挡，现如今已狼狈不堪，而那些逞强的早已经被混沌之气侵蚀吞没，没有留下一丁点，很是悲惨。亿万年修为瞬间化为乌有，如何不让人悲叹不已。
<br/>　　“大兄，不知还有多远啊？”
<br/>　　“不远了，马上就到。”
<br/>　　“三弟你一路之上不知问了多少遍了，你就不能耐心一点吗，不要那么急躁。”
<br/>　　“是是是，小弟有些莽撞了，二哥无需见怪！”
<br/>　　这三人自然就是元始三人了，三人之中，元始漫步于混沌之中，如同到了自家的后花园，闲庭散步，混沌之气不能侵蚀分毫。老子头顶天地玄黄玲珑宝塔，万法不侵，通天亦是头顶净世白莲，抵挡混沌之气，三人一路之上并没有遇到什么麻烦。
<br/>　　一路有着元始的推演，并没有用多长时间，就要到紫霄宫了。
<br/>　　“到了。”元始淡淡道了一声。
<br/>　　之后，脚下一道金光接引过来，三人踏上金光大道，金光流转，瞬间便到了一座巨大巍峨的宫殿之前，宫殿金碧辉煌，道韵浩瀚，匾额之上刻有“紫霄宫”三个大字。
<br/>　　元始第一眼看见“紫霄宫”三个大字，就感觉到一股浩大天地威压向他压来，他自是岿然不动，抬头向三个大字看去。第二眼，他看到的是无数大道交织汇集在一条河流之中，大道无涯，无边无尽，道韵瀚海。
<br/>　　瞬间，元始便沉浸在其中，不可自拔。
<br/>　　“大兄！”通天刚要叫醒元始，此时老子连忙打断他的话，轻声道：
<br/>　　“大兄似有所悟，我们就不必打扰他了。”
<br/>　　“哦！”通天悻悻然，之后在元始身边随便找了一个地方坐下，他并没有在那“紫霄宫”三字之中看到什么，也就没有什么领悟。
<br/>　　元始直立紫霄宫宫门前，抬头看着紫霄宫三个大字，面前如有万道河流，万道大河聚成海，但大海亦随时可以化作万道大河。
<br/>　　“嗯？竟然有人能够看见贫道的大道而悟道？！”
<br/>　　紫霄宫中，鸿钧道人大吃一惊，之后，微微一笑，道：
<br/>　　“你注定是我玄门中人，也罢，这是你的机缘，能领悟多少，就看你的了。”
<br/>　　之后，鸿钧老祖便没有多管，而是闭上双眼，参悟大道，头顶造化玉牒缓缓旋转。
<br/>　　元始周周身朴实无华，好似凡人，看不见丝毫修为。在他的眼中，一道道神轮嗡嗡浮现，无数道则翻飞，有如霞光，有如飘带，美轮美奂，道音阵阵，悦耳动听。
<br/>　　元始感觉自己仿佛畅游在大道之海中，循着道则源头徜徉而去。
<br/>　　元始身形似枯槁，仿佛毫无生机，陷入枯寂的状态之中，在他的周围，什么都没有。
<br/>　　他如同身处于混沌之中最古老的神邸，陷入浑浑噩噩的沉睡之中，似生非生，似死非死。
<br/>　　他的道则溯本回源，追随到大道的源头，一条条大道沉寂，变得再无任何威力，而是回归了本初！
<br/>　　回归了元！
<br/>　　回归了始！
<br/>　　回归道天地不曾出现时的混沌！
<br/>　　回归到没有任何大道，没有仙道，没有鸿蒙大道的地方！
<br/>　　万物寂静。
<br/>　　突然，一声轻微的心跳声传出。
<br/>　　这一声心跳如同宇宙尚未开辟之时，一切死寂，无光无色，无生无死，万籁寂静，没有任何生灵，没有任何声音之时，传来的第一个声音，是先天道音，是大枯寂大荒芜中的第一道生机。
<br/>　　咚——
<br/>　　元始的气息渐渐增长，越来越强，越来越壮大，他如同在混沌中的一尊魔神，无比古老的神。
<br/>　　当——
