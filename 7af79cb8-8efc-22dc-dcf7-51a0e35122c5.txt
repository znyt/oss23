    “先去那边看看吧。”陈兴望着远处的石砌祭坛说道。<br/>    <br/>    他刚说完，就联想到退路的问题，下意识地回过头。预想中的大门并不存在，只有一堵泛着微光的石墙。眼看没有后路，他皱起了眉头。<br/>    <br/>    似乎看出了他的忧虑，沈光明嘴角勾起弧度，安慰道，“这是一扇单向传送门，出口应该在别处。”<br/>    <br/>    事已至此，陈兴也没别的办法，只能继续朝前走。不一会儿，众人来到祭坛的前方。宫本桃没有急于带队离开，也跟了过来。<br/>    <br/>    祭坛的样式很古朴，或者说原始，下面两块作为支撑的石墩，上面一张十多二十厘米厚的石板。除此之外，既没有雕刻的图案，也没有精美的花纹，甚至没有认真打磨，表面十分粗糙。<br/>    <br/>    石面上沾满暗红色的污迹，像是干枯的血液。仔细观察，市面上还有一条条白痕，像是被利器划过。<br/>    <br/>    “解剖台。”沈光明指尖轻抚着台面的痕迹，判断道。<br/>    <br/>    “看那边，有图案。”“好几副呢。”有人眼尖，发现祭坛后方的墙壁上有些线条简单的壁画。<br/>    <br/>    走近看去，第一副壁画是祭祀的场景。一名长袍巫师站在祭坛前，举起双手，一手握着滴血的尖刀，一手握着血淋淋的心脏。台上躺着一个女人，胸腹从中剖开，脸色带着痛苦的表情，似乎还没死透。<br/>    <br/>    祭坛的下方跪着大片信徒，人数非常多，衣着各异，神态万千，直到画框外。<br/>    <br/>    陈兴发现，看到这幅画时，阿乔木的表情有些僵硬，一只手下意识地抓着心口的衣服，似乎有些不舒服。很难想象，这个心狠手辣的女人也有害怕血腥的一面。<br/>    <br/>    第二幅是制作巨型傀儡的场景。巨大的机械傀儡仰躺在地上，身上拉着固定的麻绳，一条大腿组装到一半，露出里面的零件和管线。胸腔敞开着，一个类似吊机的架子吊着一团大东西放入心脏的位置。<br/>    <br/>    第三幅是巫术师们围着做好的傀儡，念咒施法的场景。<br/>    <br/>    接下来还有很多副壁画，都是巫术师们的日常活动，没有价值特别大的信息。宫本桃看了会儿，就带着次郎团的人离开了。<br/>    <br/>    “沈军师，他们都上去了，我们不能落后啊。”性子比较急的轰雷催促起来。<br/>    <br/>    “你催什么，沈军师的脑子比我们好使，听他安排就行了。”火咀教训道。两人是同村的兄弟，该说什么就说什么，没有太多顾忌，<br/>    <br/>    听到两人的话，正在看壁画沈光明转过身体，朝轰雷点头说道，“轰雷兄弟说得对，是时候该上去了。”<br/>    <br/>    “哈哈，你看吧你看吧，军师都说我对了。”轰雷哈哈大笑，用力地拍了下火咀的肩膀。<br/>    <br/>    “看你嘚瑟的……”火咀推了轰雷一下。后者眉毛一拧，推了回去。<br/>    <br/>    而这时，阿乔木轻轻拉了下陈兴的衣袖，在他耳边小声说道，“我们不跟他们一路……”说话的同时，她看了眼正在和轰雷较劲的火咀，“我和那个大老粗合不来。”<br/>    <br/>    “你跟着我就行了。”陈兴回答，既没有同意，也没有否定。<br/>    <br/>    其实他也不打算跟雷火团一起行动，因为队伍中有个他无法掌控的存在——沈光明。<br/>    <br/>    他无法忍受行动受到制约，所以在雷火团选定上去的楼梯后，向火咀提出了分头行动。<br/>    <br/>    火咀显得有些意外，劝道，“兄弟，一起走吧，互相有个照应。”<br/>    <br/>    “陈团长是一团之长，有他自己的打算，我们应该尊重他的决定。”沈光明开口说道，似乎也不愿意陈兴留在队伍里。<br/>    <br/>    “不好意思啊，兄弟的好意，在下心领了。”陈兴顺水推舟，附和道。<br/>    <br/>    既然军师都开口了，火咀也没再劝，跟着队伍走向了侧面的楼梯口。快上楼的时候，又回头看了看陈兴，这才离开。<br/>    <br/>    “怎么走？”目送着雷火团的人离开，阿乔木十指交叉，向外翻去，伸了个懒腰，问道。<br/>    <br/>    陈兴环视了一圈，目光落在离四方队伍最远的楼梯口。<br/>    <br/>    “不去那边吗？”阿乔木看了眼黑骷髅离开的位置，提议道。她的意思很明确，就是跟在对方的后面，伺机抢夺胜利果实。这种地方可能设有机关，自己探索有不确定的危险，而且黑骷髅那批人看起来不强，他们吃得住。<br/>    <br/>    陈兴却摇了摇头，说道，“都不是笨人，肯定会设置陷阱，我们防不胜防。”<br/>    <br/>    “好吧，听你的。”阿乔木没再坚持，跟着陈兴走向另一端。<br/>    <br/>    进入拱门，楼梯蜿蜒向上，但不是很长，隐约能看到上方雾气缭绕的门廊。<br/>    <br/>    “又是人造领域。”陈兴在雾门前停下脚步，自言自语地说道，然后让开位置，示意阿乔木先走。他是射手，不是肉盾，自然应该走后面。<br/>    <br/>    阿乔木看了他一眼，没说什么，扭腰走进雾中。看着震颤的屁股，陈兴用力地揉了揉脸，跟了进去。<br/>    <br/>    下一秒，一排排高大的石像出现在眼前。一望不到尽头，十分有压迫力。<br/>    <br/>    这是一条长通道，两侧摆满了五六米高的石头卫兵，手持刀剑盾牌，或是长矛弓箭。<br/>    <br/>    石像的头部顶着天花板，代替了墙的作用。穿过石像间的缝隙，可以看见另一条通道的景象，同样站满了石头卫兵。<br/>    <br/>    身后的雾门没有消失，陈兴试了一下，可以回到一层的楼梯间。退路还在，让人安心了不少。随后，他取出突击步枪，跟在阿乔木后面，逐渐探索。<br/>    <br/>    石头卫兵的眼睛空洞无神，表情呆板木纳，却又说不出的诡异。<br/>    <br/>    “这东西能动吗？”陈兴问道。<br/>    <br/>    “没见过。”阿乔木回答。她也取出了突击步枪，端在手里。<br/>    <br/>    通道中的光线比下面暗，能见度大概在五十米左右。行走时，脚步声远远传去，有些渗人。<br/>    <br/>    “咯啦！”<br/>    <br/>    走了一段，侧面忽然传来异响，就像石头龟裂的声音。转头看去，一个石像抬脚跨前，手中的石剑斩向阿乔木。<br/>    <br/>    后者迅速退开，石剑“轰”的一声，砸在地上。烟尘暴起，半截剑身碎成石块。<br/>    <br/>    陈兴端起枪，凝神聚气，灌入灵能。<br/>    <br/>    石头卫兵一击不中，抡起石盾砸过去，目标还是阿乔木。<br/>    <br/>    “哒哒哒！”<br/>    <br/>    阿乔木身形灵活，一边射击一边躲避，子弹打在石像脸上，擦出大量火花。<br/>    <br/>    而这时，阿乔木身后的两个石像也动了，一左一右包围过去。动作虽然迟缓，但地方窄小，一下堵住了阿乔木所有去路。<br/>    <br/>    这一切都只发生在七八秒内，陈兴的蓄能还没有完成。正犹豫着要不要提前射击，却听见阿乔木朝他喊道：“不用管我！”<br/>    <br/>    话音刚落，一把石剑就砍了下来。阿乔木翻滚避开，然后一跃而起，跳上石像的手臂，再借力窜上肩膀。<br/>    <br/>    “嘭！”<br/>    <br/>    只见她旋身一脚，踢在石像的脑袋上。她脚力极大，踢打处碎石纷飞，缺了个大口子。<br/>    <br/>    另外两只石像同时抓过去。而陈兴也完成了蓄能，瞄准其中一只石像的后脑，扣动扳机。<br/>    <br/>    “哒哒哒！”<br/>    <br/>    三发蓝盈盈的子弹接连射出，石像的脑袋应声而碎。无头的石像却没丧失行动力，依然朝阿乔木抓去。<br/>    <br/>    “打脚！”<br/>    <br/>    陈兴喊道，再次蓄能。<br/>    <br/>    面对着抓来的巨手，阿乔木腰身一扭，像条泥鳅一样从石像后背钻下去。两只石像同时扑了个空，撞在一起，碎石稀里哗啦地掉下来。<br/>    <br/>    “呔！”<br/>    <br/>    落地后，阿乔木浑身泛起蓝光，一脚扫在石像的小腿上。只听见“啪嚓”的一声，石像小腿片片碎裂，身体一歪，倒在旁边的石像上。<br/>    <br/>    那石像似乎和它们不一样，没有生命，在巨大的冲击力下，直接倒塌，在地上摔成几节。<br/>    <br/>    陈兴蓄能完成，再次射击，将一只石像的小腿打爆。石像本来就笨拙，没了小腿，就只能在地上爬动。一只没头，两只没脚，互相妨碍，混乱不堪，已然失去战斗力。<br/>    <br/>    陈兴气定神闲地灌注灵能，射穿了一只石像的胸膛。像是核心被击穿，石像头部垂下，失去了动静。<br/>    <br/>    看来弱点在胸部，陈兴按照刚才的位置，射穿了第二只的胸部，却还在动。射爆脑袋后，终于消停下来。看来核心不一定在胸部，也可能是头部，或者别的地方。<br/>    <br/>    第三只石像，也就是一开始被射爆头部的那只，核心居然在右大腿上。虽然强度有限，但数量多
