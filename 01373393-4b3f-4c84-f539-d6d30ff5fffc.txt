　　凡是看到影天使视频直播的人充满着好奇，他们非常好奇视频中这个男人所说的有趣的方法是什么。
<br/>　　“两个房间都是钢筋混凝土结构，在没有专业工具的情况，想要徒手破拆是不可能的。”
<br/>　　视频中的影天使继续说道：“房间门是轻型装甲打造，可以抵御12.7mm以下所有口径的弹药攻击，但是我相信不会有人去做这么愚蠢的事情，如果你们想要让地铁站爆炸案重演的话，可以试试。”
<br/>　　影天使的话，令得罗昊皱眉。在地铁爆炸案发生后，K国政府对全国的地铁线路进行严查，也有一定的收获；不过从影天使现在的话来判断，显然K国政府目前所发现的炸药仅仅只是一部分而已。
<br/>　　威胁，赤-裸-裸的威胁！
<br/>　　但是面对影天使这样的威胁，罗昊却是感到无能为力，的确是无能为力，因为谁也不知道“暗月”到底在K国埋设了多少炸药，这些炸药又被埋设在什么地方。
<br/>　　罗昊自然是不会相信目前在K国活动的人只有智天使、吴云和SHADOW（影天使）三个人，以智天使的算计，“暗月”在K国肯定还有着其他人配合他们的行动。如果用地毯式搜索的方式去寻找“暗月”布置在K国的炸药，无疑是费时费力而且没有效率，再者从时间上而言，九个多小时，根本不可能找齐所有“暗月”布置在K国的炸药。
<br/>　　“这两扇经过特殊打造的房门就是这次对决的重点。”视频中，影天使继续说道：“这两扇门的门锁可以被写入程序，通过程序打开或者关闭。九个小时后，我和GHOST各自进入一间房间，我们会在对方房间的门锁中写入程序，时间是一个小时；一小时后，两个房间中塑胶炸药上的定时器会打开，进入十五分钟的倒计时，在这十五分钟内，我们两个人谁能率先攻破对方写入门锁中的防御程序，打开门锁就能从房间中逃出去，免于一死，而输的那个人就会在十公斤的塑胶炸药爆炸下灰飞烟灭。”
<br/>　　“还有九个小时零十分钟。”影天使斜眼扫了眼挂在墙壁上的石英钟，说道：“GHOST好好享受你生命中这最后的九小时十分钟吧，因为我，会是最后的胜利者！因为，今天之后，这个世界上再也不会有GHOST这个人！”
<br/>　　直播视频结束了，但是看到直播视频的人的讨论却远没有结束。全世界的黑客，不管是自由黑客还是有着官方背景的黑客，全都在盯着SHADOW和GHOST之间的这场对决，毕竟这可以说是黑客界最高水准的巅峰对决了。
<br/>　　错过了这一次，不知道要再过多少时间才能再看到这样的对决，很有可能再也看不到这样的巅峰对决。
<br/>　　影天使宣布的游戏规则很残酷，但是所有看到影天使直播视频的人更多的是觉得刺激。
<br/>　　这样的游戏规则实在是太让人觉得刺激，觉得血脉贲张了，黑客很多都是为了寻求刺激才跨入这个世界的。越是困难、难以攻破的防火墙越是让他们觉得刺激，越是能激起他们去挑战的欲望。
<br/>　　指挥中心内陷入一片沉默，没有人打破这种沉默，直到子午自己开口。
<br/>　　“你们这都是什么表情。”子午拍了拍手，说道：“一个个哭丧着脸，全都对我没有信心吗？SHADOW虽然厉害，但是不要忘了，他跟我交手的这几次全都是输给了我。”
<br/>　　“老师……”
<br/>　　云熙站起身来，看着子午，眼睛中充满了担忧。
<br/>　　子午起身朝着指挥中心外走去，对着身后众人摆了摆手，说道：“不用太担心，我先去睡一觉，养足精神。”
<br/>　　走出指挥中心的房间后，子午脸上的表情再也没有在指挥中心时的那种轻松。
<br/>　　这次跟SHADOW的对决真的如他自己所说的这么轻松吗？子午并不这么认为，SHADOW这次做了这么多的布置，很显然是有备而来，蛰伏了这么长时间，SHADOW不会再是之前自己所认识的那个SHADOW，他的技术肯定有很大的突破！
<br/>　　“SHADOW，你在进步，我也在进步。”子午走进电梯，看着缓缓合拢关闭的电梯门，呢喃着说道：“但是我有一个你所没有的优势，我有两个优秀的学生，而你没有！”
<br/>　　如子午和SHADOW这样的人都是非常自负的，他们的眼界很高，不会轻易收徒。当年子午是看到云熙这个小姑娘漂亮又聪明，才动了收徒的念头，把自己的本事悉数教授，可以说，云熙现在的水平已经完全不弱于自己。
<br/>　　至于林晓阳，则是因为战术B组建设的需要。加入龙隐部队这么长时间，子午也没有了最开始独自一人时那种闭门造车的思想，很多以前GHOST的独门绝技，也不再“独门”；所以子午也把林晓阳当做一个普通的学员进行教授，但是林晓阳展现出来的天赋，却是让他的水平远超其他人，成为了仅次于云熙的学员。
<br/>　　子午教过的电子信息对抗战学员有很多，但是能被他当做“学生”的，传承自己衣钵的只有云熙和林晓阳两个人。
<br/>　　子午有学生，但是SHADOW是不会有学生的，对于这个老对手，他实在太了解了。这个家伙是绝对不会收学生，把自己技术教授给其他人的。
<br/>　　所以，不管最后的对决结果是如何，自己都已经赢了一部分，不是吗？
<br/>　　……
<br/>　　八个小时后，子午回到了指挥中心。
<br/>　　罗昊、李然、李毅峰、慕容复、林晓阳都是战术背心在身，武器在手。
<br/>　　“出发吧！”
<br/>　　赵天宇目光从六个人身上扫过。作为这次行动的总指挥，赵天宇需要坐镇指挥中心，但是他也能够通过战场视频系统了解到现场所发生的全部情况。
<br/>　　六个人上了一辆七人座的商务车，在商务车前面是一辆黑色奔驰GLK，后面也跟着两辆GLK，这三辆GLK中，全都是战备一队的队员，战备一队队长郑钰亲自带队。至于为什么并不是K国方面的护卫，从地铁爆炸案事情后，赵天宇实在很难对K国的快速反应部队有所信任。
