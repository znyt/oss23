　　第608章
<br/>　　呼延力对白无双是言听计从，现在王城内的大小事务，白无双已可做一半的主，呼延力只顾着花天酒地，酒池肉林。不然的话，白无双也不能轻描淡写的杀掉蛮雄，而没有引起丝毫怀疑。
<br/>　　将呼延力从王城引出，在野外诛杀是最容易的法子，也能将牺牲降到最低。不过，到现在仍旧不能完全信任白无双。这说不得是一条计策，用蛮雄的人头引自己出来。怎么想，自己的命也是比蛮雄的人头值钱。
<br/>　　不过，这件事还有两个地方说不通，一是目前的自己值不值得白无双这么做，二来白无双想要杀自己，并不是没有机会，完全不需要这么大费周章。
<br/>　　野原火并不能完全放心这件事，但想要获得一些东西，有时候也要铤而走险。最后思来想去，野原火还是决定冒险。
<br/>　　他点了一千精锐好手，只等机会到了，便带着队伍出击，取了呼延力的人头。呼延力在戎族各部落间还是有些威望的，不过这些年他纵情声色，确实已经失去人心。只要能够除掉他，便可收伏原先臣服呼延力的诸部落。
<br/>　　如此大的收获，是值得冒险的。
<br/>　　已经是深冬，草原上一个人影也无，野兽也大多窝在洞穴里，准备熬过这个冬天。只有那些找不到食物的野兽，才流浪在草原上，寻找一些裹腹的食物。
<br/>　　这一天天气不错，阳光也有了久违的温暖。呼延力领着几十骑出了城，要在草原上打猎。
<br/>　　“长老年事已高，平素倒是不喜欢出来，这次倒是难得。”呼延力很久没出过城，偶然到野外吹吹风，显得比较兴奋。
<br/>　　他左边是白无双，右边则是手下第一高手鹰。鹰是和他一起长大的伴当，二人有过命的交情，虽然呼延力最近做事不太靠谱，但鹰是一直支持呼延力的。
<br/>　　他箭法如神，平时最是喜欢打猎。在冬日狩猎，往往是先用奔马将野兽赶出巢穴，再行射杀。不过，今日这位神箭手的状态不是特别好。原因很简单，他不仅和呼延力是从小到大的伴当，和蛮雄也是同样交情。
<br/>　　不过，蛮雄被杀了，被杀的理由很可笑。醉酒后闯入后宫，被王宫中的卫士当场击杀。这件事不用细想就有很多问题，蛮雄就算喝醉了乱窜，也没有理由会窜到王宫。而王宫的卫士再强，也不可能有能力诛杀这戎族第一勇士。
<br/>　　这件事究竟真相如何，背后又藏着怎样的阴谋诡计，鹰完全想不透。他本是直来直往的性格，只是觉得其中藏着一些东西。
<br/>　　会是什么呢？
<br/>　　他仰起头看了看四周，大队在往前进发，已快要接近女儿湖。冬天的湖水已经结冰，大家在草丛中纵马而行，口中大呼小叫。
<br/>　　不多时，有野兽从洞穴中被惊出。不过是一些野兔，黄羊之类的小物。大家随手射杀一些，倒也不是很兴奋，都在等待更大的猎物。
<br/>　　突然，不知谁惊动了一只野猪，它没头没脑的向呼延力撞过来。
<br/>　　一众围猎的人都兴奋起来，在草原上碰到野猪倒是稀罕事，不过今天大家都是为了打猎而来。本想猎取一些大物件，呼延力抽弓搭箭，一箭射中了野猪。
<br/>　　毕竟是草原上长大的汉子，最近他虽然有些荒废，但骨子里的东西却是没有丢掉。手下的勇士齐声叫好，呼延力的箭法虽然不错，不过大家还是奉承的意思居多。
<br/>　　这野猪中了一箭，但并没有伤到要害，本来好好在洞里窝着，突然被吓了出来，本就怒不可遏。如今更是愤怒，狂奔着向呼延力冲过来。
<br/>　　来势汹汹，便是呼延力有城中的好手护驾，也不敢大意。正这时候，鹰举起了手中的长弓，一箭射了出去，直接击中野猪的左眼。
<br/>　　那野猪发出一声惨叫，咚地一声倒地，这时候，鹰又射出了第二箭。他有连发箭的绝技，如果他想的话，可以连发十三箭，打的都是同一个地方。
<br/>　　这野猪站起来就往远处逃，鹰知道它已走不了多远，于是纵马缀在它后面，只要追下去，用不了多远，它就会因为失血过多而到地。
<br/>　　这只野兽，已经属于鹰。
<br/>　　一路追出三里地，野猪爬上一个草坡，终于再跑不动。鹰追了上去，正要命令人将猎物抬回去，无意间往前方看了一眼。
<br/>　　只见前方一队人马冲杀过来，来势汹汹荡起一片烟尘。
<br/>　　鹰心中一激灵，恍然明白了什么，拍马就往回走，口中高喊：
<br/>　　“敌袭，敌袭。”
<br/>　　呼延力冲远处看过去，只见人已经杀过来，瞬间将己方人马淹没。呼延力完全搞不清楚是怎么会事，他拔出刀，与敌人搏杀，内里其实心惊胆战。
<br/>　　“大王勿忧，我带你冲出去。”鹰护在呼延力前方，抬手射出一箭，正中一名敌人的咽喉。
<br/>　　他看到了对方的首领是一名独眼汉子，立刻认出了对方是谁。
<br/>　　“大王，是他。”
<br/>　　呼延力也认出了野原火，一边砍杀一边大喊：“他要干什么，他和我有什么关系。”
<br/>　　人越聚越多，鹰护着呼延力往外闯，他敢称神射手，本身功夫也是不弱。不过敌人的数量实在太多，二人一时半刻也冲杀不出去。
<br/>　　远远的草坡上，窝着两个人，程大雷和小白狼趴在草丛中，两个人都是大眼瞪小眼。
<br/>　　“大当家，他们在干什么？”
<br/>　　程大雷也是一头午雾水：“我怎么知道他们在做什么，也没人与我商量过？”
<br/>　　他的目光还是落在野原火身上，下意识皱起了眉头，脑海中觉得对方有些熟悉，只是想不起对方是谁。
<br/>　　不过这个念头一闪而过，接下来他的注意力完全被燕不归吸引。燕不归也在乱战当中，挥舞着一柄重剑格外显眼。
<br/>　　如果想要杀死对方，现在是最好的机会。本来对方的人数也不多，还互相杀来杀去，只要找到机会，自己就可以杀掉他。
<br/>　　程大雷已经握紧了剑，一动不动的注视着对方，寻找下手的机会。视线再未向野原火看上一眼。
<br/>(本章完)
