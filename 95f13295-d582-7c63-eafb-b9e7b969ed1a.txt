　　第257章
<br/>　　野草疯长，湖水清澈，站在高处望过去，天高地阔扑面而来。
<br/>　　而在水边，杀机充盈。
<br/>　　黑伞合拢是枪，张开是盾，宛若一块黑石向程大雷压来。程大雷接住了，可同时杀人书生的剑已经袭来，程大雷回剑格击，将对方逼退。赤发鬼的大刀砍向行者周霜，他不得不后退。
<br/>　　程大雷趁这个机会倒退，以身无彩凤双飞翼撤离战团。
<br/>　　连日以来，碰过诸多高手，力大如牛的马贼，身怀医术的江湖奇人，蠢强蠢强的鱼龙卫。
<br/>　　危险压榨出程大雷体内的潜力，对敌经验临敌反应都在疯长。
<br/>　　可终究没强到能碾压所有人的地步。行者，赤发鬼，杀人书生，对上其中任意两个人，程大雷都有把握不败，可对上三人程大雷就必败无疑。而战斗时间一旦拉长，体力的消耗会令程大雷的状况越来越危险。
<br/>　　杀人书生的剑袭来，他未必是个文人，但绝对是个专业的杀手。
<br/>　　赤发鬼和行者周霜同时攻向程大雷。程大雷退，再退，身后是湖水，他仗剑还击，逼退周霜，赤发鬼和杀人书生同时攻向他。
<br/>　　程大雷不得不倒退，左脚踩入滩涂中，肩膀一沉，重心已然不稳。
<br/>　　杀人书生的剑斩在程大雷的肩部，右肩，是程大雷的惯用手。
<br/>　　赤发鬼的刀逼退杀人书生，程大雷已经仰身跌入水中，他四肢挣扎，血水荡开，却定不住身子。
<br/>　　李婉儿一惊：程大雷不会游泳。
<br/>　　只见他身子在水中浮浮沉沉，最后只有头顶露在水面上。
<br/>　　见程大雷已经没有战斗力，诸人都松了口气，程大雷刚才给三人带来的压力太大了。
<br/>　　可是淘汰一人，三人间的战斗并没有分出胜负，养蛊一样，只有最好活下来的人才是唯一的胜者。
<br/>　　“先杀官。”
<br/>　　杀人书生爆喝，赤发鬼也攻向周霜。
<br/>　　一柄黑伞张开又合上，周霜防得滴水不漏，以一敌二，他竟不露丝毫下风。
<br/>　　“公主莫慌，我杀了这两个贼人，就救你走。”
<br/>　　他说得轻描淡写，似乎杀掉二人对他不算什么。
<br/>　　“下盘，攻他下盘。”杀人书生道。
<br/>　　赤发鬼立刻明白，以行者为号的周霜，弱点却是在双腿。他使出地躺刀的功夫，去斩周霜的双腿。
<br/>　　周霜果然露出紧张的表情，身体只能往后退。
<br/>　　有一柄剑突然刺出，扎入赤发鬼小腿，是杀人书生的剑。
<br/>　　赤发鬼一愣，骂道：“你……”
<br/>　　剑扎入他的胸膛，赤发鬼一句话没说话，声音就淹没在喉咙中。
<br/>　　这一幕就连周霜也是生，缓缓点头道：“你做得很好。”
<br/>　　杀人书生呵呵笑道：“我尊相爷的命令，来诛杀此贼，现在贼人已死，阁下可以护送公主离开。”
<br/>　　黑伞撑开，遮住周霜头顶，他和杀人书生的目的并无矛盾，两人都要程大雷死。
<br/>　　“请！”
<br/>　　“请……”
<br/>　　二人擦肩而过，相距三步时，几乎同时转头，向对方背心出手。
<br/>　　枪与伞在空中相撞，各自都往后退了一步，避开对方的杀伤范围。
<br/>　　“崔相还要公主的性命？”
<br/>　　“崔相不要，我不可以要么？”杀人书生咯咯笑道：“爱美墓贤之心人皆有之，小生也不例外。”
<br/>　　“呵，也得看你能不能活下来。”
<br/>　　“周霜，你当真以为我弱了你么！”
<br/>　　杀人书生出剑，剑锋如雨，刚才的杀机与现在完全不是一回事。
<br/>　　周霜此时才醒悟，刚才杀人书生一直在藏拙。
<br/>　　他撑开了伞，雨再大，终究盖不过伞。
<br/>　　杀人书生在藏拙，周霜又何尝不是，如这样的乱战，不到最后决定胜负的时刻，谁也不会把所有本领都使出。
<br/>　　杀人书生急攻，一剑比生儒袍的他显得很滑稽，因这滑稽心理难免都会轻视他。然而他的实力却一点不弱。
<br/>　　一剑一剑扎在黑伞上，这黑伞不知如何材质做的，铁剑竟然扎不怕黑伞的防御。
<br/>　　杀人书生屏息凝神，思考着破敌的方法，突然，从伞尖陡射出一蓬钢针。
<br/>　　杀人书生正想着如何攻，却没想过如何防，钢针如雨，扑面射进杀人书生脸上。
<br/>　　见血封喉的毒针。
<br/>　　他脸上露出错愕的表情，一张脸越来越黑，身体倒地，保持僵硬的姿态。
<br/>　　周霜将伞合上，看着地上杀人书生的尸体，不屑的抿抿嘴。
<br/>　　杀人书生看上去小丑，周霜看上去像个君子，但杀人书生不是小丑，周霜也并非君子。
<br/>　　江湖么，谁活下来谁才能讲道德。
<br/>　　他将黑伞合拢，从身上抽出一包钢针，一边填装机括，一边看着湖水中的李婉儿。
<br/>　　“公主，莫怕，我带你离开。”
<br/>　　李婉儿：“你先走开，我穿上衣服。”
<br/>　　周霜没有动：“此地危险重重，我不放心公主一人在这里。”
<br/>　　“狗奴才，我让你走开。”
<br/>　　“公主殿下不必口出恶言，我难道还不如一个山贼么。”
<br/>　　李婉儿一怔，忽然发现周霜看自己的眼神有些不同。她身子沉在水中，只有一个脑袋露在外面。而周霜紧紧的盯着她，似乎想要穿透水面。
<br/>　　“狗贼，你不怕死么？”李婉儿。
<br/>　　“公主可是值三座城，现在整个江湖都想要公主，杀人书生说得不错，爱美慕贤之人人皆有之。”
<br/>　　“鱼龙卫，要背叛父王！”李婉儿。
<br/>　　“本就来自江湖，何惧再入江湖，至于明帝，泥塑雕像而已。”
<br/>　　李婉儿一张脸已吓得苍白，她还未从失去程大雷的悲伤中走出，情况又是一变。
<br/>　　“哎呀，可真死呛死我了，这水真凉啊。”
<br/>　　忽然一个声音响起，浑身湿漉漉的程大雷从岸边爬上来，手中还握着剑。
<br/>　　周霜愣住了：“你会水？”
<br/>　　“校园四百米选拔赛亚军。”程大雷拍拍身上的水花：“怎么了，看你的表情好像不太对。”
<br/>　　周霜终于醒悟，一开始就假装被击败的程大雷，反而是四人之中最狡猾的一个。
<br/>　　“我已经杀了两个，不介意再杀一个。”周霜持住铁伞。
<br/>　　程大雷撇撇嘴，缓缓将剑举起。
<br/>　　“你有没有听过一招，叫做直来直往的剑法？”
