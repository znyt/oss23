        “那你……”
<br/>
<br/>        “小雪，婆婆给你送下午茶点心来了。”此时，冷夫人未见其人，却先闻其声，打断了苏漫雪欲要说的话。
<br/>
<br/>        童筱颖和苏漫雪循声望去，只见书房的大门那边，冷夫人带着一个女佣，端着四盘点心和一壶茶还有四个茶杯走了进来。
<br/>
<br/>        “老师也辛苦了，一起过来吃下午茶吧！”冷夫人微笑着说。
<br/>
<br/>        辅导老师非常客气地谢了声：“多谢夫人盛情款待。”
<br/>
<br/>        此时，童筱颖和苏漫雪也起身，朝一旁的圆桌前坐去。
<br/>
<br/>        冷夫人虽然格外偏爱苏漫雪一些，但是平常吃穿住行，也没少她童筱颖的份。
<br/>
<br/>        童筱颖看得出，冷夫人并非绝情的人，对傅昼景如此，所以，对她也亦如此。
<br/>
<br/>        冷夫人见童筱颖安安静静地吃着点心，便多嘴说了一句：“小景这些天都未回庄园来看过你。就算再忙，一个男人也不能放任老婆在家不闻不问。小景自然有他的不对，但是，你身上一定也存在问题。女人好学上进，是好事。但也不能忘记，夫妻之间要如何增进感情。”
<br/>
<br/>        “夫人教训得是，筱颖记住了。”童筱颖微微垂眸点了点头。
<br/>
<br/>        “有空就多向我们的小雪讨教讨教，我家小沉，这段日子倒是天天愿意回家来住。这还是托了小雪的福。”冷夫人顿时又看向苏漫雪，笑不拢嘴地说。
<br/>
<br/>        以前，她多么希望儿子和丈夫能陪在自己身边，但是事与愿违。
<br/>
<br/>        如今，儿子终于懂事了，知道要孝顺她这当妈的了。
<br/>
<br/>        傅夜沉每天回来，除了会去陪傅老太爷下棋，也会去她那儿坐坐嘘寒问暖。
<br/>
<br/>        是小雪改变了她家小沉那冷酷的性子。
<br/>
<br/>        其实，最主要的是，她一见小雪就有种很亲的感觉。
<br/>
<br/>        冷夫人自然是非常喜欢苏漫雪。
<br/>
<br/>        而且，苏漫雪也很会哄冷夫人开心。
<br/>
<br/>        童筱颖听着冷夫人的话，只笑不语。
<br/>
<br/>        “妈，您不要这样说啦！我都不好意思了！”苏漫雪故作娇羞地说道。
<br/>
<br/>        冷夫人忍俊不禁起来。
<br/>
<br/>        童筱颖看着她两婆媳关系融洽，也打心底地替她两感到欣慰。
<br/>
<br/>        其实，说到底，冷夫人也算她童筱颖的半个婆婆。
<br/>
<br/>        毕竟，傅昼景也是冷夫人一手带大的。
<br/>
<br/>        下课后，冷夫人的话，童筱颖自然是往心里去了。
<br/>
<br/>        傅昼景确实又冷落了她，每次她打电话过去，他不是说在忙就是说他正在开会，就连大晚上，他不是说他在加班，就是说他累了要休息了。
<br/>
<br/>        童筱颖心不在焉地吃晚饭，就连坐在她对面的傅夜沉，都看出来她有心事。
<br/>
<br/>        傅老太爷见自己的长孙盯着这小孙媳看，便轻轻地咳了一声，看向童筱颖问道：“筱颖，今日的晚餐，不合你胃口吗？”
<br/>
<br/>        童筱颖回过神来，慌忙摇了摇头，回答道：“没有，爷爷，饭菜很好吃。”
<br/>
<br/>        “那你就多吃点，补习重考这事，确实很辛苦。你和漫雪，都要多加注意自己的身体。”傅老太爷关心地说道。
<br/>
<br/>        童筱颖微笑着点了点头。
<br/>
<br/>        经过这段日子的观察，冷夫人是没发觉有什么不妥，傅老天爷阅历过无数人，自家长孙的小心思，他自然是看得出来。
<br/>
<br/>        晚饭过后，傅夜沉陪着傅老太爷下围棋。
<br/>
<br/>        傅老太爷一边摸着围棋罐里的黑子，一边意味深长地说道：“这筱颖和漫雪，长得就像双胞胎一样。若不仔细去分辨，还真是难以分清。”
<br/>
<br/>        “爷爷怎么突然说起这事了。”傅夜沉拿起白子，放下后，又围了傅老太爷五粒黑子。
<br/>
<br/>        傅老太爷看着自己孙儿的招数，不禁摸了摸下巴上的银须，会心一笑：“姜还是老的辣！”
<br/>
<br/>        傅夜沉看着傅老太爷突围，反吃了他十粒白子，不得不感慨：“爷爷在跟孙儿耍心计了！”
<br/>
<br/>        “我看，是你在跟爷爷耍心计吧！你妈虽然看不出你的心思，你爷爷我虽人老，但心不老。”傅老太爷洋洋得意地瞥了傅夜沉一眼。
<br/>
<br/>        傅夜沉抬眸对上傅老太爷的目光，微笑着反问：“爷爷倒是说说，我跟爷爷耍什么心计了？”
<br/>
<br/>        “你真正喜欢的人，是筱颖吧！”傅老太爷放下了手中的棋子，一脸认真地看着傅夜沉。
<br/>
<br/>        傅夜沉怔了怔，随即若无其事地口是心非地回答道：“爷爷老糊涂了，我喜欢的人是漫雪，不是筱颖。”
<br/>
<br/>        “呵，我看是你把爷爷当老糊涂吧！这段日子，你派人将漫雪名下所有的豪车、门面、房子和公司，全都转移到了筱颖名下。还有，每次吃个晚饭，你那双眼睛，总在筱颖身上，自己还特意挪去了偏院居住。你就怕筱颖吃不好住不好。你的心思，可以瞒过你妈，但是瞒不过我这个老家伙！”傅老太爷振振有词地说道。
<br/>
<br/>        傅夜沉沉默了一会儿，仍旧矢口否认，并一一给出了合理的解释：“漫雪名下的豪车、门面、房子和公司，是我答应阿景送给他和筱颖的新婚贺礼。我挪去偏院居住，是漫雪想要过去陪筱颖，所以我才陪着她一起住过去的。每次吃饭的时候，我只是刚好看着前方而已，筱颖又巧好坐在门边那方的位置。”
<br/>
<br/>        “你倒是能说会辩！不过，爷爷得给你提个醒。小景并非单纯的男人，他跟你堂叔冷岩峰一样，对爷爷的傅氏集团既有私心也有野心。小景虽说是你爸的儿子，但是他终究只不过是个私生子，这么多年来还委屈了你妈妈。爷爷怎么说，都不会站在小景那边。傅氏集团自然也不会交到小景手上。爷爷百年之后，小景是得不到爷爷的一份遗产。所以，爷爷只是希望，小景日后的年薪，由你来给。你能明白爷爷的意思吗？”傅老天爷语重心长地说道。
<br/>
<br/>        傅夜沉却微微皱起了眉头。
<br/>
<br/>        “至于筱颖，爷爷知道你喜欢她。本来，爷爷是想私下命令小景，让他跟筱颖离婚的。但是爷爷通过这段时间的观察，看得出，筱颖她心里喜欢在乎的人是小景。如果爷爷把一个不爱你的女人，强行留在你身边，不是帮你，反而是害了你。所以，爷爷希望，你就这样把这份爱默默地藏在心底就好，不要去打扰他们夫妻两。这是你为人处世的本分。此外，既然漫雪和筱颖长得一模一样，漫雪又那么喜欢你，你可以尝试着去接受漫雪。”傅老太爷又说道。
<br/>
<br/>        傅夜沉听后，只是淡淡地回答道：“对于筱颖的事情，爷爷真的想多了。我只是把筱颖当亲妹妹看待，仅此而已。爷爷放心，什么事情该做，什么事情不该做，孙儿自有分寸。”
<br/>
<br/>        “你懂就行，爷爷就不多说了。”傅老太爷也识趣，孙儿坚决否认他自己对童筱颖的感情，他也不好执意坚持，顿了顿后，他又语重心长地提醒道，“小沉，你要记住爷爷的话，害人之心不可有，防人之心不可无。你堂叔冷岩峰也好，你同父异母的弟弟傅昼景也好。在利益面前，你都得暗中防一防才是！”
<br/>
<br/>        “爷爷的话，孙儿记住了！”傅夜沉随即站起身来，向傅老太爷微微颔首。
<br/>
<br/>        只不过，他的话表面上虽是这样应和着爷爷的意思，但是心里却不这么认为。
<br/>
<br/>        堂叔冷岩峰，不用爷爷提醒，他一直就在防着。
<br/>
<br/>        但是，阿景的话，他和他兄弟情深，他信得过阿景，是爷爷多心了。
<br/>
<br/>        傅老太爷以为傅夜沉真把自己的话听进去了，这才稍稍放心了些。
<br/>
<br/>        傅夜沉从傅老太爷那儿出来后，又去陪了一会儿母亲冷夫人。
