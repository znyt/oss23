        好，她就去“夜惑”里，让费尔南帮他找个女人过去陪睡便是！
<br/>
<br/>        童筱颖兜兜转转，又回到了“夜惑”里，找上了费尔南。
<br/>
<br/>        “连少说，需要一个女人去傅氏集团大厦的总裁办，过去给他陪睡。才肯帮忙阻止‘夜惑’被停业整顿。所以，你给连少安排一个女人过去便是。”童筱颖说完后，伸出手来直接管费尔南要东西，“我的话已经带到了，那药是不是可以给我了？”

<br/>        “诶！别急嘛！这事情还未办成，就急着过河拆桥可不行。”费尔南起身走到酒柜前，从格子间里抽出了一瓶红酒。

<br/>        这里是他费尔南的办公室，在他费尔南的办公室里最多的不是书，而是各大品牌的红酒应有尽有。
<br/>
<br/>        童筱颖知道费尔南爱喝红酒，所以便没在意他这突如其来的开酒举动。
<br/>
<br/>        “你这跑来跑去，也辛苦了，陪我喝杯酒，我先拿一点点那药给你试试，你试过了，会用了。我再多拿点给你，如何？”费尔南将红酒倒入了两个高脚杯中，然后端起杯子朝童筱颖走去，将自己右手中的杯子，递给了童筱颖。

<br/>        童筱颖不接，果断地拒绝：“谢谢，我不喜欢喝酒。”
<br/>
<br/>        “就当是陪我喝，不行吗？再说了，我都答应先拿药给你了。你难道就不应该先赏个脸给我吗？”费尔南嘴角微扬，噙着一抹意味不明的笑意。
<br/>
<br/>        童筱颖抿了抿唇，犹豫了一下，才接过费尔南手里的高脚杯。
<br/>
<br/>        费尔南顺势跟童筱颖碰了碰杯，微微一笑：“祝我们合作愉快！”
<br/>
<br/>        语毕，费尔南举起高脚杯，温文儒雅地喝了起来。
<br/>
<br/>        童筱颖见费尔南快要将高脚杯里的红酒喝完了，于是也举起高脚杯，将里面的红酒一口饮尽。
<br/>
<br/>        费尔南顿时忍俊不禁道：“哪有你这样喝红酒的，红酒是要慢慢地滑入口中，细细地去品尝。你这牛饮，怎么能品出红酒的美味来！来来来，我再给你倒一杯！”
<br/>
<br/>        “不用了！我不喜欢喝酒！”童筱颖再次拒绝，然后向费尔南伸出手来，接着说道，“把那药给我！”
<br/>
<br/>        “药？我不是已经给你了吗？”费尔南勾唇一笑。
<br/>
<br/>        童筱颖一脸怔愣地看着费尔南，反问：“你什么时候给我呢？”
<br/>
<br/>        费尔南抬起手来，打了个响指，一手托着另一只手的手肘，指了指童筱颖的肚子，意味深长地笑了笑：“已经在你的肚子里了！”
<br/>
<br/>        “什么？”童筱颖身心一怔，下一个瞬间，她原本清澈的眸子，突然间双目无神。
<br/>
<br/>        费尔南见童筱颖不说话了，向童筱颖勾了勾手指，命令道：“小谢，今晚，你可要好好伺候‘连少’，知道了吗？”
<br/>
<br/>        “是，小谢遵命！”童筱颖面无表情，双目呆滞地看着前方回答道。
<br/>
<br/>        这种药，确实是叫“老实药”，可以让中药者，乖乖地听任何人的话。
<br/>
<br/>        那个连少要女人，自然是要她小谢，他若是换了别的女人去，万一把那连少给惹火了，让“夜惑”被火上浇油，把“夜惑”给毁了，那上头那十位大爷，岂不是要把他费尔南给宰了。

<br/>        为了自己的生命安全，也为了“夜惑”能继续营业，费尔南也不得不出此下策了。

<br/>        幸好唐倩兮已经不在“夜惑”了，若是她还在，他若是这样对待她的小姑子，她指不定会跟他闹得天翻地覆。
<br/>
<br/>        要怪，就怪这丫头涉世不深，看不懂人心的两面。

<br/>        他费尔南可不是什么慈善家，所以，他也不是什么好人，做个坏人倒是绰绰有余。

<br/>        当童筱颖被戴上白羽假面，亲自走到傅夜沉面前时，让傅夜沉眼前一亮。
