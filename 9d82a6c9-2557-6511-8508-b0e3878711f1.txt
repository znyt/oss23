　　在景区等大巴的时候，四宝突然拉住叶少阳，很严肃地说了一句：“山羊，你之前应该让那个娘炮试一试封神石的。”
<br/>　　“娘炮？”
<br/>　　“长的不是挺娘炮的吗，名字也娘炮。”
<br/>　　“试那个有什么用，他也没撒谎，我早就知道自己不是天选之子。”
<br/>　　四宝凑近一步，压低声音道：“你呀，你是不是，跟他有什么关系，合着你不是，他就一定是了？”
<br/>　　叶少阳皱眉道：“他若不是，怎么敢主动去试？”
<br/>　　“假如，是他以为自己是，但实际不是呢？你从前不也以为是天选之子。”
<br/>　　叶少阳心中一动，深思起来：慕寒对自己是天选之子，之所以这么自信，八成是张无生或道渊真人，大概早就告诉了他，然后让他保密，等到天劫降临的时候才说出来——统一法术界，带领大家对抗一切侵犯人间的黑恶势力……
<br/>　　只是，叶少阳陡然想到，如果他真的是天选之子，那陈悦呢？还有自己和冷玉，四人之中，只有一个天选之子，一个是叶法善传人（自己），一个转世鬼童，一个是轮回左手，这四个人，到底谁是谁？
<br/>　　这一刻，叶少阳才算深刻意识到了无极鬼王和酆都大帝联合下的这盘棋有多大：救世四子，也是灭世四子。双方各占两个，谁也不知道哪个是自己人，哪个是对方的人（最多知道一个），谁都不敢轻举妄动。
<br/>　　这件事……简直水太深了。
<br/>　　不过这次龙虎山之行，至少有一个收获，那就是找到了救世四子中的最后一个人，救世四子这下子齐了，将来慕寒不管做什么，都逃不过大家的眼睛。
<br/>　　而且，虽然叶少阳对这个自大的家伙非常不喜欢，不过内心也是赞同他的作法，即整合法术界的力量，对抗轩辕山。
<br/>　　上了景区大巴，叶少阳跟四宝凑在一起讨论眼下三界的局面，天劫降临，有能力搅乱三界的主要是三股势力：太阴山无极鬼王，轩辕山法术公会，天弃山尸族。
<br/>　　太阴山的对头是阴司，尸族的对头是空界联军，而轩辕山的对手便是法术公会。
<br/>　　星月奴已经斩尸，实力深不可测，还有清长风等另外几个长老，也都不是善茬，在他们的背后，很可能还站着那个隐藏级的boss：轩辕圣帝。
<br/>　　最近星月奴一干人不知道去了哪里，暂时没什么动静，但是天知道他们在鼓捣什么计划，眼下人间法术界必须团结在一起，否则可能真的会有灭顶之灾。
<br/>　　“你跟慕寒的想法，其实都一样。我能感觉出，他是想干大事的，不是投机分子，你跟他的分歧，就是谁来当这个老大。”四宝分析道。
<br/>　　叶少阳道：“你是知道的，我任何时候都没想过当老大，否则哪里等到今天，只是……他这人太狂妄冒进，我很担心他会把大家带进沟里去。”
<br/>　　四宝点头沉吟道：“这倒是，这小子看上去就自负的很，有点像道风。”
<br/>　　叶少阳摇头笑道：“他跟道风比不了，自负是需要实力支撑的，他实力是不错，但把自己想的过分强大了。”两人正讨论着，叶少阳突然发现前座一个戴眼镜的小伙从座位夹缝里回头望着他们，愣了一下问道：“你干啥？”
<br/>　　“你们是导演，或者制片人什么的？”
<br/>　　“什么鬼！”
<br/>　　“不是吗，你们刚才说的那些，什么鬼呀神呀的，不是剧本吗，或者是小说？”
<br/>　　叶少阳无语，只好敷衍了他几句，内心却哼了一声，自己要是真有这么好命，能当个普通人就好了，每天吃喝玩乐泡妹子，什么天劫之乱，什么转世鬼童，什么都不知道，什么也都不用去管了。
<br/>　　两人坐车到鹰潭市里，订了明天走的车票，然后去车站附近找了个快捷酒店住下，四宝分文没有，开房是叶少阳请的，叶少阳最近也没任何收入，但他是茅山掌教，茅山收的香火钱有他一份，虽然从没主动要过，但苏钦章每个月都发红包给他，有多有少，平均下来每个月都有快到一万块。
<br/>　　对叶少阳来说，这真是一笔相当高的收入了，也算是他当掌门之后最大的实惠。
<br/>　　尽管如此，他一向还是省吃俭用，这次就为了省钱，只开了一间房，被四宝骂了半天抠门。
<br/>　　开门进去，四宝眼尖，看到门缝下边有三张小卡片，宝贝似的捡起来，看上面的文字，一边念了出来。
<br/>　　“啧啧，还有小鲜肉，现在这社会真的发达了，哈哈。”
<br/>　　瓜瓜也爬到他身边，跟他一起看，然后两人一起研究三张卡片上的美女哪个最漂亮，叶少阳无语，知道四宝是有贼心没贼胆，懒得理会，去卫生间洗澡去了。
<br/>　　正洗着，突然感到手掌发烫，低头一看，是橙子的那条魂印在跳动，一长三短，说明有急事发生，心下一惊，赶紧激活魂印，将位置告诉她，然后飞快地冲了淋雨，刚把浴巾裹上，面前就出现了一道虚空裂缝，接着橙子便冲了出来。
<br/>　　“咋啦？”叶少阳一手提着浴巾，朝她身上看去，头发是散的，嘴角流着血，身上衣服还破了少许，眼神有些涣散。
<br/>　　“老大救我！”橙子一下扑到叶少阳怀里。
<br/>　　虚空裂缝立刻消失，但在完全消失之前，有一双手从里面伸出来，抓住了裂缝的两边，然后一只硕大的脑袋挤了出来，头上生着双角，跟个夜叉似的，手里捧着一件明晃晃的兵器，二话不说，照着橙子后心就刺下去。
<br/>　　叶少阳箭步上前，把橙子护在身后，一把抓住这把兵刃，霎时一股阴寒之气朝自己体内蔓延，当下打了个激灵，运气作法，将一身罡气逼入双手，敌住阴气，倒灌入兵刃，朝那邪物身上蔓延过去。
<br/>　　“啊！”
<br/>　　邪物哆嗦了一下，扔掉兵刃，双手朝叶少阳刺过来。
<br/>　　叶少阳单手结印，拍了过去，将邪物震飞。
<br/>　　后面又有邪物从裂缝挤出来，有鬼有妖，一个比一个长的奇怪，也不说什么，跟日本鬼子似的，刚出来就朝叶少阳扑过去。
