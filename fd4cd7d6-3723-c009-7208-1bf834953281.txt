　　然后有一个尖细嗓门的说道：“得嘞，咱爷几个先去避避雨！”
<br/>　　说话之间，几道人影骑着马闪进了破庙的外间。叶少阳定睛看去，这些人都穿着粗布汗衫，高矮胖瘦都有，但脸上五官有一个共同的特点：凶神恶煞。一看就不是善类。衣服右侧腰间一个个都鼓鼓的，好像是别着家伙。
<br/>　　土匪……
<br/>　　叶少阳跟陈三互相看了一眼，陈三也吓得不轻。这时候翠云听见动静也醒了，看见外面来了这么些人，吓的刚要开口，叶少阳一把捂住她的嘴，让她往后面点。
<br/>　　“三当家，三当家！”一个土匪先拴好马，往庙堂里走，一眼看到了叶少阳三人，怔了一下，去拉扯身边那个戴斗笠的。
<br/>　　很快，所有人都发现了叶少阳三人，三当家的一马当先，从腰间拔出了盒子枪，瞄准三人，恨声问道：“做啥子的？”
<br/>　　陈三立刻上前抱拳，“大爷，大爷，咱三个是赶尸的先生，路过宝地，这不是下雨走不了吗，我们这里生了火，大爷们快进来烤烤衣服。”
<br/>　　三当家的听他这么一说，神色缓下来，把盒子枪放下，招呼兄弟们进去。
<br/>　　陈三立刻拉着叶少阳和翠云往后面让，给他们挪地方。
<br/>　　这一群土匪一共有五六个，进来之后，立刻霸占了火堆和蒲团，直接把衣服都脱下来，一个个只穿着大裤衩，在火堆前烤衣服。
<br/>　　翠云又羞又怕，缩到叶少阳身后去了。
<br/>　　土匪……叶少阳还是第一次见到真正的土匪，跟电视剧里演的那些不太一样，延续举止间没有那么夸张，不过这些人在坐下之后，叶少阳立刻敏锐地觉察到了一股股的杀气飘过来，说明这些土匪是真杀过人的，而且不少。
<br/>　　不过这些土匪虽然残暴，但有一点，不会随便杀人，因此才会不介意三人在场。
<br/>　　一个土匪瞥了缩在叶少阳身后的翠一眼，笑道：“还有姑娘啊，出来给爷看看长啥模样。”
<br/>　　翠云一听，浑身颤抖。好在那三当家的开口说道：“老子进门就瞄了，腰粗的跟冬瓜似的，满脸麻子。”
<br/>　　几个土匪一听，顿时也就失去了兴趣。
<br/>　　三当家的一边烤火，一边问陈三有没有东西吃，陈三连忙从背包里摸出干粮，外带五块大洋，送了上去。“请大爷赏脸，喝茶，喝茶。”
<br/>　　三当家的很满意，接过钱和干粮，瞟了他一眼说道：“你是公堂义庄的吧，总听说你们赶尸，这是头一回遇到，你一个跑腿的马仔，我也不为难你，你回去给你们当家的带个话，俗话说见山拜山，让他没事多上上贡，招子放亮点，不然下回再碰上你们，给你们来个三刀六个洞，让你们长长规矩。”
<br/>　　陈三连忙赔笑，各种点头。
<br/>　　几个土匪不再理他，开始讨论下山之后的安排。叶少阳三人在旁边听着，知道他们是去赶绑票的勾当，虽然愤怒，但也无可奈何，只求雨赶紧停，这些家伙好赶紧走。
<br/>　　叶少阳一只手捏着七星龙泉剑，准时准备应付突发情况。
<br/>　　几个土匪吃了东西，衣服也烤干了，但外面雨还么停，不由有点无聊，不知道是哪个坑爹的带的头，走到墙边上去挨个看那些尸体。
<br/>　　这些都是一贯杀人越货的，根本不害怕尸体，只是好奇。
<br/>　　三当家的手里玩着一把小刀，挑起了一具尸体脸上的灵符，看了一会，斜眼看了陈三一眼，道：“我说，这些尸体，真的一听你们念咒就会走？”
<br/>　　“啊，当然，我们就三个人，总不能背着这些尸体。”陈三赔着笑，小心翼翼地回答。
<br/>　　“这还真是奇了，来，给爷几个展示一下！”
<br/>　　陈三顿时哑口无言，望着叶少阳，叶少阳一时间也不知道怎么办。只好拱手说道：“这位爷，尸体一动，就不好消停，还请高抬贵手，大家都不容易。”
<br/>　　“去你妈的！”
<br/>　　叶少阳目光一寒，心中动了杀机，不过考虑到陈三和翠云的安危，一时间没行动。毕竟这些人身上都有枪，而自己要照顾两个同伴，眼下只能先忍了。
<br/>　　那个三当家的虽然骂了他一句，但也没有坚持，自顾自地一具具尸体看下去。然后，让叶少阳担心的事发生了：那个三当家的站定在那具女尸面前，上下一打量，嘿嘿笑道：“身段还挺好的吗，不知道脸长的怎么样。”
<br/>　　一抬手把女尸面前的灵符揭了下去。
<br/>　　叶少阳和陈三根本来不及阻拦，陈三看看叶少阳，叶少阳摇了摇头，女尸身上一共有三道符，被撕下脸上的一道，也没太大问题。
<br/>　　不过，那个三当家看到女尸长相之后，立刻就被吸引了，跟前几个土匪也围上去，一通围观。
<br/>　　“这姑娘好特么俊啊！”有土匪赞叹起来。
<br/>　　三当家的呆了片刻，伸手捏了捏女尸的脸，弹性十足，跟活人一样，又仔细看了一会，冲着叶少阳三人吼道：“这特么是死人！”
<br/>　　陈三连忙摆手说道：“回大爷的话，这当然是死人，活人怎么可能不动呢？”
<br/>　　“刚死的？”
<br/>　　“对对，刚死不久。”陈三顺着他的话说。
<br/>　　三当家皱眉又打量了一会这具女尸，又捏了一把脸蛋，叹道：“可惜了，长这么俊。”
<br/>　　几个土匪嬉皮笑脸地走过去，跟着一起捏女尸的脸蛋，还有手贱的甚至扒女尸的衣服，灵符是靠着灵力贴在衣服上的，被这土匪一抖，两张灵符陆续掉下来。尸体一下子倒了下去。
<br/>　　“把灵符贴上！”
<br/>　　叶少阳情急中走过去。
<br/>　　几个土匪不知道怎么回事，听他突然喊了一嗓子，吓了一跳，有两个掏出了盒子枪，对准了叶少阳。
<br/>　　叶少阳只好不动了。
<br/>　　就在这个时候，他看到一道鬼影，从对面墙上的窗户飘了进来，朝着女尸飞去。
<br/>　　那个一直潜伏在附近的厉鬼！
<br/>　　叶少阳刚想拔剑，看到对面两个土匪手中漆黑的枪管，犹豫了一下，就在这犹豫的工夫，那鬼影飘进了女尸的体内。
