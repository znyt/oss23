　　这一道声音的突然出现，便是将在场所有人的注意力，都给吸引了过去。
<br/>　　众人似乎都没有料到，在这云羽天宫之中，居然还会有着这样一道声音的突然响起。
<br/>　　无数视线望了过去，便是看见就在这云羽天宫的彩虹阶梯之上，一道身影突然从中走了下来。
<br/>　　“有人？”
<br/>　　看到这人影的一刻，楚风眠都有些惊讶。
<br/>　　云羽天宫，已经是在九域之中消失了足足七万年的宗门，谁能想到，在这云羽天宫居然还有人的存在？
<br/>　　七万年的时间，在强大的武者都不可能活得下来，就算是曾经位列九域巅峰的无敌人物，都可能活到七万年之久的时间。
<br/>　　不止是楚风眠，在场的无数人看向这人影的到来，都是惊讶无比。
<br/>　　“欢迎各位，来到这云羽天宫之中，七万年来，你们是第一批客人，老朽也已经等待多时了。”
<br/>　　这一道身影落下，是一尊满头白发的老者，看起来苍老无比，但是却无人敢小看这老者的存在。
<br/>　　他们可是眼睁睁看着这老者，是从彩虹阶梯之上落下来的，上面的七彩霞光，对着老者没有任何意义的话。
<br/>　　这便是代表着，这老者的身份，至少是跟云羽天宫有着极大的关系。
<br/>　　云羽天宫七万年的岁月，都没有开启过，现在这老者的出现，谁又敢小看？
<br/>　　就连楚风眠看向这老者的时候，眼神都是极为凝重，以他现在的灵识，居然都一眼无法看出这老者的身份。
<br/>　　甚至是就连他的境界，楚风眠都无法看出来，一眼望过去，这老者就跟那种手无缚鸡之力的平凡老人一样。
<br/>　　但是这里，可是云羽天宫，一个手无缚鸡之力的老者居然会出现在这里？绝无可能。
<br/>　　“请问阁下是什么身份？”
<br/>　　还是荒圣君看到这人影，突然率先开口道。
<br/>　　荒圣君率先开口，在场的无数武者便是也都已经将视线，看向了那老者。
<br/>　　在场的所有武者，也都跟楚风眠一样，都想要知晓这老者的身份。
<br/>　　“老朽，不过是这云羽天宫遗留的一个管家罢了，诸位到此前来，也都是为了云羽天宫的宝藏来的吧。”
<br/>　　老者笑了笑开口道。
<br/>　　“云羽天宫，七万年没有开启，你们是第一批进入其中的武者，不错，云羽天宫的所有宝藏，甚至是传承，都留在了里面不错。”
<br/>　　“宝藏！”
<br/>　　“管家？莫非这云羽天宫，是他所开启的？”
<br/>　　“其中会不会有诈？”
<br/>　　“无论如何，这云羽天宫我们都要闯一闯，谁能够得到其中的传承，在九域之中，都会无敌！”
<br/>　　在场武者，不少人都是一个个的讨论起来。
<br/>　　这老者的出现，的确是让他们的心中，有些忌惮，但是在这云羽天宫的宝藏面前，没有任何一人，会选择退缩。
<br/>　　“恩？器灵？”
<br/>　　就在楚风眠思考之际，在他的脑海之中，星玄的声音，突然响起。
<br/>　　一直以来沉睡的星玄，如今突然苏醒过来。
<br/>　　“器灵？你说什么？”
<br/>　　听到星玄的话，楚风眠一时间有些不解询问道。
<br/>　　“怎么，小子，你没看出来吗？你面前这个老头，不就是一位器灵吗？”
<br/>　　星玄语气有些嘲弄的开口道。
<br/>　　“本来以为你知道的挺多的，没想到这点眼力都没有。”
<br/>　　眼前这老者，是器灵？
<br/>　　楚风眠猛然之间，看向这老者。
<br/>　　一眼望去，这老者的身上，的确是没有任何灵力的感觉，甚至是没有武者生命的感觉，似乎是有些虚无缥缈。
<br/>　　这种感觉，的确是跟器灵，极为相似。
<br/>　　只是一个器灵？居然会变幻的跟武者一样？
<br/>　　一般器灵的出现，几乎所有武者，都可以一眼认得出来，毕竟器灵身上的气息，跟武者的截然不同。
<br/>　　可是如今在场的所有武者，都没有认出来，眼前这老者，居然会是一尊器灵。
<br/>　　就连楚风眠，若是没有星玄的提醒，他都没有看出来这一点。
<br/>　　“一个器灵，居然足以变化成跟武者相同的样子？”
<br/>　　楚风眠惊讶之余，也是无比震惊，这要是多强大的器灵，才可以变化成现在跟武者几乎一模一样的样子。
<br/>　　同时这云羽天宫，可是足足毁灭七万年了，就算是器灵，都不一定能活下去七万年之久的时间。
<br/>　　“不用惊讶，这家伙的本体，已经是接近到了极道仙兵的地步，所以别说是七万年，就算是十万年，他都不会毁灭。”
<br/>　　星玄的声音从楚风眠脑海之中响彻到。
<br/>　　“没想到你的运气真不错，这种灵器，居然都被你碰到了。”
<br/>　　接近极道仙兵的灵器？
<br/>　　星玄的这个评价，楚风眠眼神都是无比激动。
<br/>　　他的祖龙至尊剑，战龙宝车，在天级灵器之中，都足以称得上是其中的佼佼者。
<br/>　　但是这样，这两件灵器也依然达不到接近极道仙兵的地步。
<br/>　　毕竟极道仙兵，可是跟天级灵器截然不同的存在，比起地级灵器跟天级灵器之间的鸿沟，还要巨大。
<br/>　　“这云羽天宫之中，究竟是什么宝物，才可以诞生如此强大的器灵？”
<br/>　　楚风眠思来想去，也不记得，这云羽天宫之中到底是记载过有着什么强大的灵器。
<br/>　　似乎没有任何一件，可以得到这种品级才对。
<br/>　　除非。
<br/>　　楚风眠突然睁开眼睛，望向天空之上的云羽天宫。
<br/>　　“莫非这老者，就是这云羽天宫的器灵？”
<br/>　　楚风眠看着天空之上，足足是有着数百座宫殿，组成的云羽天宫，突然开口道。
<br/>　　“看来你还不笨。”
<br/>　　星玄冷哼一声开口道。
<br/>　　“这次你算是撞到了大运，这一次云羽天宫开启，这老者显然是打算给云羽天宫，找一个传承者，你要是运气好，也许你就可以得到这云羽天宫的传承，倒时候这整个云羽天宫，都是属于你。”
<br/>　　整个云羽天宫的传承！
<br/>　　楚风眠的眼神，都是炙热无比，这云羽天宫的传承，绝对是比起任何一切宝物，都要有用的多。
<br/>　　“你们想要得到云羽天宫的宝物吗？”
<br/>　　老者看着下方众人，突然大笑连连起来道。
