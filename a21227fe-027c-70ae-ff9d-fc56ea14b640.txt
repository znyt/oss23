　　说到底，真正杀慕容槐的还是四长老。
<br/>　　两人的战斗本是平手，处于平衡，萧逸只是打破了这个平衡，才让四长老势如破竹，一举杀了他。
<br/>　　玄阶中级武技之威，岂是单靠血肉之身能挡的。
<br/>　　若是慕容槐能稍稍用真气抵挡一番，崩山斩绝对伤不了他，毕竟萧逸与他的境界修为差距太大。
<br/>　　怪只能怪慕容槐自己太过大意。
<br/>　　“萧逸。”四长老忽然看向萧逸，道，“之前在议事厅，你对我打眼色，示意拖延时间时，我就感觉到自己背后有一股奇怪的力量，不过数分钟，便把体内黑墨曼陀罗的武魂力量消失了。”
<br/>　　六长老也面露疑惑，道，“我也有这种感觉，到底怎么回事？”
<br/>　　“那是控火兽的力量，能够焚烧其他人的武魂特效。”
<br/>　　萧逸随口找了个借口，冰鸾剑以及太阴太阳之眼都不是普通东西，绝对不能轻易托出。
<br/>　　不是信不过两位长老，只是，在自己有足够的自保之力前，这些底牌都不能泄露。
<br/>　　四长老没有怀疑萧逸的话，点点头，道，“控火兽虽然是最垃圾的武魂，但终归是有其作用的。”
<br/>　　六长老连忙道，“萧逸，赶紧帮执事和族人们恢复修为吧。”
<br/>　　“不。”萧逸摇摇头，道，“人数这么多，我帮他们全部解决黑墨曼陀罗的压制，不知要耗费多久的时间。反正，要不了几个时辰，压制效果就会消失，他们会自动恢复实力的。”
<br/>　　萧逸面露急色，道，“刚才听那慕容槐的意思，慕容家是准备和萧家开战了，我得赶快回家族。”
<br/>　　四长老笑道，“臭小子，你是在担心依依吧。”
<br/>　　“放心吧。”六长老拍了拍萧逸的肩膀，道，“家族里有大长老和二长老坐镇，慕容家的人翻不起风狼。”
<br/>　　萧逸摇摇头，“我放心不下，总感觉会有什么不好的事情会发生。”
<br/>　　四长老道，“你这是关心则乱。”
<br/>　　萧逸摇摇头，“我在矿脉周围巡查一番，看还有没有慕容家的人躲起来，而后立马动身回家族。”
<br/>　　两位长老见萧逸脸色凝重，便没有多说什么，开始四处检查起来。
<br/>　　毕竟现在所有族人都没有实力，若慕容家的人伺机埋伏，矿脉里的族人将会很危险。
<br/>　　待得两位长老离开，萧逸走到慕容槐的尸体上，将他的武魂力量吸收。
<br/>　　之后，萧逸回到议事厅，分别在那八个执事的身体上摸索一番，顺便将他们的武魂力量一并吸收。
<br/>　　幸好太阴太阳之眼的漩涡乃是无形的，肉眼无法看见。
<br/>　　而议事厅内的萧家执事又暂时修为全失，无法感知到萧逸在做什么。只以为萧逸在慕容家执事们的尸体上搜索乾坤袋。
<br/>　　萧逸感受了一下，体内的控火兽武魂，仍旧只是黄色，没有进阶，但颜色深了许多。
<br/>　　赤橙黄绿青蓝紫，七色武魂，每一个阶级间，差距都极大，没那么容易进阶。
<br/>　　绿色武魂，整个紫云城都没人拥有，也没有出现过。可想而知，绿色武魂必然很强，也很难领悟，力量比之黄色武魂强了太多。
<br/>　　萧逸猜测，若是能再吸收十个八个黄色武魂，应该就能进阶了。
<br/>　　大概在周围搜查了一番，没有发现慕容家的人，萧逸便准备离开了。
<br/>　　这时，四六长老来到萧逸面前，道，“萧逸，你先走一步。我们在这里看着族人们，等他们恢复实力了，我们立马赶回家族。”
<br/>　　现在矿脉内的族人都修为尽失，自然需要人保护。
<br/>　　“好，我便先走一步。”萧逸点点头，立刻出发。
<br/>　　四长老吹了个口哨，一匹骏马嘶吼一声，快步跑来。
<br/>　　值得一提的是，这匹骏马，正是之前萧逸骑来矿脉的那头。虽然萧逸那时丢下它自己赶路，不过它还是自己跑回了矿脉。
<br/>　　萧家的骏马，全部经过训练，自己会认路。
<br/>　　“骑马赶路太慢了，我自己跑回家族就行。”萧逸说了一声，脚下气爆声惊人，几个呼吸间已经离开了矿脉。
<br/>　　“好快的速度。”四长老看着萧逸已经消失的身影，惊讶道，“这小子，对武魂的利用还有火焰的操控力，远超我们的想象。”
<br/>　　六长老笑道，“要不然怎么是武道天才呢，你惊讶的声音，只是表达出你是个蠢货罢了。”
<br/>　　“你骂谁是蠢货...”四长老脸色一黑，与六长老争执了起来。
<br/>　　.....
<br/>　　陨星山脉，在紫云城南面百里处。但这是直线距离，中间还横亘着不少林子和高山，真正的路途，大概在数百里以上。
<br/>　　所以，就算是骑着骏马赶路，也差不多要一天时间。
<br/>　　虽然骑马更轻松，但萧逸不想等这个时间。不知为何，他如今心中一阵急躁，好像会有什么不好的事情会发生。
<br/>　　这是一种直觉，是对未知危机的预判。
<br/>　　但，萧家乃是紫云城三大家族之一。而且，在家主失踪多年的情况下，仍旧没有其他势力敢染着，证明萧家实力之强。
<br/>　　特别是大长老和二长老的存在，更是让萧家无人敢撼动。
<br/>　　萧逸曾经听闻过一个传言，多年前，江家曾趁着萧家家主失踪，欲大举进攻萧家。但最后，大长老以一己之力，便让整个江家不战而退，彻底打消了念头。
<br/>　　萧家，是三大家族中唯一拥有两个先天九重武者的家族。其余两家，江家与慕容家，都只有家主是先天九重。
<br/>　　正常情况下，萧家绝对不会有事。
<br/>　　但，萧逸更加相信自己的直觉，他的直觉很少出错。且这次的直觉如此浓郁，让他如此心急如焚，便证明更加错不了。
<br/>　　所以他迫不及待地要赶回家族。
<br/>　　全力赶路，甚至连升龙都用上了，他的速度飙升到极致，如同一道火红色的疾风飞奔着。
<br/>　　大概在两个时辰后，萧逸终于赶回了紫云城。
<br/>　　刚入城，萧逸明显感觉到城中的气氛很不对劲，一股压抑和肃杀飘荡在空气之中。
<br/>　　忽然，一道巨大火刃冲天而出，将紫云城上空炸出一道惊雷之声。
<br/>　　巨大火刃冲出之处，正是萧家府邸。
<br/>　　“不好，果然开战了。”萧逸脸色一凛，脚步一动，立刻往家族而回。
