　　大食人虽然对大唐南海水师有所顾忌，但战事一开，那些大食人的确是显得凶悍异常，面对来势汹汹的大唐战舰，为了给同伴争取时间，立刻有二十余艘木质战船迎了上来。
<br/>　　长弓短弩向着大唐战舰的甲板上抛射而至，甚至有些大食人的战船上面还装备有小型的投石机和小型床弩，乱箭齐发之下倒也颇有些威势。
<br/>　　尉迟宝琪在命令舰队撞毁了数只大食人的战船之后，便停止了这样杀敌一千自损八百的举动，毕竟战舰虽然是铁的，比大食人的木头船结实不少，可这也是相对而言的。
<br/>　　这就像后世的汽车一样，汽车是铁的，你对汽车丢石头可能是会砸出一个坑，可是如果你用汽车去撞东西，就算是撞完之后还能开，车身也会产生一定程度的变形。
<br/>　　一次两次或许没有问题，可多撞几次之后就会发现，车身强度不行了。
<br/>　　战舰也同样如此，偶尔撞一下或许没有问题，但是你接二连三的撞，就大唐的钢铁拼接工艺来说，战船根本无法承受如此大力的撞击，几次之后估计龙骨就会变形，到时候战船只怕就要报废了。
<br/>　　另外还有一点就是大唐的战船是用螺旋桨驱动的，那东西十分脆弱，一直这样撞下去，天知道会不会有什么东西卡在上面，万一有东西把螺旋桨卡住，到时候只怕又是不小的麻烦。
<br/>　　所以当发现敌人已经开始反击的时候，尉迟宝琪果断下令，所有战舰火力全开，对大食人展开真正的进攻。
<br/>　　一群大头兵早就已经等的迫不及待，听到命令之后立刻蜂拥而出，各自冲向自己的位置，数十门新式火炮掀开身上披着的外衣，露出狰狞的獠牙。
<br/>　　仅仅用了片刻时间，那些大食人的战船已经基本上冲到了钢铁战舰的边上一块块被打磨过的石头被丢到战船上，发出一声声沉闷的响声。
<br/>　　可是钢铁战舰毕竟是钢铁战舰，石头和床弩的攻击对于它来说与挠痒痒没有什么区别，被砸中之后除了掉点漆之外，连个印子都没有留下。
<br/>　　发起反击的大食人瞬间就懵了，他们无论如何都想不到大唐的战船竟然是特么铁的，铁的啊！钢铁怎么可能浮在水面上呢？这东西不是应该见水就沉么？
<br/>　　事实上，在他们的印象中，大唐的船就是大，所谓身大力不亏，靠着这一个‘大’字，大唐的战舰才能有如此强悍的战斗力。
<br/>　　可是当战争真的发生时，当钢铁战舰真的出现在大食人的面前时，这些大食人甚至连反应的时间都没有。
<br/>　　“咻咻咻……”就在大食人愣神的那么一瞬间，大唐战舰的船舷边突然出现数个人影，人影的肩膀上似乎扛着什么东西，可还没等他们意识到什么，那些人影身上扛着的东西已经吐出一条白烟。
<br/>　　接着，大食人的战船船身微微一抖，惊天的爆炸声伴随着升腾而起的火焰，一艘战船随之停了下来。
<br/>　　一艘，两艘……，大唐的攻击来的太过突然，而且又太不合常理，没有强弩的对射，没有阵天的喊杀声，就那么消无声息的射出一个东西，然后就是一阵爆炸，接着船就开始漏水，漏着漏着船便开始下沉。
<br/>　　为什么？凭什么？凭什么大唐人的铁船都没有沉，而自己的木头船却深了？这到底是为什么？
<br/>　　无数大食人带着无尽的怨念被海水吞噬，更多的大食人因为逃走及时，漂浮在海面上。
<br/>　　“舰炮准备完毕，随时可以发射，将军！”战事一开，钢铁战舰上所有人都兴奋的要命，副将更是脸都涨的通红。
<br/>　　“那还等什么，全力进攻，把这些大食人全都送到海底下去。”尉迟宝琪瞪着牛一样大的眼睛说道。
<br/>　　这家伙想当年就是跟着‘海狼’一起出来的，海战经验非富，这几年‘海狼’去了倭岛，他对陆战没有兴趣便回了长安照看自家老头子。
<br/>　　可是大家都是年轻人，看着别人在外面仗，天天捷报频传，自己却只能缩在家里奶孩子，伺候老爹，这种感觉如果说不憋屈那才是怪事。
<br/>　　所以在知道南海这边‘打起来了’之后，尉迟宝琪第一个响应号召，反正他家老头子也已经跟着太上皇去外海玩儿去了，他在家中也没有意思，他大哥还在天竺那边驻守，一时半刻也回不来，索性凭借着老‘南海’的资历，抢到了这个任务。
<br/>　　要知道，现在如今真可以称得上四海升平了，满大唐不管是属国也好，邻国也罢，在陆地上根本就没有人敢站出来挑衅的，因为全都被打怕了，动不动就灭人全国的法让所有人都小心翼翼的与大唐保持着关系，生怕引起大唐的不满情绪。
<br/>　　所以别看大唐那几个驻外的将军得得瑟瑟牛气的紧，但实际上回去之后也是每天闲的数腿毛玩。
<br/>　　这次好不容易遇到一个敢跟大唐叫板的二货，长安城几乎都要吵翻天了，为的就是争一争这个领军的名额，尉迟宝琪能来那也可以说是杀出重围才得到的机会。
<br/>　　副将领命离开，时间不大，船头以及船尾的数门新式炮台开始移动起来，炮口转向装弹，炮手拿着大锤站到炮位之上，狠命一轮，‘哐’的一声。
<br/>　　“轰轰……”巨炮发出一连串的轰鸣，震的人肝儿都在发颤，不过爽啊！
<br/>　　再看那些还在借机四下散开的大食人战船，随着那一连串的轰鸣声明显顿了一下，接着就看到对方的战船之间炸起无数的水花，其中有三艘比较倒霉一些的战船竟然直接就被炸成了漫天的碎木头。
<br/>　　大食人彪悍归彪悍，可是再彪悍那也有个限度，人只要不傻那就不会用脑袋去往铁上撞。
<br/>　　可能说这些大食人在面对奔马的时候敢冲上去，面对大象的时候也敢冲上去，可是当你面对一炮就可以把几百人送上天，而且打动，咬不烂，甚至连看都看不到的敌人时，就算他们再彪悍又能怎么样？
