　　“吱呀~”
<br/>　　身穿紫色西服的中年男子刚写说些什么，却被一阵开门的声音打断，几人都是警戒的转过身来。
<br/>　　门口站着一名喰种，脸上带着月山集团标志性的红色面具，但他们不记得有安排人员在监控室布防。
<br/>　　一旦打起来，这边就会成为指挥室，只要他们几个在这里就已经足够了。
<br/>　　要知道他们也是S级的喰种，等闲敌人根本不可能突破进来。
<br/>　　“咦，有什么事情吗，还是……”紫西服向前走了几步，有些疑惑的开口，话语说到一半却戛然而止。
<br/>　　一颗头颅冲天而起，掉落在地面上，滚动了几下，浓郁的血腥味瞬间将监控室覆盖。
<br/>　　此时他们也反应了过来，这个戴着红色面具的喰种，并是不是月山家的下属，在他们印象中月山家根本没有如此强大的人物。
<br/>　　几乎一个照面，就秒掉了一名S级喰种，更加可怕的是，对方的动作他们根本没有看清！
<br/>　　光是从速度上分析，敌人就已经远超大部分高阶喰种了，能做到这一步的，最起码也是SSS级。
<br/>　　剩余的两名喰种脸色一沉，连忙摆出防御的姿态，可惜在如此巨大的实力差距下，他们的反抗根本起不到任何用处。
<br/>　　几秒钟的时间，两人先后被摘取了头颅。
<br/>　　甩掉了拳套上的鲜血，刘哔对三具喰种尸体进行了吞噬，随后，用西服外套将三颗头颅串起，挂在了房门拐角处。
<br/>　　他不太喜欢杂乱的环境，既然要在这里呆一段时间，那他也不介意随手收拾一下，让自己更加舒心一些。
<br/>　　只是那满地的鲜血，却没时间进行打扫了，透过监控室的电脑屏幕，他看到了陆续涌进来的一群白鸽。
<br/>　　对于这次行动，CCG给予了足够的重视，在这群白鸽里面，刘哔发现了很多熟悉的身影。
<br/>　　在众多白鸽身上查探了一圈，最终他的目光停留在领头的白衣男子身上，有马贵将，他果然来了。
<br/>　　……
<br/>　　美林大厦二十四楼，木嶋式带着自己班级的搜查官们，正在对此楼层的喰种进行清理。
<br/>　　即使是在市中心，这栋楼也算非常高的了，足足30多层的高度，让人很难搜索到这次行动的目标。
<br/>　　但是木嶋式并不担心，这些事情自有其他人来解决，他只需要保证自己的任务不出差漏就可以了。
<br/>　　“旧多君，可不要因为地上的血滑到哦。”手持一柄电锯形状的库克因，木嶋式将眼前的喰种一分为二，调笑了旧二多福一句。
<br/>　　这孩子，是第一次参加这种大型行动，他想用这种玩笑般的话语，来缓解自己下属的紧张。
<br/>　　“啊，是！”旧二多福乖巧的站在木嶋身后，不时出手协助其击杀周围的喰种。
<br/>　　这些喰种，实在是太多了，密密麻麻的，光是这一个楼层，就有近三十只，这还没算上那些藏在暗处没有动手的。
<br/>　　惨叫声充斥着整个楼层，地面和四周的墙壁上，洒满了浓稠的血液，受到血腥味的刺激，剩余不多的喰种都疯狂了起来。
<br/>　　事实上，不光是木嶋班，几乎所有的楼层的状况都是这样，无数的搜查官和喰种厮杀着，生命在这个地方，变的无比廉价。
<br/>　　所有人，都在用最快的速度清理楼层，然后向联络器上指定的地点集结着。
<br/>　　但是喰种实在太多，大部分的搜查官，都被死死挡在了下面，也只有像木嶋式这样个人能力十分强大的搜查官，才能提前清理完楼层，和先锋组汇合到了一起。
<br/>　　和其他楼层比起来，三十八楼相对来说很是安静，虽然下方厮杀声震天，但这里却没受到任何影响。
<br/>　　“松前，CCG已经攻下了下面的楼层，先锋是有马贵将，我们一起，去将他们阻止到下面。”走廊尽头的房间里，一名身穿棕色西装的喰种，站在月山习身旁。
<br/>　　与他交谈的，是月山习的侍卫，也是她将月山习从别墅中撤离出来的。
<br/>　　“我明白了，习少爷，来接您的直升机很快就要到了，请您在屋顶稍等片刻。”目光微微闪动了一下，松前从月山习身前走过，向屋外走去。
<br/>　　他们心中清楚，这次下去，可能再也没有机会上来了。
<br/>　　但是从小被月山观母培养，为的就是这样一刻，这是他们的职责。
<br/>　　有马贵将，他们也十分好奇，所谓CCG的白色死神，到底强大到了什么地步。
<br/>　　“松前，为什么不带面具？”月山习站在原地，脸色有些悲切，作为曾经活跃在20区的S级喰种美食家，他自然知道有马贵将这个名字代表着什么。
<br/>　　但他，没有选择。
<br/>　　为了让他活下去，家族牺牲的已经太多了，他必须接受，用这些已经几乎算是亲人的下属性命，来换取自己生存的希望。
<br/>　　“能向我发誓吗，能和我约定，你们一定会回来我身边吗？”蠕动了几下嘴唇，他感觉自己的话语如此苍白。
<br/>　　听到月山习的话语，即将要迈出房间的两人，顿住止住脚步，松前的肩膀微微颤抖了起来。
<br/>　　片刻，收拾好自己的心情，松前转过身来，伸出了自己的右手。
<br/>　　这是只有她们才能互相看懂的动作，曾经，她们用这个动作，做了无数个约定。
<br/>　　她从来都没有失约过，至少在习面前，没有。
<br/>　　强忍着鼻腔的酸意，松前平静的看着月山习走上前来，心中一颤，艰难的收回了右手。
<br/>　　“我无法和您做出约定。”将头微微垂下，松前的声音有些颤抖：“习少爷，请您一定要好好活下去，我这条命，本来就是为了你而存在的。”
<br/>　　“请您，千万不要白费了。”鞠了一躬，松前转过身来，决绝的走出了大门。
<br/>　　这是她第一次拒绝月山习的要求，如果不出意外的话，应该也是最后一次了。
<br/>　　虽然很无礼，但对于自家少爷来说，她一直当做自己的亲生弟弟一般疼爱，而这次同样，她也会拼尽全力的去保护他。
<br/>　　她将会用自己的生命，为月山习争取到逃亡的时间。
