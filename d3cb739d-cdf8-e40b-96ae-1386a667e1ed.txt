　　药王宗前山，
<br/>　　“桀桀！看来今天还真是个好日子，不但那个叶狂仙自投罗网，连火神宗的余孽也全部到齐了！”
<br/>　　药王宗宗主云天青，望着被众人围在中央的一名青年男子，满脸的得意之色。
<br/>　　这是一名，满头红发的青年男子，整个人周围，被一团炽热的火焰所包裹，气势颇为不弱，竟然不在药王宗宗主和四大长老之下！
<br/>　　“云天青！你杀我父母，残害我同门，今日我要让你血债血偿！”
<br/>　　那青年男子暴喝一声，直接凌空一步迈出，瞬间便已经冲到云天青身前百米，然后狠狠一拳砸了出去！
<br/>　　轰隆！
<br/>　　这一拳砸出，顿时一条火龙，自青年男子的身体中发出，直奔那云天青而去。
<br/>　　“哼！”
<br/>　　云天青脸上露出一抹凝重之色，当即冷哼一声，也同样一拳迎了上去！
<br/>　　轰隆！
<br/>　　一条青色巨龙虚影，自云天青身体中发出，威势还在对方之上。
<br/>　　只不过，这青龙遇到火龙之后，顿时便处于了下风。
<br/>　　轰隆隆！
<br/>　　两人的拳劲相撞，周围空间轰轰作响。
<br/>　　最终竟然是旗鼓相当！
<br/>　　很显然，明眼人都可以看出来，若论单修为的话，自然是云天青更胜一筹，只不过他所修炼的是木系功法，这青年男子所修炼的，则是火系功法。
<br/>　　火系功法杀伤力强，木系功法则偏向于温和，而且火系功法对药王宗的毒功，也有极大的克制作用。
<br/>　　所以，云天青虽然修为要比对方高深不少，却也只斗了一个半斤半两，甚至还还稍微处于劣势。
<br/>　　云天青先是脸色一变，继而冷冷一笑，道：
<br/>　　“桀桀！不愧是火神宗的少宗主，潜龙榜首祝明飞！想不到你年纪轻轻，竟然就已经突破了元婴境！”
<br/>　　“倘若你一直躲着不出来，再勤加修炼几十年，说不定还真能成为，蓬莱仙界千年以来，第一个突破天君境界之人！”
<br/>　　“可惜啊！你为了你那个废物弟弟，竟然不惜以身犯险，既然你今日来了，那我药王宗说什么也留你不得了！”
<br/>　　云天青说完这话，冲着其余众位长老使了一个眼色，众人纷纷上前，开始对那青年男子呈包围之势，显然是打算围攻了。
<br/>　　原来，
<br/>　　这一头红发的青年男子，赫然就是祝小白的哥哥，十年来一直稳居潜龙榜首位，号称蓬莱仙界第一天才的火神宗少宗主，祝明飞！
<br/>　　祝明飞环视了一圈周围众人，脸上也露出一抹凝重之色，
<br/>　　以他的实力，也就和云天青旗鼓相当，若是再加上四大长老，他说什么也不是对手。
<br/>　　更何况，药王宗四大长老之上，还有一位实力更加恐怖的云千秋！
<br/>　　“云宗主，今天其他四大宗门的宗主也都在，祝某只想问你一句，我火神宗到底犯了什么错，你们为何要屠杀我们宗门之人，你们药王宗眼中，可还有王法吗？”
<br/>　　原来此时，周围除了药王宗的众高手之外，其他四大宗门的宗主，也已经全部抵达药王谷，此刻正远处冷眼旁观。
<br/>　　可惜，面对祝明飞的质问，其他四大宗门，只是远远的看着，丝毫没有一点要插手的意思。
<br/>　　云天青似乎也早就料到会是这么一个结果，脸上顿时露出一抹嘲弄之色，
<br/>　　“祝明飞，你还真是天真啊！修真界向来都是弱肉强食，你火神宗不过区区一个二流宗门，竟然敢挑衅我药王宗的威严，这便是天大的罪过！”
<br/>　　其余四大宗门的宗主们，听到这话，也顿时纷纷露出一抹冷笑，依旧冷眼旁观。
<br/>　　祝明飞见状，一颗心顿时沉到了谷底，脸色也变得越发冰冷，
<br/>　　“好好！就算是如此，你们如今已经灭了火神宗，又何必要赶尽杀绝？”
<br/>　　“我弟弟小白，不过才十四岁而已，又有什么罪过？你们抓他做什么？无非就是想引我出来罢了！”
<br/>　　“你们药王宗，堂堂五大宗门，竟然做出如此卑鄙下作之事，就不怕被世人耻笑吗？”
<br/>　　云天青和四大长老，顿时脸色微微一变，
<br/>　　说起来，此事如果传扬出去，的确有损药王宗五大宗门的形象。
<br/>　　不过，云天青两眼一转，已经计上心来，冷冷笑道：
<br/>　　“祝明飞，你可要搞清楚，你弟弟祝小白，可不是我们抓来的！”
<br/>　　“明明是他勾结世俗界的歹人，对我们药王宗图谋不轨，害死了我们药王宗的公孙长老，和十几名弟子！”
<br/>　　祝明飞两眼一瞪，
<br/>　　“这不可能！我弟弟从来没有进入过世俗界，怎么会勾结世俗界的歹人？你们欲加之罪，何患无辞？”
<br/>　　云天青冷冷道：
<br/>　　“那个世俗界的歹人，如今被我们困在万毒窟之中，我药王宗数千名弟子，全都是见证！”
<br/>　　“祝明飞，你弟弟勾结世俗之人，在我药王宗行凶杀人，这个罪过他可是逃不掉的！”
<br/>　　说到这里，云天青忽然两眼一转，计上心来，话锋一转道：
<br/>　　“这样吧！祝明飞，只要你愿意进入万毒窟，将那人的人头取来，本宗主倒是可以考虑饶他一条性命！”
<br/>　　其余四大长老听到这话，顿时便明白了云天青的意思，嘴角纷纷露出一抹阴笑。
<br/>　　万毒窟九死一生，只要祝明飞答应，他们便可以不费吹灰之力，而且名正言顺的，将这个蓬莱第一天才除掉！
<br/>　　祝明飞眉头紧皱，
<br/>　　虽然他也隐隐猜到了云天青的用心，可是眼下他似乎也没有更好的办法。
<br/>　　论实力，仅凭他一人之力，万万无法与药王宗相抗衡。
<br/>　　而云天青提的这个条件，显然是救他弟弟祝小白的唯一办法！
<br/>　　更何况，周围四大宗门的巨头大佬们也都在场，只要他做到云天青的条件，谅这云天青好歹也是一宗之主，应该也不至于会反悔。
<br/>　　“好……”
<br/>　　就在祝明飞一咬牙，准备答应下来的时候，
<br/>　　忽然，
<br/>　　一道犹如神雷般的声音，陡然在天空上响起，
<br/>　　“药王宗的杂碎们，你们还真是够不要脸的呀！”
<br/>　　药王宗的众人，听到这道声音，慌忙循声望去，继而脸色全都为之大变，
<br/>　　“那小子，竟然活着从万毒窟出来了？这怎么可能！”
<br/>(本章完)
