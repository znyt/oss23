        “筱颖，好久不见。”季思妍走到苏漫雪的跟前停住了脚步，她迎上苏漫雪的目光，淡淡微笑，唇边浮出不易觉察的嘲弄，“你变漂亮了！”
<br/>
<br/>        “噢？难道我以前不漂亮吗？”苏漫雪轻笑了一声。
<br/>
<br/>        傅昼景身边的秘书是吗？
<br/>
<br/>        可她怎么觉得她身上有股子“狐狸精”的气息。
<br/>
<br/>        苏漫雪轻蔑地将季思妍打量了一番。
<br/>
<br/>        季思妍总觉得“童筱颖”不对劲，又不知道该从何说起，于是释然一笑，回答道：“那倒没有，以前的你，从来不化妆。难得见你化了妆，又改变了自己的穿衣风格。突然间，你成熟了几分，自然就比以前更漂亮了！”
<br/>
<br/>        “你也是啊！长得挺漂亮的！”苏漫雪微微一笑。
<br/>
<br/>        季思妍抿了抿红唇：“我们去咖啡馆坐坐，傅总有事情交代给我，是让我替你办的。”
<br/>
<br/>        “什么事？”苏漫雪不禁好奇地问。
<br/>
<br/>        季思妍微微一笑：“先上车，去了咖啡馆，我们边喝咖啡边聊。”
<br/>
<br/>        “好。”苏漫雪二话不说就答应了。
<br/>
<br/>        季思妍在市里挑了一家档次高档的咖啡馆。
<br/>
<br/>        苏漫雪去得最多的地方是酒吧和夜店，像这种咖啡馆、茶馆什么的，她还是第一次来。
<br/>
<br/>        所以，季思妍点的是什么咖啡，苏漫雪便照着季思妍的姿态，跟服务员点了一杯一样的。
<br/>
<br/>        季思妍有些难以置信地看着“童筱颖”，她人还是这个人，声音也是这个声音，怎么性格上就……
<br/>
<br/>        “说吧！是什么事？”苏漫雪坐在软皮椅子上，翘起了二郎腿，双手抱臂，直接开门见上地问。
<br/>
<br/>        她的话，打断了季思妍的思绪，季思妍随即从自己的手提包里，掏出几份合同和资料，朝“童筱颖”递了过去。
<br/>
<br/>        苏漫雪一脸茫然地看着季思妍递过来的合同和资料，怔了怔后，故作淡定地接过手。
<br/>
<br/>        “傅总虽然是傅氏集团旗下地利置业的总经理，但是傅总经过自己的努力，在外创立了自己的公司。本来，公司的法人应该是傅总他自己的名字才对。但是，傅总说，他创立公司是为了你，所以，这家公司是你的了。”季思妍微笑着说道。
<br/>
<br/>        而殊不知，是她和傅昼景合伙创立的公司，被工商局的领导派人下来暗查了。
<br/>
<br/>        她和傅昼景利用了傅氏集团与政府的合作关系，私吞了傅氏集团的“远东城军机部项目”，幕后控权四方，将所有收益纳入他们自己囊中。
<br/>
<br/>        上头的人虽然暂时还未查到她和昼景的公司上来，但是她知道，他们很快就会查出来。
<br/>
<br/>        所以，季思妍得在他们查出来之前，有所行动才行。
<br/>
<br/>        这家公司的法人名字不能是她季思妍，也不能是他傅昼景。
<br/>
<br/>        那么，只有她童筱颖的名字最合适。
<br/>
<br/>        童筱颖和傅昼景是合法夫妻，公司又是在他们婚后成立的，那么到时候上头查出了问题来，大可说，是童筱颖利用傅昼景在傅氏集团的职务便利，创立自己的公司，私下做的那些“大动作”。
<br/>
<br/>        毕竟，童筱颖是傅昼景的合法妻子，就算牵扯到傅昼景，季思妍到时候便给傅昼景帮腔，说是她童筱颖一人所为，傅昼景完全不知情。
<br/>
<br/>        若是没被查出来，童筱颖只不过是个挂名法人，公司的盈利，仍旧是她和昼景的。
<br/>
<br/>        季思妍心里打着如意算盘。
<br/>
<br/>        苏漫雪将合同和资料一起翻看了一遍又一遍，完全看不懂这白纸黑字里写的到底是什么意思。
<br/>
<br/>        “你给我看的这些东西，到底是用来做什么的？”苏漫雪噘了噘红唇，纳闷不已。
<br/>
<br/>        季思妍意味不明地笑了笑，并给“童筱颖”递了一支笔：“怎么会看不懂呢？很简单的啊！只要你在这里签上你的名字，把你的身份证和银行卡拿给我。傅总的这家公司，就是你的了！”
<br/>
<br/>        “噢！这个我懂了！”苏漫雪咧嘴一笑。
<br/>
<br/>        这不和上次，她在临海城让刘叔帮她收购她和筱颖曾经实习的那家小装饰公司的流程差不多嘛！
<br/>
<br/>        苏漫雪毫不犹豫地从季思妍的手中接过笔，正准备签上自己的“苏漫雪”的名字时，停顿了一下，才写上了“童筱颖”这三个字。
<br/>
<br/>        这么好的事情，居然跟她一点关系也没有。
<br/>
<br/>        苏漫雪觉得自己得拿到一些钱，才合适。
<br/>
<br/>        就在季思妍准备收回“童筱颖”签好的合同和资料时，“童筱颖”却突然伸手压住合同和资料，不让她拿回去。
<br/>
<br/>        “你该不会是对我使了什么诈吧？”苏漫雪漫不经心的扯了下红唇。
