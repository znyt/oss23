        苏漫雪恨透了童筱颖，为什么她童筱颖能得到傅夜沉的心，她苏漫雪却得不到。
<br/>
<br/>        隐约间，连华生从苏漫雪身上闻到了一股子的酸味，而这股子的酸味倒是让他纳闷了。
<br/>
<br/>        苏漫雪为什么要吃童筱颖的醋呢？
<br/>
<br/>        连华生心里想着，两只贼溜溜的眼睛，下意识地看向傅夜沉。
<br/>
<br/>        他莫名其妙地觉得，傅夜沉和这两个女人之间有着什么他这当兄弟的都不知道的故事。
<br/>
<br/>        不过，真的是越来越有意思了。
<br/>
<br/>        冷冷清清的家，一下子因为来了这两个女人而变得格外热闹起来。
<br/>
<br/>        “筱颖，既然你常年与药草打交道，那么有几个问题，我可否请教你。”连华生不再跟苏漫雪套近乎，转眼间又看向童筱颖，说起了自己感兴趣的事情。
<br/>
<br/>        童筱颖赧然地笑了笑，腼腆地回答道：“请教就不敢当了，不过，我可以把我知道的都告诉你。”
<br/>
<br/>        “那去我实验室里坐坐，咱们好好聊聊。”连华生微微一笑，抬起手来指向了楼上，接着向童筱颖迎了迎手，“请！”
<br/>
<br/>        童筱颖抿了抿嘴，点点头就跟着连华生去了。
<br/>
<br/>        就在连华生经过傅夜沉的身旁时，傅夜沉快速低语道：“不要对筱颖下手。”
<br/>
<br/>        连华生忍俊不禁地应了声：“放心吧！我对已婚妇女不感兴趣，我只是对她所知道的药草知识感兴趣罢了。”
<br/>
<br/>        已婚妇女，这四个字，就像四根长针一样“突突突突”地扎入了他傅夜沉的心脏。
<br/>
<br/>        似乎所有人都在提醒他，筱颖已经是他的弟妹了，让他不要再对筱颖痴心妄想。
<br/>
<br/>        傅夜沉只觉心痛，所以沉默不语。
<br/>
<br/>        苏漫雪倒是意外，连华生竟然也住在这里，而且在这里还有他的实验室！
<br/>
<br/>        连华生的来历虽然让苏漫雪感到好奇，但是她的直觉告诉她，这个连华生不是个好惹的角色。
<br/>
<br/>        在这个家里，刘婶似乎不大待见她，童筱颖对她的态度大不如从前那般亲密，傅夜沉对她也是冷冰冰的，而对连华生，她还得处处提防着才是。
<br/>
<br/>        苏漫雪觉得自己在这个家里像是被孤立了一样，这令她心里很不是滋味，而这一切，都是童筱颖加之在她身上的。
<br/>
<br/>        以后，她一定会在童筱颖的身上慢慢讨回来。
<br/>
<br/>        傅夜沉见连华生和童筱颖一起上楼去了，伸手就将自己的胳膊从苏漫雪的手中抽了出来。
<br/>
<br/>        苏漫雪不依不饶地再次挽住傅夜沉的胳膊，嘟起红唇，柔声柔气地说道：“夜沉，今晚让我跟你睡，好不好？人家一个人睡觉，好怕！”
<br/>
<br/>        “滚！”傅夜沉再次甩开苏漫雪的手，看都不看她一眼，直接下楼朝大门口走去。
<br/>
<br/>        “夜沉，这么晚了，你这是要去哪里啊？”苏漫雪瘪了瘪嘴，急得直跺脚。
<br/>
<br/>        傅夜沉完全不理会她的话，自顾自地出了门。
<br/>
<br/>        童筱颖跟着连华生进了他的实验室里后，被实验室里的陈设震惊得目瞪口呆。
<br/>
<br/>        墙壁上的白板上，还写着一些她看不懂的化学元素分解公式；一排排整齐的白色架子上，还摆放着一罐又一灌的人体内脏标本；好几张桌子上，化学用试剂瓶，大大小小，长长短短，一一在置物架上陈列。
<br/>
<br/>        就连窗户那边的两排长方形的白色铁丝笼里，还养了数只可爱的小白鼠。
<br/>
<br/>        童筱颖对小动物很感兴趣，刚走到笼子前准备弯腰凑过去逗小白鼠的时候，被连华生给叫住了。
<br/>
<br/>        “你不要离它们太近，会影响我的实验。”连华生好心提醒道。
<br/>
<br/>        童筱颖这才直起身子，后退几步，远离连华生的实验小白鼠。
<br/>
<br/>        连华生忍俊不禁道：“你和那位大少奶奶虽然长得一模一样，这性格倒是截然不同。”
<br/>
<br/>        “就算是双胞胎，也会有不同的性格啊！”童筱颖微笑着说。
<br/>
<br/>        连华生看着童筱颖，忽然想起了什么，便直话直说地感慨：“可惜啊！你已经结婚了……傅家那位二少爷，我倒是没见过。不过听刘婶提起过，说他性格比较孤僻，因为是私生子的缘故，那位二少爷心里有点儿自卑，很早就从傅家大庄园里搬出去住了，他似乎是巴不得和傅家脱离了关系就好。”
<br/>
<br/>        “我不觉得昼景性格孤僻，也不觉得昼景会因为自己是私生子而感到自卑。他很努力，很温柔，心肠也很好，他有自己的理想，而且他很坚强，重情重义。”童筱颖不假思索地反驳了连华生的话。
<br/>
<br/>        连华生不禁会心一笑：“你这到底是二少爷的女人，一心都向着他。”
<br/>
<br/>        “因为，他是我老公啊！”童筱颖一提到傅昼景，她那娇俏又美丽的脸蛋上，总会露出甜蜜的笑容。
<br/>
<br/>        连华生看着童筱颖脸上那无邪的笑容，突然想起了那句“向天真的女孩投降”的话。
<br/>
<br/>        每个女孩，心里其实都向往一段美好又单纯的爱情，因为向往了，所以真正爱上了才会在爱情面前，将自己最天真无邪的一面展现出来。
<br/>
<br/>        但如果，这美好而单纯的爱情被毁灭了……
<br/>
<br/>        连华生忽然间回过神来，发现自己想远了后，对童筱颖笑着说道：“祝你和二少爷幸福。”
<br/>
<br/>        “谢谢。”童筱颖微笑着颔首道谢。
<br/>
<br/>        连华生接下来拿了一些他不知名的药草让童筱颖帮他辨认，童筱颖并未认全，剩下的几样她不认识的，便去房里取来了外婆留给她的《千草集》和那四大古医书，帮连华生查阅。
<br/>
<br/>        幸好，连华生带回来的药草，童筱颖在外婆的手札《千草集》里找到了学名和功效。
<br/>
<br/>        连华生向童筱颖借过她手中的那本破旧的手札看了看，不禁好奇地问道：“这本东西你是哪来的？”
<br/>
<br/>        “我外婆自己编的，她还写了行医日记。”童筱颖回答道。
<br/>
<br/>        连华生黑白分明的眼珠子一转，客客气气地问：“那可以把你外婆的那本行医日记也借给我看吗？”
<br/>
<br/>        “当然可以！”童筱颖二话不说就答应了，然后转身回自己屋里取来了外婆写的那本行医日记交给了连华生。
<br/>
<br/>        连华生从童筱颖手中得到这两本手稿后，在书桌前坐下，仔仔细细地翻看了起来。
<br/>
<br/>        童筱颖见他一脸认真，全神贯注的样子，都不好意思再打扰他了，只好独自一个人欣赏着他实验台上那些盛满了五颜六色液体的试剂瓶。
<br/>
<br/>        连华生看了多久，童筱颖就在他的实验室里逛了多久。
<br/>
<br/>        童筱颖只是随便看看，哪怕再好奇，也未对连华生的东西动手动脚。
<br/>
<br/>        连华生大致地看了一遍后，问道：“你外婆是苗医，既能医人也医兽？还会巫蛊之术？”
<br/>
<br/>        “嗯。”童筱颖回过神来，看向连华生点了点头。
<br/>
<br/>        “这两本手稿，我可以借走拿去复印一份吗？我保证，不侵犯你外婆的著作权。”连华生非常诚恳地询问。
<br/>
<br/>        童筱颖并未一口答应，而是蹙着柳眉犹豫了。
<br/>
<br/>        连华生见状，又急急地补充了一句：“要不，你开个价，多少钱，我都买！只要你能给我复印一份！”
<br/>
<br/>        “我不是那个意思，我只是觉得，你认为这两本手稿对你有用的话，拿去复印便是。而我希望我外婆的手稿能帮助到你，救治更多的病人。你除了不侵犯我外婆的著作权以外，我还希望你不要用于商业盈利，就是那种批量贩卖我外婆的药方。我知道，我外婆的那本行医日记里，有一些疑难杂症，对症下药的配方。”童筱颖深沉地说，顿了顿后，又一脸无奈地接着说道，“可惜，我未能继承我外婆的衣钵。”
