　　叶少阳点点头。
<br/>　　陈三怔了半天，过去检查了一下几具尸体，确定没有异样，这才松了一口气，在叶少阳边上坐下来，感慨了一番之前发生的事情。
<br/>　　“幸亏叶天师你机灵，不然我们几个真的要交代在那了……”
<br/>　　翠云仍然惊魂未定，吓的手发凉，叶少阳一直握着她的手安慰她。
<br/>　　“只可惜那女尸被鬼附身了，被土匪抢走了，这下子麻烦很大。”
<br/>　　叶少阳道：“有什么麻烦？”
<br/>　　“没法跟苦主交代啊，赶尸的把尸体弄丢了，这可真是……”
<br/>　　“那个苦主也不是什么好人，我还要找他麻烦呢。”叶少阳沉吟了一下说道，“你这样，既然人形煞不在了，你一个人赶这些普通尸体，应该没问题吧，不如你先把这些尸体赶走，然后等着我。”
<br/>　　陈三和翠云听他这么说，一下子都懵了，忙问他要干什么。
<br/>　　叶少阳解释起来：“这具女尸，是人形煞，一旦解封，煞气外泄，很快就会成煞，那个女鬼自以为捡了个好身体，其实不然，煞气外泄，会裹住占领躯体的灵魂，到时候这女鬼想离开身体都不行了，人形煞没有灵智，但跟一般僵尸一样，有杀人吸血的本能，到时候女鬼反而会成为傀儡，被煞气感染，到处杀人吸血，用以修炼。我是法师，这种事让我遇到，不可能不管。”
<br/>　　陈三傻傻地看着他，说道：“你意思，她会把这些土匪都杀光？”
<br/>　　“是这样，人形煞一旦成型，再加上那个女鬼的灵智，成为鬼尸，实力大增，普通枪支弹药对她没用，那些土匪必死无疑的。”
<br/>　　“这……叶天师，恕我直言，这些土匪都是打家劫舍杀人放火的王八蛋，就算都死了也没什么关系，你难道还想去救他们不成？”
<br/>　　叶少阳瞪了他一眼道：“你这话说的，好像她杀光了土匪就完事了似的，她杀光了这些土匪，必然会下山去找别的活人吸血，到时候修为越来越强，会成为为祸一方的祸害，再想剿灭就不容易了。”
<br/>　　陈三恍然大悟，“那，叶天师你打算怎么办？”
<br/>　　“我偷偷上山，找机会灭了这鬼尸。你先赶尸去交工，至于这具人形煞的苦主……你这样，你就说因为这尸体邪性，有更厉害的法师赶过来，让他等我几天，最迟三五天我就过去。”
<br/>　　翠云问道：“那我呢？”
<br/>　　“你……你跟三爷走，到地方等我，你放心，我绝不会有事，一定去平安找你。”
<br/>　　翠云各种劝他一起走，但叶少阳晓以大义，翠云也没办法。
<br/>　　“那我就陪你留下！我是你姐，你去哪我去哪！”翠云咬着牙说道。
<br/>　　叶少阳心中大动，捏着她的手说道：“姐。不是我不让你留下，我要去的是土匪窝，有你在身边，实在不方便，姐你放心，我说了去找你，我什么时候让你失望过。”
<br/>　　又是这句话。翠云心中一动，抬头凝视着他，最后看了一眼边上的陈三，嗫嚅道：“我不想跟他一起。”
<br/>　　叶少阳想起陈三之前对翠云有点意思，献过殷勤，翠云当然也感觉的到，于是走到陈三跟前，一只手搭着他的肩膀，盯着他，严肃地说道：“我把我姐交给你了，我相信你，你不会欺负她，否则的话……”
<br/>　　陈三听他这么说，立刻正色说道：“叶天师啊，我这人虽然有点不着调，不过咱们也共患难过，都是自家兄弟，我陈老三能干出那种事吗，我保证把翠云妹子给你带到，万一出事，我提头来见！”
<br/>　　叶少阳听他这么说，心中也很感动，陈三的人品，自己还真的是放心的，当下拍了拍他的肩膀，说道：“不用这样，你们俩都不要出事，到地方等我就行。”
<br/>　　陈三拿出一份地图，给叶少阳画了路线和终点。“你按照我给你画的走，到平南县，找一家二道坝客栈，我跟是翠云妹子，就住在这间客栈等你。”
<br/>　　叶少阳答应下来，跟翠云依依道别，然后催促他们上路。
<br/>　　翠云嘱咐了他很多，又给他分了足够的干粮，这才一步三回头地跟着陈三，赶着尸体离开。
<br/>　　叶少阳目送他们走远，自己在树林里思考了一会要怎么办，本想乔装上山，但又觉得不妥，想想还是偷偷摸上山，找机会解决了那具人形煞。
<br/>　　叶少阳回到之前那条路上，这时候那几个土匪早不见了，那个三当家的尸体也被弄走了。
<br/>　　叶少阳害怕有埋伏之类的，不敢走山路，钻进边上的树林里，一路上山，走了有个把小时，快到山顶的时候，隐约能看到山顶的建筑物，是一道木质的围栏，有三四米高的样子，将整个山头都围起来，在围栏的后面，依着山势修建着一些建筑，很多都是木结构或者干脆就是竹楼，看上去像个寨子。
<br/>　　怪不得土匪的老窝一般都叫山寨，估计就是这么来的，叶少阳猜测，因为土匪窝一般都在山顶，材料不好运上山，而且因为身份的缘故，也不好大张旗鼓地去采购，只好就地取材，多用木头竹子之类的。
<br/>　　在山寨的正面，直通那条山路的斜坡上，居高临下地修建着一座很大的城门，当然只是大，因为是木制的，也没什么威严的感觉。
<br/>　　城门上方，有两个背着枪的土匪站在上面放哨。叶少阳看了一下四周的环境，从正面闯进去的话，肯定会被当成枪靶子，不过叶少阳想着这么大一座山峰，肯定不会滴水不漏，总会有地方能绕进去，于是朝山寨的左侧走去，临走前瞄了一眼山寨的大门，发现城门上悬挂着一朵很大的白花，想来应该是祭奠他们的三当家。
<br/>　　叶少阳不由苦笑，得亏他们山寨还给他祭奠，八成是大当家觉得他拼死为自己带回一个美女做压寨夫人，很是感动，哪里想到其实带来个祸害，如果大当家的知道真相，估计想把他挖出来挫骨扬灰，不过八成这大当家的也没这个机会了。
