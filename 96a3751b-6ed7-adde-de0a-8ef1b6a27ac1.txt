<br/>    陈明看了一眼，并没有怎么在意，而是感受到了烟生笑的存在，这么浓郁的星辰的力量，除了烟生笑还能是谁？“烟生笑在对面的院子？”

<br/>        大祭司有些吃惊地看了陈明一眼，“殿下如何知道的？”

<br/>        陈明道：“星龙之间的特殊感应。”

<br/>        大祭司对于陈明的身份认可了几分，星龙，还真是神秘之极啊。

<br/>        对面的院子中，大门忽然打开，烟生笑出现在门前，身着朴素，并没有什么特殊之处，但是依旧是步履生莲，如同仙子降临凡尘，自有一股圣洁的气质。

<br/>        陈明看向烟生笑，笑道：“还真是巧啊。”

<br/>        而后啪的一声，关上了院门，说实话，陈明并不知道烟生笑到底为什么对自己很厌恶，应该就是个智障。

<br/>        烟生笑回到院子中，身旁有侍女问道：“小姐，你怎么这么生气啊。”

<br/>        侍女连忙道：“是的，龙雀世子殿下还在南门观，龙雀世子一直喜欢小姐，他可比星龙太子好多了，而且经常询问小姐的身体安恙。”
<br/>    
<br/>        烟生笑又是一巴掌打在侍女的脸上，打得侍女嘴角溢血，“龙雀，算是什么东西，肮脏低贱的龙雀血脉，也配的上和星龙血脉比？”
<br/>    
<br/>        侍女是真的不知道说什么好了，似乎她说什么小姐都会生气，烟生笑道：“去把龙雀世子请来。”
<br/>    
<br/>        待得侍女离开了院子，烟生笑这才看向对面的院子，淡淡道出八个字，“宁为女帝，不做帝后！”
<br/>    
<br/>        若是没有星龙太子，那么自己必定是女帝，执掌妖族的权柄，可是现在，星龙太子出现了，那
<br/>    
<br/>        么自己就会沦为生殖的工具，这不是她想要的。
<br/>    
<br/>        若是对星龙太子拥有一丝情意，那么等待自己的，将会是永居深宫。
<br/>    
<br/>        至少，要等自己成为仙王之后再说。
<br/>    
<br/>        陈明和青龙道人在院子里住下，青龙道人并没有离开，从龙之功，他可不会错过分毫。
<br/>    
<br/>        青龙道人和陈明在院子里喝茶，青龙道人问道：“殿下，要不要去看看星龙仙子，那毕竟是命中注定的姻缘，星龙一族，可是只有殿下和仙子了。”
<br/>    
<br/>        陈明笑了笑道：“烟生笑和我的关系并不好，想必，她会很快来的。”
<br/>    
<br/>        大门很快被踢开，冲进来一个怒气冲冲的年轻人，年轻人剑眉星目，倒是颇为俊朗，不过眼中，却是有着压抑的怒气。

<br/>        青龙道人看了一眼，便认了出来，这是龙雀世子，父亲乃是妖族的龙雀仙王，而且一直喜欢烟生笑。
