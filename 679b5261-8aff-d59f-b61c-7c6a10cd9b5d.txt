　　第749章
<br/>　　李婉儿由一顶小轿抬着出宫，这当然不是帝国公主的待遇，正如程大雷所言，曾经的明玉公主已经死在草原上，活下来的是李婉儿。
<br/>　　多多少少，李婉儿未必肯原谅一力促成和亲的李乐天。
<br/>　　而李乐天心中何尝不是愁绪万千，再见李婉儿，早已不是当初的刁蛮任性，而变得成熟稳重，知道什么话该说，什么话不该说。
<br/>　　但在李乐天心里，还是更喜欢那个刁蛮任性的李婉儿多些。
<br/>　　身在帝王家，不得不长大，她、李行哉、包括自己在内，都是身不由己。既然在这个位置，便不得不担起自己的责任，接受命运的摆布。
<br/>　　踏出殿宇，李乐天来到宫中的校武场，见到马孟起一人在那里操练武艺。
<br/>　　李乐天不得不承认世间有天赋这种东西，李乐天自己的盘龙八打枪其实练得也不如何。八种招式，他只精通一两招，其实都是只知道练法，但却抽不出时间扎实武艺。
<br/>　　自己只是粗浅给马孟起讲解一番，结果他便领悟了精髓，经过几天操练，已经是有模有样，再给他些时间，怕就会到炉火纯青的地步。
<br/>　　啧，李家枪能在其手中发扬光大，也未必不是好事。
<br/>　　见到李乐天过来，马孟起停住手中的木枪，躬身行礼。这么多时日过去，他也不可能一点规矩也不懂。
<br/>　　李乐天挥挥手示意他免礼，道：“不错，短短几天便到这种地步，当真是天纵英才，天下用枪的人中，未必能有几人胜得过你。”
<br/>　　马孟起昂起头：“能胜过程大雷么？”
<br/>　　李乐天顿了顿：“这个朕也说不好，你觉得呢？”
<br/>　　马孟起摇摇头，从表情看上去便知道他也没多少自信。
<br/>　　李乐天武艺不如何，眼光还是有的，他道：“高手相争，本就无有十全把握。你没有把握胜过程大雷，他也未必能稳胜过你。”
<br/>　　马孟起点点头，多少得到了几分安慰。少年意气，最怕碰壁，碰来碰去，便裹足不敢向前。李乐天得到如此得力的手下，还盼着他大放光辉，可不愿他在程大雷受了窝囊气，便折损了一往无前的精气神。
<br/>　　“你年纪轻轻，本应该韬光养晦，不宜锋芒毕露。不过，既然你想出头，朕便给你一个出头，趁着这次藩王入京，让天下人知道你的名姓。”
<br/>　　马孟起竟有几分腼腆，道：“这个不太好吧。”
<br/>　　李乐天白了他一眼，马孟起欲拒还迎，但指不定心里有多高兴呢。
<br/>　　“陛下打算如何安排？”
<br/>　　“我会让百里胜准备一场晚宴，招待各路藩王，到时候，有你扬名露脸的机会。”
<br/>　　马超需要这次机会，李乐天同样需要这次机会。他要告诉天下诸侯，他手中也有马孟起这样的虎将，他，并不是好欺负的。
<br/>　　一顶小轿出得宫门，晃晃悠悠，穿过几条街道，拐入一座宅邸。
<br/>　　这便是李行哉当初在长安时的六王子府，倒也奢侈，不过后来遭了战乱，又没人修缮，其实现在已经有些萧索破败。
<br/>　　李婉儿刚下轿，便看到了李行哉，他身旁还立着一壮汉，魁梧身材，黑紫脸膛。
<br/>　　这是李行哉刚刚招到麾下的徐问天，李行哉与其分别，大步奔到李婉儿面前。
<br/>　　“小妹，你可知我在街上撞见了谁？”
<br/>　　“谁？”
<br/>　　“不是旁个，正是小妹你朝思暮想哪个？”
<br/>　　李婉儿脸上一红：“胡说八道，他已经到了长安？”
<br/>　　“嗳？”李行哉无辜的看着李婉儿：“我都没说是谁，倒要问妹妹一声，他姓甚名谁？”
<br/>　　“讨打，你再不说我可真要生气了。”
<br/>　　李行哉苦笑着求饶，将今日的经历一五一十说了，捎带也提了徐问天几句。
<br/>　　“小妹，你这朝思暮想的，日渐消瘦，这次他可是近在咫尺，你还不去看看，他住的地方可是不远？”
<br/>　　李婉儿一路受他冷嘲热讽，到现在也算是适应了。口中冷哼一声：“既然离得近，走几步路的事，难道他不会过来？”
<br/>　　此时此刻，程大雷坐在厅前饮茶。不多时刘发财从外面回来，已经打探清楚李行哉的住处，除此之外，他随行还有李婉儿。
<br/>　　住的地方确实距离程大雷不算远，只不过隔了一道街。
<br/>　　徐神机坐在一旁，手指拨弄着茶壶：“大当家，我们要不要去拜访一下？”
<br/>　　“嗳，拜访什么，他若想见我们，怕已是早早过来。”
<br/>　　徐神机道：“哪有姑娘登门的道理，这种事大当家应该主动去呐？”
<br/>　　“什么主动不主动，我们是清白的好不好？”
<br/>　　说这话时，程大雷有些做贼心虚。
<br/>　　当初他将李婉儿还给李行哉，在李婉儿的理解中便是神女有梦，襄王无意。而程大雷委实也不算太清白，只不过一时绕不开心思。
<br/>　　他坐在那里，眉头紧锁，想着徐神机的话也未必没有道理。完全可以提些礼物，登门拜访一下，谁又知道自己想见的是李行哉，还是李婉儿。
<br/>　　心里正琢磨着，门外突然来了客人。
<br/>　　这令程大雷觉得有些稀罕，因为他毕竟是山贼出身，即便现在，身上也没有洗去一个贼字。所以来到长安后，门前冷落，任谁也不会登门和一个山贼套近乎。
<br/>　　将人请进来才知道，原来是百里胜下得请柬，请他今夜在楼船一叙。根据传信的人所说，不单单请程大雷一人，此外入京的藩王大多收到了请柬。
<br/>　　“如此说来，去的不止我一个了？”
<br/>　　传信人老实道：“小人负责给程当家送请柬，稍后还要去逍遥王处，其他藩王，也有人传信。”
<br/>　　“你瞧瞧，这可是辛苦你了。”程大雷让刘发财拿出三十两银子：“快快去，可别将侯爷的正经事耽搁了。”
<br/>　　传信人走后，程大雷如沐春风，心里想着今夜无论李婉儿去不去，自己都能和李行哉碰着面，只要能搭上线，再见面也便容易了。
<br/>　　他赶忙沐浴更衣，将头发整齐梳好，各路藩王都是大富大贵的主儿，自己也不可坠了威风。
<br/>　　之后，程大雷便百无聊赖的等着天色暗下来。
