　　一日，夏禹正在与自己的妻子涂山氏吃饭，突然听到有人来报。夏禹皱了皱眉，走出宫殿，对那人问道：
<br/>　　“何事如此惊慌？”
<br/>　　“启禀大王，长江、黄河流域这段时间突然泛滥，江河倒流，山河崩塌，滔滔河水，朝着两岸人族部落冲击而去，不少人族部落都已遭难。”
<br/>　　那人脸上满是惊慌的说道。
<br/>　　“什么？竟有此事！”
<br/>　　夏禹脸上一惊，之后脸色顿时阴沉下来，但却没有怀疑事情的真假，这种事情没人会说谎，于是马上说道：“速速召集众臣，并通知我师尊、师叔等人，我先行一步。”
<br/>　　之后，大禹也没有多说，甚至都来不及告知妻子一声，便飞速朝着黄河、长江流域飞去。
<br/>　　“是！”
<br/>　　那人急忙应了一声，便朝着众臣所在之地奔去。
<br/>　　“轰隆隆！”
<br/>　　当大禹来到现场之时，发现事实果真如那人所言，两江流域的河水已经开始泛滥成灾了。而且事实比那人说的还要严重。
<br/>　　只见那浑浊的河水滔天而起，直上虚空数万丈，然后狠狠地冲击下来，拍击到河道两岸，巨大的力道将河道冲击垮了，河道坍塌，滔天河水径直朝着两边倾泻而出，滚滚大江如同洪荒猛兽，冲击而去。
<br/>　　无边树木倒下，无数山石倒塌，无尽房屋坍塌，无穷庄稼被淹没，无数人族被淹没在大将江之中，挣扎不已！
<br/>　　灾难，无尽的灾难降临，带走无穷生命。
<br/>　　夏禹脸色阴沉，看着那在水中挣扎不断的人类，脸上露出悲痛之色，他大喝一声：“九鼎镇世，济世天下！九鼎，出！”
<br/>　　“一鼎镇天！”
<br/>　　夏禹大手向着虚空一指，顿时，从他头顶飞出一道白色流光，流光冲天，瞬间便直入九霄，化作一方巨大的白鼎。那白鼎一圆口，两方耳，三巨脚，鼎身上可着一副神兽图案，正是螭吻，龙生九子，九子之一。
<br/>　　一道神兽刻纹，布满整个鼎身，一双眼睛中有种莫名的神韵，好似是活的一般。鼎内白茫茫一片，看不清其内的情况，但却可以感受到一股缥缈、悠远、广阔的气息，如同天界。
<br/>　　大鼎散发出淡淡的光晕，向着四方扩散而去。
<br/>　　白鼎倒挂，大口朝下，三脚顶天，散发出强烈无比的威压，像是要将这片天都镇压下去，地面此刻狠狠地震了一震，普通凡人，哪还有能力抵抗。
<br/>　　“一鼎镇地！”
<br/>　　夏禹大帝再次伸出一指，指向大地。此次是从他的脚下飞出一道黄色的流光，流光遁地，开始变大，瞬息之间，便化作一方巨大的黄色元鼎，如之前的白鼎一般形状，只是此次上面刻的是霸下，亦是龙之九子之一，似龟有齿，喜欢负重，鼎内为黄色一片，让人看不清虚实。
<br/>　　黄色光晕向着四方扩散开来，一股莫名的沉重感出现在这片虚空中，让人感受到脚下的大地似乎是厚重了一些，踩上去更让人感觉到踏实。
<br/>　　此鼎并没有像之前那白鼎倒放，似乎是要埋葬什么，它却是四四方方地正放于大地中央，脚底与大地相连，有着源源不断地大地之力注入其中，似乎它本就是大地的一部分，使之变得坚不可摧。
<br/>　　“两鼎镇阴阳！”
<br/>　　夏禹再次向着虚空一指，顿时从他双眼之中飞出两鼎，飞入他前后两方虚空。两鼎缓缓旋转，发出莫名道韵，向四周扩散。
<br/>　　两鼎一金黄色，一冰蓝色，散发出炙热与寒冰的气息，炙热的如同大日降临，冰冷的如同坠入九幽，两鼎似乎是存在某种联系，遥遥相对，互相牵引。
<br/>　　两鼎形状一如之前那般，只是刻纹神兽不同，分别是龙之九子中的蒲牢与狴犴。两神兽刻纹发出耀眼光芒，将这片虚空笼罩在其中，似乎是酝酿着什么东西。
<br/>　　“四鼎镇风水山泽！”
<br/>　　夏禹再次一指，青、黑、银、褐四道流光从他胸口飞出，遁入虚空，稳稳地坐落在四面八方。此四鼎上面的图案分别是嘲风、负屃、睚眦、囚牛。
<br/>　　“一鼎天罚！”
<br/>　　一道紫色流光从他丹田飞出，落到虚空正中央，紫光闪烁，兹兹兹地不断发出电光的声音，好似有一种莫名的天威降临。此鼎身上刻的是狻猊神兽。
<br/>　　九鼎聚九子，九鼎镇九天。九鼎合，祖龙现！
<br/>　　这时，九鼎之间仿佛存在某种特定的气机，瞬间全部放出耀眼的光芒，之后相互牵引，将整片天地都禁锢在其中。
<br/>　　在这片虚空之中，水不能流，风不能吹，树不能摇，山不能晃，天不能暗，地不能震，日月之光不能入侵，一切都仿佛被静止在其中，这片空间完全被禁锢了。
<br/>　　之前那些被洪水冲走的人类，此刻被夏禹大手一招，便全部被移送出来。
<br/>　　夏禹大帝手捏印诀，九鼎轮转，发出惊人的吞噬力，将四周泛滥成灾的洪水吞噬，九鼎就仿佛是九尊滔天巨兽，在猛吞江水与残垣断壁，破败草木之类的东西，来多少，它便吞多少，完全看不到极限。
<br/>　　滚滚河水逆流而上，灌注到九鼎之中，随着河水的吞噬，九鼎愈发显得沉重厚实，给人一种古朴的感觉。
<br/>　　不过是片刻的时间，那滔天洪水便被吞噬一空，而此时，幕后黑手也出现了。
<br/>　　在夏禹的视线中，出现了一头奇特的水怪。这水怪形似猿猴，塌鼻子，凸额头，白毛青身，火眼金睛，他的头颈长达百尺，嘴角露出雪白的獠牙，放出摄人的光芒。
<br/>　　“原来是妖族余孽在兴风作浪，祸乱天下啊，怪不得天下会兴起如此风雨，水患滔天，水怪吃人，种种祸乱，不断出现。”夏禹见到此妖，顿时便明白了这次以及上次天下出现水患的原因了。
<br/>　　“哼！妖族余孽，竟然还敢来冒犯人族，真是不知死活！”
<br/>　　知道是妖在作怪后，大禹更加气愤了。上古时候，妖族视人族为其口粮血食，随处无故屠杀人族，犯下滔天血孽之罪，之后人族崛起，三族大战，巫妖败亡，想不到如今妖族竟还敢对人族动手，简直是不知死活，还真当人族好欺负不成？
<br/>　　大禹眼中满是凌厉的寒光，那是一种刻骨的仇恨，用血都洗不清他们的罪孽。
<br/>　　人族大劫，他虽然没有经历过，但也听说过，那段岁月中，人族是如何的屈辱，悲惨。本以为妖族不会再放肆，可谁知如今还有妖族出来祸乱天下，害死无数无辜百姓，真是该死！
<br/>　　“轰！”
<br/>　　就在大禹要将这罪魁祸首水怪妖猴抓住之时，突然不知为何，那水怪竟然突破了夏禹的九鼎封禁，破开这片虚空，想要逃走。
<br/>　　“什么？竟然破开我的九鼎禁空之法，怪不得敢来人族之地兴风作浪，原来是有点本事啊！哼！在我面前，还想逃走，做梦！”
<br/>　　夏禹先是惊讶，然后就释然了，毕竟他的九鼎还没有大成，最多只能禁锢大罗圆满或者是弱一点的准圣，他的修为也是大罗后期，此妖能破开封禁，想来是有点本事。
<br/>　　眼看水怪就要逃走，大禹手捏印诀，九鼎飞转，瞬间破空而至，瞬间追上妖猴，阻断他的去路，同时，紫色雷鼎中一道神雷轰击过去，直直劈向妖猴！
<br/>　　“吼！”
