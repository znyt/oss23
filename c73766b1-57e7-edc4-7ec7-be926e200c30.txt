        “你、你放我下来！”诺筱颖猛然回神，拍了拍傅夜沉的胸膛。
<br/>
<br/>        在傅夜沉的眼里，她这个动作，就像是在跟他撒娇。
<br/>
<br/>        “不放！”傅夜沉难得一次用这种俏皮的语气跟自己心爱的女人说话。
<br/>
<br/>        诺筱颖用力推了推傅夜沉，谁知，傅夜沉一个转身，将她放下来后，直接按到了墙上动弹不得。
<br/>
<br/>        “宝贝，到底哪个，才是真正的你？”傅夜沉一手撑在诺筱颖脑后的墙壁，一手抓住了诺筱颖的右手，紧紧地按在了他的心口上。
<br/>
<br/>        隔着男人这身精贵的西装，诺筱颖能感觉到自己的手下，是他那强劲有力的心跳。
<br/>
<br/>        “你为什么每次都要跟我说一些莫名其妙的话？”诺筱颖反问。
<br/>
<br/>        傅夜沉会心一笑，二话不说，低头便噙住了诺筱颖的粉唇。
<br/>
<br/>        能让他逗留的时间不多了，所以，他要珍惜与她在一起时的每一分每一秒。
<br/>
<br/>        诺筱颖用力推开这男人后，谁知这男人反而变本加厉，丝毫不给她说话的机会，再次俯身吻了上来。
<br/>
<br/>        “宝贝，什么话都不要再说了，让我吻你一会儿，好吗？”傅夜沉微微离了诺筱颖的唇，用那透着磁性的嗓音，暧昧又温柔地哄道。
<br/>
<br/>        诺筱颖一脸懵，完全不知道他为什么每次一见到她，就这般强吻她，好像弄得他俩是情侣一样。
<br/>
<br/>        但其实，她至今连他的名字都不知道。
<br/>
<br/>        他对她来说，只不过是一个熟悉的陌生人罢了！
<br/>
<br/>        这男人西装下那件一尘不染的衬衫衣领，轻轻擦过她的脸庞，有一股淡淡的清香味。
<br/>
<br/>        这样的气氛太危险，到处都弥漫着一股荷尔蒙气息的味道。
<br/>
<br/>        “我……唔……”
<br/>
<br/>        下一刻，诺筱颖刚到嘴边的话，又被面前这个霸道的男人给吻了回去。
<br/>
<br/>        他微微低头，抬起手来扣住了她的脑袋，吻上了她那粉嫩的双唇。
<br/>
<br/>        接下来的吻，缠绵而温柔，柔情而难舍。
<br/>
<br/>        在傅夜沉心里，从她被刘管家接回来起，就代表着她已经接受他了。
<br/>
<br/>        她就是他的未婚妻，就是他心爱的女人。
<br/>
<br/>        所以，他吻她，理所当然。
<br/>
<br/>        空气仿佛都凝固了一般，诺筱颖大脑里一片空白，浑身有种被触电的感觉，双手抵在傅夜沉的胸膛上，想要推开他，却怎么也推不开。
<br/>
<br/>        傅夜沉的手在她身上游走，诺筱颖的身体猛地战栗了一下，脑子一片空白。
<br/>
<br/>        她……她这是怎么了？
<br/>
<br/>        傅夜沉再次吻上诺筱颖的唇，试图就这样让两个人一起沉沦下去。
<br/>
<br/>        但是，诺筱颖忽然回过神，猝不及防地推开了他，一边摇着头，一边声音细弱蚊蝇地抗拒：“不、不要……不要碰我！”
<br/>
<br/>        傅夜沉怔了一下，看着惊慌失措的诺筱颖，沉默了良久。
<br/>
<br/>        诺筱颖惊惧地瞪着他，让他在她的眼睛里看到她对他的畏惧。
<br/>
<br/>        忽然，傅夜沉的手再次伸进了诺筱颖的衣服里。
<br/>
<br/>        诺筱颖刚想反抗，却忽然意识到，他只是在细细地帮她扣回胸衣。
<br/>
<br/>        而后，他的吻再次落上她的额头，蜻蜓点水似的离开。
<br/>
<br/>        傅夜沉爱怜地抚摸着诺筱颖的脸颊，刚刚那还有些粗重的呼吸声，此刻已经渐渐地平息：“宝贝，对不起，是我太心急了，我不强求你。”
<br/>
<br/>        诺筱颖不明白，他为什么每次见到她，都要对自己做这些只有情侣之间才能做的亲热之事。
<br/>
<br/>        傅夜沉看着面前这个犹如一只受了惊吓般的小鸟依人一样的女人。
<br/>
<br/>        那种保护欲再次萦绕在心头，让傅夜沉忍不住将诺筱颖搂入了怀中。
<br/>
<br/>        “对不起，我真的太想你了。才会忍不住想要吻你，想要要了你。”傅夜沉紧紧地抱着诺筱颖，生怕自己会失去她一样。
<br/>
<br/>        诺筱颖却恼羞成怒地推开傅夜沉，谩骂：“你这个举止轻浮的家伙，不知道男女授受不亲的吗？”
<br/>
<br/>        傅夜沉看着诺筱颖那漂亮的脸蛋红得像个熟透了的红苹果，忍俊不禁地说道：“是是是，夫人说得对，是我太急躁了。以后再见夫人的时候，一定会控制住自己。”
<br/>
<br/>        “拜托你不要对我……”说这些莫名其妙的话，好不好？
<br/>
<br/>        “滋滋——”
<br/>
<br/>        诺筱颖想说的话还未说完，傅夜沉口袋里的手机便震动了起来。
<br/>
<br/>        他直接掏出手机接听了电话。
<br/>
<br/>        下一个瞬间，诺筱颖只见面前这男人眉心紧蹙，像是听到了什么不好的事情一样十万火急。
<br/>
<br/>        傅夜沉一边拿着手机贴在耳边继续接听电话，一边拉起诺筱颖的手，在她的手背上轻轻地吻了吻后，手中不知何时突然冒出一把锋利的口袋刀，将诺筱颖吓了一跳。
<br/>
<br/>        “你想……”做什么？
<br/>
<br/>        诺筱颖心悸地瞪大了眼睛，就在这慌神之际，一抹刀影晃过余光，“啪嗒”一声皮筋断开的声音，让她不知所措。
<br/>
<br/>        她那束成马尾辫的长发，犹如泼墨的丝绸般披散了下来，傅夜沉看着她俏丽又乖巧的模样，薄唇勾起一抹爱慕的弧度。
<br/>
<br/>        等到诺筱颖再定睛的时候，只见面前这男人的手心上多了一绺青丝。
<br/>
<br/>        “以后有你陪伴我出任务，我就会告诫自己，一定要活着回来见你！”傅夜沉上前一步，合上眼帘，在诺筱颖的额头印上自己的唇迹，语毕后，他直接转身，加快步伐，径直离开了。
