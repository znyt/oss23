　　“只是偶然获得了一些机遇。”
<br/>　　天火尊者微微颔首，既然慕清霄不想说，他也不会特地去问，有些事情知道的多未必就是好事。

<br/>　　天火尊者畅快大笑，道：“能让传说中的斗圣请求，就凭这一点，老夫就没理由拒绝你，说吧，只要老夫能做到的，一定会替你完成。”

<br/>　　七阶天毒蝎龙兽魔核与菩提化体液的作用能够让小医仙将体内毒气凝聚，压缩成毒丹，类似修士的金丹。

<br/>　　当初答应过药老帮助他复活，由萧炎收集灵药，他负责斗宗的骨骸。
<br/>　　原本，慕清霄就打算带小医仙和彩鳞去中州逛逛，毕竟他在本世界还有一个任务没有完成，收集异火。
<br/>　　但小医仙体内毒气虽然有害，但何尝不是一种机遇。
<br/>　　将毒丹凝聚，体内的毒气就会成为她的能量，源源不断的吸收，实力绝对突飞猛进。
<br/>　　此事，尽快落实的好。

<br/>　　……
<br/>　　与此同时，内院一座宽敞的阁楼中，数道身影齐聚在此，正是迦南学院的各位长老。

<br/>　　琥乾瞥了眼当时在场的三位长老，道：“事情是因你们而起，不知道三位可有什么应对的方法？”

<br/>　　回想起斗圣降临时，仿佛天都要塌下来的场面，与后者留下的话语，他们心中顿时凉飕飕的。

<br/>　　奇怪的是，阁楼中陷入沉默，没有哪位长老开口否定，随后目光一个个看向紫袍老者。
<br/>　　“此事已经不是我们能做决定的，只是现在联系不到院长，若是能平息斗圣的怒火，将白山交出去，未尝不是一件好办法。”

<br/>　　这名老者话语刚落，便察觉到几道视线落在了他的身上，目光怪异，像是在看智障……
