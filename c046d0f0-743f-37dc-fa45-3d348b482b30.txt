　　“哪有，我就是想劝她不要杀生，她没等我说完就跑了！”叶少阳很委屈。
<br/>　　“她跑不远的，我去追她回来！”瓜瓜说着往山下去追她。
<br/>　　叶少阳稍微放心，想到碧清方才奇怪的表现，纳闷道：“就算她是误会了，她为什么哭了啊？”
<br/>　　大伙都笑着不说话。
<br/>　　周静茹却叹了口气，幽幽说道：“少阳哥，你还是别问了，女孩子的心思，你还是别问了，问了你又负担不起。”
<br/>　　“负担不起？什么意思？”叶少阳挠着头。
<br/>　　周静茹笑道：“说了让你别问。”
<br/>　　叶少阳不明所以。不过“女孩子”三个字，让叶少阳觉得怪怪的，碧清是女孩子……好吧，就算她是女孩子吧。
<br/>　　叶少阳把包子也赶走，让他跟瓜瓜一起去找碧清。
<br/>　　一行人正要下山，老郭突然走到叶少阳身边，戏谑道：“小师弟，身正不怕影子斜。”
<br/>　　“啥意思？”叶少阳不解地看着他，不明白他为什么突然说这句话。
<br/>　　“我是说，身正不怕影子斜，做什么事，自己没那心思就好……”老郭装作不经意地说道，但叶少阳却听出了不对劲：“影子”两个字拖了重音。
<br/>　　影子？
<br/>　　叶少阳猛然意识到什么，装作不经意地回头扫了一眼，眼角余光扫到自己的影子……心猛地咯噔了一下，再看老郭，老郭也在看他，眼神一交汇，彼此都明白了。
<br/>　　叶少阳拍了拍脑袋，道：“你们先走，我要去山下那个小屋看看，把那个出口再封印下，白眉你跟我一起。”
<br/>　　吴嘉伟不明所以，嗯了一声跟上他。
<br/>　　刚走了没几步，周静茹从后面看着他，突然皱眉，惊叫道：“少阳哥，你的影子怎么跟别人不一样啊！”
<br/>　　坏了！
<br/>　　叶少阳猛然转身，自己身后的影子突然紧缩，沿着自己双腿飞快地蔓延上来，凡是被黑影贴住的部位，立刻就酸麻无力。
<br/>　　黑影爬到腿弯处，叶少阳顿时感觉双膝发软，险些跪下，不过也是他早有准备，当即把手从裤兜里拔出来，手里抓着两张暗金神符，分别贴在自己膝盖上，灵符遇到黑影，呲的一声燃烧起来，爆发出地狱之火，黑影瞬间朝双脚褪去。
<br/>　　“天地无极，潜龙勿用，紫月清风，陈黄横渡！急急如律令！”
<br/>　　叶少阳一翻手，拿出太乙拂尘，在符火上蘸了下，朝双脚上扫过去，将黑影从双脚上驱赶下去，但人在光线下面不可能没有影子，纵然将黑影驱到脚底，一旦放松，黑影又会爬上来。
<br/>　　叶少阳趁着黑影被符火赶走，急忙摸出了了两根红线，挂着两枚铸母大钱，绕在自己脚踝上，松开太乙拂尘，果然黑影再度蔓延起来，到了脚踝的位置，被红线挡住，冒出黄色的烟，不断炙烤着挂在红线上的铸母大钱。
<br/>　　本来冰凉的铜钱，一下子变得烫起来，像是两块烙铁，炙烤着小腿肚子上的皮肤。叶少阳疼得龇牙咧嘴，忍着痛，迈步往前狂奔起来。
<br/>　　“少阳哥！”周静茹不知道叶少阳身上发生了什么，急的要冲过去，老郭一下把她拦住，又把大家都拦住，让他们不要过去，为叶少阳分心。
<br/>　　只有吴嘉伟跟着叶少阳奔跑，眨眼到了山脚下。叶少阳感觉那两枚铸母大钱已经陷进了皮肉里，疼得肌肉都在颤抖，但不敢停留，一步冲进了小屋，吴嘉伟随后跟了进去。
<br/>　　叶少阳把门推上，这小屋本来就没有窗户，门合上之后，一点光都没有，到处一片漆黑。
<br/>　　没有光，自然也就没有了影子。
<br/>　　叶少阳感觉自己的双腿立刻就不疼了。
<br/>　　“天眼，打开天眼！”
<br/>　　叶少阳招呼吴嘉伟，自己一边打开了天眼，在黑暗中朝四周望去。
<br/>　　天眼不能让人在黑暗中看见东西（否则就成了超能力），但是就像红外线一样，能看到附近的邪物。
<br/>　　叶少阳没看到邪物，所看到的……还是一片漆黑。
<br/>　　不应该啊！
<br/>　　“哈哈……”
<br/>　　一个男人的笑声响起来。叶少阳四下张望，找不到声音的方向，感觉这笑声无处不在。
<br/>　　“叶少阳，我从来都觉得你是个聪明人，没有想到，你在情急之中也犯了这么严重的错误……这房间里没有光，也就没有了影子，但你想过没有……黑暗，本来就是阴影，在这里，我无处不在……”
<br/>　　叶少阳手持七星龙泉剑，冷静地看着四周，这才明白为什么自己打开天眼之后看到的仍然是一片黑暗：影魅无处不在，占满了这里的全部空间！
<br/>　　“你一直跟着我？”
<br/>　　“不错，本来想在九星阁里擒住你，不成想竟然半路来了鸠摩上师，只好放你出来……但我怎能放你回去。叶少阳，这一战我等了好久了，让我灭了你的肉身魂魄，将你的元神带去轩辕山！”
<br/>　　这可真是哔了狗了，有句话怎么说的来着，刚出虎穴又入狼窝，对，就是这种处境。
<br/>　　叶少阳深吸一口气，强迫自己冷静下来，道：“你是太阴山的长老，也是有身份的，这样行吗，为了公平，咱俩出去打怎么样？一对一单挑！”
<br/>　　“公平？哈哈……”
<br/>　　影魅狂笑起来，“叶少阳，你在人间也算无敌了，居然这么幼稚，手段只是为了结果而服务的，单挑，呵……”
<br/>　　突然一阵风起，有无数的好像丝线一般的东西燃烧上了叶少阳的身体。
<br/>　　叶少阳点燃一道符火，就势看去，是浓的化不开的黑气，如同头发丝一般往自己身上攀援，急忙用符火拍打身体。
<br/>　　符火所到之处，黑气退却，但立刻又从别的方向涌来，叶少阳干脆又点了几张灵符，念咒生成了旋灯火，绕着自己旋转，将黑影逼退出去。
<br/>　　但黑影渗透的力量极强，不断压迫着符火，没一会工夫，符火就要熄灭了。
<br/>　　叶少阳又摸出了几张灵符抵点燃，护住周身。
<br/>　　“我这黑暗之力，取之不尽，我倒要看看你身上有多少灵符。”
<br/>　　影魅以一种戏谑的口吻说道，不急不慢地操控着黑影，去消耗符火的灵力。
