　　第464章
<br/>　　李行哉仍处于一脸懵圈状态。
<br/>　　他在蛤蟆寨待了很久，对于徐神机和程大雷自然都很熟悉。而且据他观察，程大雷和徐神机的关系相当不错。
<br/>　　虽然徐神机平时有些不正经，当然理解为放浪形骸也是行的，但在正经事上他应该能把持得住。这次崔相想要他背叛程大雷，怕是痴心妄想了，徐神机绝对不会这么做。
<br/>　　然而，现实总是出人预料。
<br/>　　徐神机和崔相相携而坐，你一言我一语，一见如故相谈甚欢。
<br/>　　崔相轻咳一声：“徐军师既然有如此壮志，不若就留在我军中……”
<br/>　　“你不用说了。”徐神机大手一挥：“老实告诉你一件事，你想不想知道？”
<br/>　　“什么？”崔相和李行哉都睁大眼睛。
<br/>　　“我其实是被你们故意擒住的。”徐神机亢奋道：“我早就看那程大雷不顺眼了，我是读圣贤书出身，怎么可甘心做一个山贼，委实是生活所迫啊。”
<br/>　　崔相难以置信道：“你的意思是？”
<br/>　　“我被招安了，我愉悦的被招安了。”
<br/>　　李行哉仍然处于懵圈状态，徐神机目光看到他，突然激动起来：“嘿，李兄弟，你也在这里，太好啦，以后我们就一起做大事，干掉程大雷。”
<br/>　　“呃……”李行哉无言以对。
<br/>　　崔相轻咳一声：“徐军师，这里不是山贼窝子，对殿下还是要保持尊敬。”
<br/>　　“喔喔。”徐神机忙点头，冲李行哉抛了个眼色，握紧拳头：“做大事！”
<br/>　　“好啦，徐军师能够认清形势，及时弃暗投明，当真是我军的幸事。来人，带徐军师下去休息，晚上摆好酒宴，本相要好好招待徐军师。”
<br/>　　“随便啦，我这个人不挑食，有口吃的就好。”
<br/>　　徐神机被带了下去，大帐内只剩下李行哉和崔相二人，其实不仅仅是李行哉，崔相也有些懵圈。
<br/>　　这徐神机的表现，和想象中相差太大了。
<br/>　　“殿下，依你对徐神机的了解，他是真有本事，还只是徒有虚名？”
<br/>　　李行哉沉默片刻，道：“如果只是一个酒囊饭袋，程大雷为何对他如此看重。”
<br/>　　“是啦，空穴难来风，既然有此盛名，必然会有些缘由。”崔相道：“只不过他答应得如此痛快，令我担心他会不会包藏什么祸心，殿下觉得怎样？”
<br/>　　李行哉现在也把不准徐神机的脉，谁知道他打得什么鬼主意，不过，目前看来，徐神机是没有生命危险。
<br/>　　于是李行哉只有虚伪的应承着：“防人之心不可无呐。”
<br/>　　“臣明白。”崔相道：“是真金还是烂木头，试一试就知道了。如果装疯卖傻，想要给我耍什么诡计……”
<br/>　　崔相眼睛眯起，杀意弥漫整座山寨。
<br/>　　到了夜里，军营内大摆宴席，一为庆功，而就是因为徐神机的加入。军中的智囊、军师、谋士都围着徐神机。
<br/>　　他们之前都得到了崔相的指示，要摸一摸徐神机的底细。他是真有本事，还是假有本事。另外，他是否真心归顺。
<br/>　　徐神机换过新衣服，一身标准的儒士长袍，倒是达成了毕生溯源。坐在重要的位置上，面对山珍海味，他本就不是太讲究的人，也的确饿了几天，坐在那里打开腮帮子，撩开后槽牙，开始谈狼吞虎咽。
<br/>　　陪坐的人，话语中多试探之意，询问其对一些军事政事的看法。一为摸他的底，另外，其中也有一些真心佩服他的人，毕竟大家都是同行，而在军师这个行业中，徐神机的名声的确称得上不错。
<br/>　　徐神机享受到了他从未享受过的尊重，不禁有些飘飘然，饮了一些酒，嘴巴也放松了。
<br/>　　徐神机在智谋方面的确没什么成长，还和当年在青牛山时一样，甚至可以说有过之而无不及。但在忽悠人这方面……他当真很难遇到对手。
<br/>　　酒宴散去，喝得醉醺醺的徐神机被人扶进大帐，他混的的确不错，刚来第一天，就有专人服侍。当然，也有监视的意思。
<br/>　　喝醉的徐神机呼呼大睡，但崔相却并没有睡着。夜来，他的大帐内灯火通明，几名谋士立在其面前。
<br/>　　“今天酒已喝了，话已说了，对于他的底细，你们摸出多少？”崔相。
<br/>　　“禀大人，这徐神机绝非酒囊饭袋之徒。”一名白衣秀士道。
<br/>　　“喔，为何如此确定？”崔相认得这白衣秀士是来自陆家的年轻人，名叫陆承。
<br/>　　“大人，我已话语试探蛤蟆城的情报，他看似糊涂，言语却密不透风，不露半点有用的信息。”陆承道：“大醉之时，还能如此自制，怎会是酒囊饭袋。”
<br/>　　崔相点点头：“言之有理，不过他不向我们透漏情报，那这归顺之心，怕就不是真的？”
<br/>　　“很有可能。”陆承道：“但或许也是担心狡兔尽走狗烹。”
<br/>　　“的确，若他真是聪明人不会不担心这一点。”崔相自言自语道了一声，随之挥手一笑：“本相又不是那种小肚鸡肠，容不下人才的人。他只要真有本事，哪怕出身不好，又算什么……”
<br/>　　目光扫了一圈，道：“你们都是这一代年轻人里的人才，虽然现在还比不上徐神机，但也不必妄自菲薄，毕竟你们年纪还小。如今他就在咱们身边，你们要多长一双眼睛，多长一双耳朵，多听多看，多从他身上学到一些东西。”
<br/>　　“是！”
<br/>　　众人一起答应，退出了军帐，行走间还交头接耳，议论着这件事。能成为家族着力培养的人，他们自然不会是庸才，也是一心向上，想多一些本事的。
<br/>　　已经有人悄悄盘算着，如何把徐神机的本事学过来。或者成为他的弟子，也是大大一件露脸的事情。
<br/>　　……
<br/>　　夜，草原之上。
<br/>　　程大雷骑着黑牛，行走在月下，身后是三百余养精蓄锐的义字军。
<br/>　　他们一路过来，向青牛山而去，路上并没有遇到什么敌人。但程大雷一颗心，并没有因此而麻痹大意。
<br/>　　除了自身安危外，程大雷还担心徐神机。
<br/>　　嗳，也不知道徐军师现在情形如何。
<br/>(本章完)
