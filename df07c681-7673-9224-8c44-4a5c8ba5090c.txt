　　第71章
<br/>　　李大头来到苏宅，在大屋外等了一会儿，李氏才从屋里出来。
<br/>　　“二妹，事情有些糟糕，对面生意旺了起来。”
<br/>　　“那家破旧楼，已经荒了多久，搁到她手里生意就好起来，她是陶朱公再世么！”李氏刚给苏四海喂过药，手里还端着药碗。
<br/>　　“还真差不多，没几天就起死回生，现在想在湖底捞吃顿饭，都要提前排很久。可他们宁肯在对面排队，也不来四海楼吃饭。”
<br/>　　李氏秀气的眉头皱紧，这件事情实在出乎她的预料。本来想把她嫁到黑石城，一来可以让她离开苏家，二来也是为打通苏家到黑石城的上路，可谁能想到她半路上，竟被山贼劫了。如今给她一座濒临倒闭的酒楼，只想能借此将她赶出黑石城，没料想，她竟然把酒楼救活了。
<br/>　　还真是属狗皮膏药的，怎么也摆脱不了。
<br/>　　“二妹，你想个法子，再这样下去，四海楼倒是要先倒闭了。”
<br/>　　“你真是不中用，让酒楼生意好起来你做不到，把生意搞坏你也不会么！”李氏颦起眉，吐出三个字：“找杜茂！”
<br/>　　杜茂两个字恍惚有莫名的魔力，李大头一激灵，道：“二妹，你想好了么，那杜茂……可是请神容易送神难，咱们真要和他扯上关系么。”
<br/>　　“专业的人做专业的事，杜茂再如何，也无非是要钱而已，给他钱。”
<br/>　　李大头好不容易才鼓起勇气，他从李氏处取了一百两银子，用包袱皮裹了。离开苏宅后，他沿着胡同往东走，见到河的时候往北走……约走了七八里路，前面看到一棵大槐树。
<br/>　　大槐树后有个灰墙院子，李大头走进后，停了片刻，才鼓起勇气敲了敲门环。
<br/>　　“干嘛！”
<br/>　　从门里探出个大脑袋，秃头黑脸，眼睛像铜铃一样大。
<br/>　　“找……找杜爷。”
<br/>　　大脑袋左右看看，最后才把目光落在李大头身上：“进来吧。”
<br/>　　手里将那蓝布包袱裹在怀里，李大头走进院子。院子里摆着一排兵器架，石磨，石锁等物，七八条大汉坐在太阳底下，人人都赤着臂膀，操练着院子里的器械。
<br/>　　“老疤，过两天幽州那边过来个兄弟，你给支应一下。”
<br/>　　“做哪一行的？”
<br/>　　“水上的，刚劫了一批货，打算在这里避避风头，手上东西不少……”
<br/>　　看到李大头进来，谈话声嘎然而至，几乎所有人的目光都落在李大头身上。李大头身子瞬间矮了几寸，天空上艳阳高照，可无端端却涌来一种阴寒。
<br/>　　院内有棵槐树，打下一片树荫，在树荫下坐着一白袍男人。他穿着长袍，酌着清茶，和院里赤条大汉相比截然相反。
<br/>　　一动一静，可这不动声色的静，却能压住噪杂的动。
<br/>　　只有在目光放到李大头身上时，李大头才有一种被老虎注视的感觉。
<br/>　　这人就是杜茂，字文才，绰号懒大虫，据说双臂有千斤之力，曾经肉拳打死过山林中的老虎。
<br/>　　李大头几乎是一步步挪到对方身前，将手中的包袱搁在石桌上。
<br/>　　“杜爷，劳您帮个小忙……”
<br/>　　杜茂用手掂了掂，也就估出了大概数目，他挥手丢给那个脸上有疤的大汉：“疤子，把事情帮人办了。”
<br/>　　“得了，爷，您就瞧好吧！”
<br/>　　……
<br/>　　这天下午，湖底捞门口出现两个人。
<br/>　　“就是这里了。”
<br/>　　“生意还挺好。”
<br/>　　“走，进去瞧瞧。”
<br/>　　二人踏入大门，过了中午饭点，生意竟然还不错。二人等了好大一会，才轮到自己的位置。
<br/>　　这是土砌的灶台，下面填着木柴，有烟道把烟引出去。如此，二楼就没办法布置座位，只在一层有三十几台座位。
<br/>　　“疤哥，这里生意这么好，菜真有外面传得那么好吃。”脸上纹着一只青蛇的男人问。
<br/>　　“呵，咱兄弟走南闯北，龙肝凤髓，什么没吃过，好吃，又能有多好吃。”
<br/>　　“那也不着急，咱们兄弟先尝尝，再做事。”
<br/>　　苏樱正在后院喂鸽子，她叫小蝶过去蛤蟆寨，给程大雷养了几天信鸽，总算可以让三只信鸽在两地之间来往。
<br/>　　这本是为有要事时联系方便，可程大雷一天三封信雷打不动，如果多少有些事也算了，可『吃了吗』『睡了吗』『洗澡了吗』这种事也要每天都问么。现在苏樱觉得，直到现在为止，自己还没有摸清楚程大雷的套路。
<br/>　　大厅内，刀疤和青蛇两人吃得大块朵颐，汗水直冒。
<br/>　　肉片在红汤里翻滚，花椒滚到一边，筷子从天而降，追逐着汤底的丸子。
<br/>　　“疤哥，你饶我这一次。”
<br/>　　“嗨，长幼有序。”
<br/>　　“小二，再加份肉。”
<br/>　　吃得酒足饭饱，二人倒在凳子上打嗝，目光相对，似乎都记起了什么。
<br/>　　“哇，老鼠！”
<br/>　　刀疤忽然一声大叫，大袖子在锅上一挥，就有一死老鼠被抛在锅里，随着锅底翻滚。
<br/>　　一片大哗，酒楼还有的几座客人都惊叫起来。目光透过来，看到锅里那只露着牙齿的死老鼠时，都做出恶心反胃的表情。
<br/>　　门外的廖甲廖乙两兄弟率先过来。
<br/>　　廖甲看着红光满面的二人，再看看锅里翻滚的老鼠，嘴角浮现一抹冷笑。
<br/>　　“兄弟，这把戏可真够老的，吃到最后才吃出老鼠，二位的眼力可是有够差的。”
<br/>　　“是啊，兄弟从小眼力就不好，这老鼠都吃了半拉。”
<br/>　　廖甲一怔，锅里的老鼠明明完整无缺。
<br/>　　正想着，忽然见刀疤从身上拔出短刀，将锅里的老鼠挑出来，眼神示威般盯着廖甲，慢慢把那老鼠放在口中，连皮带骨一起嚼。
<br/>　　凡是看到这这一幕的，都大惊失色，胃里的酸水往上涌，他们几乎是逃也似的离开酒楼，半路上还能听见哇哇的呕吐声。
<br/>　　廖甲却是脸色不变，盯着刀疤，直到他咬下去一半，再将死老鼠吐出了。
<br/>　　“喔，原来你们是……流……氓啊！”
<br/>　　“好说好说。”刀疤用短刀剔牙，然后冲廖甲拱了拱手：“这位兄弟是……”
<br/>　　廖甲双手抱拳，气沉丹田，口中吐出两个字。
<br/>　　“同……行。”
<br/>　　
<br/>(本章完)
